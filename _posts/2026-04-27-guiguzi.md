## 全书概览

《鬼谷子》是中国古代游说、谋略与识人之学的源头之一，与《战国策》互为表里——策士之言，多本于此书。全书十二篇，从**开口闭口**（捭阖）到**揣摩决断**（揣、摩、权、谋、决），构成一套完整的**人际博弈与说服操作系统**。

与兵书、史书不同，鬼谷子回答的不是「如何排兵」或「何以兴亡」，而是：

> **在信息不对称、利益多元、权力不对称的对话中，如何察意、言说、试探、权衡并决断？**

> 智用于众人之所不能知，而能用于众人之所不能，潜谋于无形，常胜于不争不费。

读鬼谷子，宜记三问：**对方真正要什么？我何时开阖？决断窗口在哪？**

### 战国纵横语境：为何需要「捭阖」？

战国策士无固定封地，凭**口才与情报**换信任与权位。列国兼并加速，盟约易变，**一句话可退兵、一句话可亡国**。鬼谷子之异于一般游说术，在于把对话拆成**可练的操作序列**：

| 压力 | 鬼谷子式回应 |
|------|--------------|
| 不知对方虚实 | 捭阖：开阖有时，辨有无 |
| 对方闭口不言 | 反应：反以观往，钓语引话 |
| 无信任基础 | 内揵：楔子打入，层层递进 |
| 对手强盛无隙 | 抵巇：察缝隙，乘虚击 |
| 需要绑定合作 | 飞钳：扬辞钳势 |
| 时势突变 | 忤合：因势合忤，不泥一主 |

可与 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})并读：鬼谷子察**交换背后的情志**，管理论**交换结构是否可持续**。

## 一、捭阖篇 — 开口与闭口

<figure>
<img src="{{ '/assets/images/guiguzi/01-baihe.svg' | relative_url }}" alt="捭阖篇示意图" />
<figcaption>图 1 — 捭阖：一开一闭，辨虚实而定言说之道</figcaption>
</figure>

### 语境要点

- **捭**（开）：张扬、引导、让对方吐露真情
- **阖**（闭）：收敛、守密、让对方自行显露
- 核心：「审定有无与其实虚，同异与其能否」——先辨虚实，再定开阖
- 圣人之道：「一阴一阳之谓道」——言说亦分阴阳，不可一味强攻
- 与 [孙子]({{ '/2026/06/08/sunzi.html' | relative_url }})「形人而我无形」：捭阖是**对话层**的示形与藏形

### 典型史案

**张仪戏楚**  
张仪以六百里商於之地诱楚怀王绝齐，实为虚辞。先**捭**（开大利益，引楚王心动），待楚绝齐后**阖**（闭口不认，反受秦攻）。楚王因贪开而不能阖，终至丧师失地。


### 用法与场景

| 场景 | 捭（开） | 阖（闭） | 具体操作 |
|------|----------|----------|----------|
| 客户访谈 | 开放式提问 | 沉默倾听 | 问「最痛一环」后停 5 秒 |
| 谈判开局 | 抛愿景框架 | 底价闭口 | 先谈价值，关键数字后出 |
| 向上汇报 | 展开成果选项 | 风险先阖 | 未定事项等领导追问 |
| 面试识人 | 「谈谈最大失败」 | 听完不评价 | 看是否继续自曝 |
| 冲突降温 | 阖：少辩解 | 待对方泄尽再捭 | 避免抢话 |
| 销售 | 捭需求 | 阖方案 | 多听少推 |

## 二、反应篇 — 反观往复

<figure>
<img src="{{ '/assets/images/guiguzi/02-fanying.svg' | relative_url }}" alt="反应篇示意图" />
<figcaption>图 2 — 反应：反以观往，覆以察来，象比钓语</figcaption>
</figure>

### 语境要点

- 「反以观往，覆以察来」——用过去推未来，用对方之言反推其意
- 核心手法：**象比**（以类比诱导）、**钓语**（以饵引话）
- 欲闻其声，先默；欲张之，先阖——反向操作
- 与 [非暴力沟通]({{ '/2026/06/01-nonviolent-communication-rosenberg.html' | relative_url }})「倾听」可对照：反应篇更重**策略性反推**

### 典型史案

**苏秦说秦王**  
苏秦初说秦惠王，秦王不纳。苏秦**反其道**：转投六国合纵，待秦惧而合纵成势，再令天下知秦之强不可独抗。以「往之不纳」观「来之必惧」，终使秦国重视连横之策。

**现代镜：需求澄清**  
用户说「我要更快」→ 反问「目前最慢一环在哪？」——以彼之言导彼之实。

### 用法与场景

| 场景 | 反应术 | 具体操作 |
|------|--------|----------|
| 需求澄清 | 反问链 | 「更快」→「哪一步最慢？」→「慢的成本是？」 |
| 冲突调解 | 覆 + 反 | 复述甲方 → 引乙方对比 → 不直接否定 |
| 产品调研 | 钓语 | 「你们现在怎么解决？」让其自曝流程 |
| 绩效面谈 | 自述优先 | 让员工先说不足，再给建议 |
| 谈判 | 覆确认 | 「您的意思是…若我理解错请纠正」 |
| 向上管理 | 观往察来 | 领导过去否决什么？暗示未来要什么？ |

## 三、内揵篇 — 内合外应

<figure>
<img src="{{ '/assets/images/guiguzi/03-neijian.svg' | relative_url }}" alt="内揵篇示意图" />
<figcaption>图 3 — 内揵：以信任楔子打入，上中下三层递进</figcaption>
</figure>

### 语境要点

- **揵**：楔子，打入对方内部建立信任与影响力
- 三类人：上揵（君主级）、中揵（权臣级）、下揵（执行级）
- 「事皆有内揵，素结本始」——一切游说必先有内应或信任基础
- 远而亲者，有阴德也；近而疏者，志不合也
- 与 [韩非子]({{ '/2026/05/02/hanfeizi.html' | relative_url }})「术」：内揵是横向信任，法术势是纵向控制

### 典型史案

**张仪内揵于秦**  
张仪落魄入秦，先结纳掾史、司马错等中层（中揵），再献连横之策于秦惠王（上揵）。若无内廷信任，纵有良策亦难施行。


### 用法与场景

| 场景 | 内揵策略 | 具体操作 |
|------|----------|----------|
| 大客户销售 | champion | 先搞定使用者/技术负责人，再触达 CFO/CEO |
| 跨部门项目 | 联盟 | 各部门找利益一致者，建联络人网络 |
| 组织变革 | 意见领袖 | 先争取 KOL 试点，再全员推广 |
| 海外拓展 | 本地合伙人 | 下揵/中揵后再触达政府或总部 |
| 内部提案 | 预沟通 | 会前一对一，会中少意外 |
| 求职 | 内推链 | 团队成员背书 > 海投 HR |

## 四、抵巇篇 — 乘隙击虚

<figure>
<img src="{{ '/assets/images/guiguzi/04-dixi.svg' | relative_url }}" alt="抵巇篇示意图" />
<figcaption>图 4 — 抵巇：察缝隙、乘虚击，五类巇各有其用</figcaption>
</figure>

### 语境要点

- **巇**：缝隙、裂痕——组织、人心、盟约中的薄弱环节
- 「自天地之弥合，终复归其巇」——事物盛极必衰，隙必生
- 五类巇：明巇、暗巇、贼巇、内巇、间巇
- 圣人见巇而填之、或因而用之
- 与 [孙子]({{ '/2026/06/08/sunzi.html' | relative_url }})「避实击虚」：抵巇偏**人心与组织裂痕**，孙子偏**战场部署**

### 典型史案

**孙膑减灶诱庞涓**  
魏军强势，正面不敌。孙膑察庞涓骄矜之**巇**（轻敌），减灶示弱，引其深入马陵——以敌之隙，成致命一击。


### 用法与场景

| 场景 | 抵巇术 | 具体操作 |
|------|--------|----------|
| 竞争分析 | 找薄弱环 | 产品、交付、售后、组织变动期 |
| 危机干预 | 早填小巇 | 1:1 早谈，避免演变成大裂 |
| 团队管理 | 察信任裂 | 跨组项目注意「间巇」 |
| 市场切入 | 长尾/下沉 | 大厂忽视处即巇 |
| 自我防守 | 填己之巇 | 复盘自己流程断点 |
| 联盟谈判 | 察伙伴隙 | 勿在对方内斗时强推条款 |

## 五、飞钳篇 — 飞而钳之

<figure>
<img src="{{ '/assets/images/guiguzi/05-feiqian.svg' | relative_url }}" alt="飞钳篇示意图" />
<figcaption>图 5 — 飞钳：先扬辞令其心服，再钳势使其不脱</figcaption>
</figure>

### 语境要点

- **飞**：飞扬其辞，令对方飘飘然
- **钳**：钳制其势，使其不得脱
- 用 praise、利益、名声笼络，再以利害钳住
- 「立势而制事」——先造势，再制人
- 与 [菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})「侠气素心」：飞钳宜**真诚认可**，飞而不实则谄

### 典型史案

**范雎说秦王**  
范雎入秦，先谦抑自贬（飞其谦德），秦王再三恳请，方献远交近攻之策。秦王既敬其才，又赖其策，范雎遂钳持秦相之位。


### 用法与场景

| 场景 | 飞钳术 | 具体操作 |
|------|--------|----------|
| 人才招募 | 认可 + 使命 | 真诚讲对方过往成就，再以愿景绑定 |
| 合作谈判 | 抬举 + 排他 | 先尊对方行业地位，再谈独家/深度条款 |
| 演讲说服 | 开场赞美 | 建立好感后再提挑战与方案 |
| 平台生态 | 流量飞、数据钳 | 先给资源，再建迁移成本（须合规） |
| 客户成功 | 成就飞、续约钳 | 帮客户出成绩，再续费自然 |
| 注意边界 | 分寸 | 飞而不实则谄，钳而过甚则胁 |

## 六、忤合篇 — 合纵连横之机

<figure>
<img src="{{ '/assets/images/guiguzi/06-wuhe.svg' | relative_url }}" alt="忤合篇示意图" />
<figcaption>图 6 — 忤合：因势而合，因势而忤，苏秦张仪各得其时</figcaption>
</figure>

### 语境要点

- **忤**：背逆、不合；**合**：契合、联合
- 「世无常贵，事无常师」——没有永远的对立面，只有当下的势
- 择主而事，择时而动；知其开阖，说之以其类
- 谋臣之职：因时因势，忤而合之，合而忤之
- 与 [周易]({{ '/2026/05/07/zhouyi.html' | relative_url }})变通、[通鉴]({{ '/2026/05/25/zizhi-tongjian.html' | relative_url }})党争镜：忤合是**势**，非站队标签

### 典型史案

**苏秦张仪交替**  
苏秦合纵抗秦，张仪连横破纵。二人同窗不同路，各择其主、各合其时。苏秦得势于六国衰危之际，张仪得势于秦强六国惧之时——**忤合**于势，非忠于一人。

### 用法与场景

| 场景 | 忤合术 | 具体操作 |
|------|--------|----------|
| 职业规划 | 择势 | 行业下行忤（离开），风口合（加入） |
| 战略合作 | 竞合 | 今日竞合明日联盟——以利益与时势为准 |
| 产品 pivot | 方向 | 原方向忤，新信号合 |
| 政治智慧 | 进谏窗 | 不在逆势时强谏，待可谏之窗 |
| 投资 | 周期 | 忤高估值，合低估优质 |
| 个人品牌 | 叙事 | 势变时更新定位，非死守旧标签 |

## 七、揣篇 — 揣情察意

<figure>
<img src="{{ '/assets/images/guiguzi/07-chuai.svg' | relative_url }}" alt="揣篇示意图" />
<figcaption>图 7 — 揣：以耳目观外在言行，推度内隐情志</figcaption>
</figure>

### 语境要点

- **揣**：度量、揣测——从外在言行推内在情志
- 「其变在外者，常有馀；其变在内者，常有不足」
- 观其所好，以知其志；观其所恶，以知其情
- 不直接问，而观其应激反应
- 与 [认知神经科学]({{ '/2026/05/12/gazzaniga-cognitive-neuroscience.html' | relative_url }})：人常事后合理化；揣重**行为与应激**

### 典型史案

**烛之武退秦师**  
郑国将亡，烛之武夜见秦穆公。不直接求援，而**揣**秦之虑：灭郑无益于秦，反壮晋国之东境。以秦之私心（惧晋）破秦郑之隙，秦遂退兵。

### 用法与场景

| 场景 | 揣术 | 具体操作 |
|------|------|----------|
| 用户研究 | 行为 > 言语 | 看点击、留存、抱怨重复点 |
| 招聘面试 | 观察应激 | 谈失败、谈前老板、谈钱时反应 |
| 商务宴请 | 细节 | 点菜、敬酒、谈伴选择 |
| 投资尽调 | 创始人 | 如何对待小供应商、早期员工 |
| 合伙 | 压力测试 | 小项目看分钱与担责 |
| 家庭 | 少问多听 | 看行动模式 |

## 八、摩篇 — 摩意试探

<figure>
<img src="{{ '/assets/images/guiguzi/08-mo.svg' | relative_url }}" alt="摩篇示意图" />
<figcaption>图 8 — 摩：微触试探，如操钩临渊，饵投而察其应</figcaption>
</figure>

### 语境要点

- **摩**：摩擦、试探——如以羽毛摩物，微触而察其应
- 与揣篇相连：揣是静观，摩是主动轻触
- 「古之善摩者，如操钩而临深渊，饵而投之，必得鱼焉」
- 欲使对方动，先以小激之
- 与产品 **MVP / A/B** 同构：小成本探反应

### 典型史案

**毛遂自荐**  
平原君欲合纵赴楚，门客不足。毛遂**摩**其急：主动请行，至楚廷以三寸舌摩楚王——先以弱国之急轻触，再层层递进，终定盟约。

### 用法与场景

| 场景 | 摩术 | 具体操作 |
|------|------|----------|
| A/B 测试 | 小流量 | 5% 用户试特性，再放大 |
| 价格探测 | 阶梯报价 | 先高后低，观反应点 |
| 内部提案 | 会前摩 | 非正式聊关键人态度 |
| 社交破冰 | 低风险话题 | 天气、行业 → 再深入 |
| 销售 | 试用/POC | 小单验证再大单 |
| 求职 | 咖啡聊 | 正式面试前先摩文化 |

## 九、权篇 — 权衡利害

<figure>
<img src="{{ '/assets/images/guiguzi/09-quan.svg' | relative_url }}" alt="权篇示意图" />
<figcaption>图 9 — 权：权衡利害轻重，计寡者智，计多者乱</figcaption>
</figure>

### 语境要点

- **权**：秤锤，权衡——比较轻重、利害、远近、虚实
- 「计多者乱，计寡者智，计不信者诚，计拙者诚，计利者从，计害者避」
- 说人者：务在博其说，信其辞，杂而合之，乱而取之
- 因事为制，权衡于心
- 与 [孙子庙算]({{ '/2026/06/08/sunzi.html' | relative_url }})：权偏**对话中的利害呈现**，庙算偏**战略对比**

### 典型史案

**蔺相如完璧归赵**  
秦王以十五城换和氏璧，蔺相如**权**之：秦强赵弱，直拒则危；遂许献璧至秦，察秦王无诚意（权其虚），乃持璧以撞柱相胁，终完璧归赵。

### 用法与场景

| 场景 | 权术 | 具体操作 |
|------|------|----------|
| 方案评审 | 权重表 | 3 选项 × 5 维度，计寡 |
| 资源分配 | ROI | 寡计而精行 |
| 谈判让步 | 对价 | 每次让步必有换回 |
| 技术选型 | 长短期 | 维护成本 vs 交付速度 |
| 说服 | 利害叙事 | 不只讲利，讲不做的害 |
| 个人决策 | 二选一 | 48 小时内权清，再迭代 |

## 十、谋篇 — 阴谋阳略

<figure>
<img src="{{ '/assets/images/guiguzi/10-mou.svg' | relative_url }}" alt="谋篇示意图" />
<figcaption>图 10 — 谋：阴中策划，阳中成事，商鞅变法为例</figcaption>
</figure>

### 语境要点

- 「谋之于阴，成之于阳」——策划在暗，成事在明
- 为人臣者，莫难在於进说，莫妙在於得君之旨
- 同利相导，同害相救——找共同利益点串联多方
- 不见其类而为之者，见逆；不得其情而说之者，见非
- 与 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }})：阴谋阶段小范围试错，阳略阶段公开固化

### 典型史案

**商鞅变法**  
谋之于孝公之前（阴）：三次说以帝道、王道、霸道，终以强国之术合其旨。成之于秦国法令（阳）：徙木立信，公开颁法。阴谋得君心，阳略行天下。

### 用法与场景

| 场景 | 谋术 | 具体操作 |
|------|------|----------|
| 产品发布 | 内测 + 发布 | 灰度收集反馈，发布会定调 |
| 组织改革 | 试点推广 | 单部门验证，再全员 |
| 联盟构建 | 私下对齐 | 一对一谈妥，再签公开协议 |
| 个人影响力 | 积累亮剑 | 默默专业口碑，关键场合一次 |
| 危机公关 | 预案 | 阴：沙盘与口径；阳：声明与行动 |
| 进说 | 得旨 | 先摸上级真意图，再配方案 |

## 十一、决篇 — 决断之机

<figure>
<img src="{{ '/assets/images/guiguzi/11-jue.svg' | relative_url }}" alt="决篇示意图" />
<figcaption>图 11 — 决：五机之中速断疑虑，机不可失</figcaption>
</figure>

### 语境要点

- **决**：决断——在五机（危、安、死、生、誉、辱）中速断
- 「决断以决疑」——犹豫是最大的成本
- 善决者：先察是非之语，别成败之纪，察同异之党，权当势之宜
- 度远而近，审古而察今
- 与 [说「不」的艺术]({{ '/2026/05/21-art-of-saying-no-refusal.html' | relative_url }})：决包含**对己对彼的止损决断**

### 典型史案

**鸿门宴范增举玦**  
项羽宴刘邦，范增数举玉玦示杀意。项王**不决**，沛公脱险。范增叹曰：「竖子不足与谋！」——机不可失，决而不断，反受其乱。见 [通鉴]({{ '/2026/05/25/zizhi-tongjian.html' | relative_url }})谋断篇。

### 用法与场景

| 场景 | 决术 | 具体操作 |
|------|------|----------|
| 创业 pivot | 数据触发 | 指标连跌 N 周即关停 |
| 人事决策 | 去留 | 察其已久，触发线即执行 |
| 投资退出 | 止损 | 设线触发，不讨价还价 |
| 日常管理 | 二选一 | 权势之宜后 24h 内决 |
| 谈判收口 | 截止日 | 无进展则 BATNA |
| 个人 | 五机 | 问：辱/死/危是否已近？ |

## 十二、符言篇 — 持符守信

<figure>
<img src="{{ '/assets/images/guiguzi/12-fuyan.svg' | relative_url }}" alt="符言篇示意图" />
<figcaption>图 12 — 符言：耳目心协调，君臣各守其符</figcaption>
</figure>

### 语境要点

- **符**：信符、符节——君主与谋士之间的信任契约
- 安徐正静，柔节先定；善与而不静，先意从欲
- 目者可以观色，耳者可以听声，心者可以知情
- 总纲：耳目心三者协调，言听计从而不妄动
- 与 [菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})符言篇：谋臣不越位，主公不微服乱谋

### 典型史案

**管仲相桓公**  
桓公以鲍叔牙荐，授管仲全权。管仲**持符**于桓：「国有政于仲，无得而问焉。」君臣各守其位，管仲谋、桓公信，遂成霸业。

### 用法与场景

| 场景 | 符言术 | 具体操作 |
|------|--------|----------|
| 顾问角色 | 边界 | 提供选项与风险，不替客户签字 |
| 授权管理 | 书面符 | 明确决策权清单，上级不越级 |
| 自我修炼 | 耳目心 | 会议多听少抢话 |
| 长期关系 | 信符 | 勿因小利毁约 |
| 上下级 | 各守其位 | 下属不代老板决，老板不微操 |
| 合伙 | 协议 | 股权与决策权符节清晰 |

## 阅读边界

《鬼谷子》教的是**术**，不是**德**。读此书宜：

1. **明其理，不泥其诈** — 识破套路，亦守己之诚
2. **识势而不逐奸** — 势可借，道不可弃
3. **古今异制** — 战国策士以口舌换封地，今人宜用于正当商业与职业场景
4. **与儒家互补** — 鬼谷子教「如何成」，孟子教「为何成」；术德并修，方为上乘
5. **忌术成表演** — 未真揣情而表演捭阖，易露怯
