## 全书概览

《周易》是中国最古老的经典之一，核心只有一句话可记：**易者，变也**。它以阴阳两爻叠成八卦，再重成六十四卦，每卦有卦辞、爻辞，描述人在不同情境下如何察势、守位、变通。后世儒家、道家、兵法、医理、风水皆由此出，堪称中华文明的**变化操作系统**。

> 天行健，君子以自强不息。地势坤，君子以厚德载物。

## 一、阴阳八卦 — 生成之本

<figure class="diagram-figure">
<img src="{{ '/assets/images/zhouyi/01-yinyang-bagua.svg' | relative_url }}" alt="阴阳八卦示意图" />
<figcaption>图 1 — 太极生两仪，两仪生四象，四象生八卦</figcaption>
</figure>

### 语境要点

- **阳爻**（—）与**阴爻**（- -）是基本单位；万物皆可用阴阳关系描述，非简单善恶二分。
- **八卦**：乾天、坤地、震雷、巽风、坎水、离火、艮山、兑泽——八种基本「情境原型」。
- **重卦成六十四卦**：上卦为外、为远；下卦为内、为近。读卦先看整体象意，再析爻位。
- 《系辞》：「一阴一阳之谓道。」变化本身即是规律。

### 阳爻与阴爻

| 名称 | 符号（常见写法） | 性质 | 记数（占筮） |
|------|------------------|------|--------------|
| **阳爻** | `—`（一实线） | 刚、动、明、奇 | 少阳 **7**、老阳 **9** |
| **阴爻** | `- -`（一断线） | 柔、静、暗、偶 | 少阴 **8**、老阴 **6** |

- **静爻**：摇卦得出 7（少阳）或 8（少阴），爻象不变，只参与组成本卦。
- **动爻**（变爻）：老阳 **9**、老阴 **6**——爻性会变（阳变阴、阴变阳），本卦之外再成**变卦**，示事态发展趋向。
- 读辞顺序：先看**本卦卦辞**定大局，再按动爻位置读**爻辞**；无动爻专看卦辞，一爻动读该爻，多爻动有「多爻断法」（宜先熟一爻动）。

### 六爻：一卦之结构

六十四卦每卦由**六条爻**自下而上叠成。读卦**从下往上**写、从上往下**审势**：

```
上六  ← 第六爻（卦之终、结果、极处）
九五  ← 第五爻（君位、尊位，多主大事）
六四  ← 第四爻（近君、入上卦，多涉外务）
九三  ← 第三爻（下卦之极，多险多慎）
六二  ← 第二爻（中位、臣位，多尚中正）
初九  ← 第一爻（始、潜、基层；卦之脚）
─────  下卦（内卦）= 初、二、三爻
─────  上卦（外卦）= 四、五、上爻
```

| 层次 | 三爻 | 象意 |
|------|------|------|
| **下卦（内卦）** | 初、二、三 | 自身、内部、近事、根基 |
| **上卦（外卦）** | 四、五、上 | 环境、外部、远势、归宿 |

| 爻位 | 别称 | 贵贱与角色 |
|------|------|------------|
| 初 | 始 | 事之始、位卑，宜潜不宜躁 |
| 二 | 中 | 下卦中位，多尚「中正」，臣位 |
| 三 | 人 | 下卦之极，多「多凶多慎」 |
| 四 | 近 | 近君、近上，进退两难常见 |
| 五 | 君 | 尊位，多主卦中「主爻」 |
| 上 | 终 | 物之终、势之极，过则变 |

- **阳位**（初、三、五）：阳爻居之为「当位」，多顺；阴爻居之为「不当位」，多乖。
- **阴位**（二、四、上）：阴爻居之为当位；阳爻居之，或刚猛过甚，宜观爻辞裁断。
- 爻名：**九**为阳爻（如九三），**六**为阴爻（如六二）；唯「用九」「用六」为乾、坤两卦特例。

### 八卦速查

| 卦 | 象 | 德性 | 现代联想 |
|----|-----|------|----------|
| 乾 ☰ | 天 | 健、创始 | 战略、领导、开创 |
| 坤 ☷ | 地 | 顺、承载 | 执行、支持、落地 |
| 震 ☳ | 雷 | 动、惊 | 启动、突破、震荡 |
| 巽 ☴ | 风 | 入、顺 | 渗透、沟通、渐进 |
| 坎 ☵ | 水 | 陷、险 | 风险、考验、深潜 |
| 离 ☲ | 火 | 丽、明 | 文明、曝光、依附 |
| 艮 ☶ | 山 | 止、静 | 边界、暂停、坚守 |
| 兑 ☱ | 泽 | 悦、说 | 交流、和悦、开口 |

### 案例

**伏羲画卦传说**：观天地文理、鸟兽迹，始作八卦。无论传说真伪，其核心是：**符号来自对自然模式的抽象**——与现代数据建模「找 pattern」同构。

### 用法

| 场景 | 八卦思维 |
|------|----------|
| 项目诊断 | 问：当前更像「水」（险而进）还是「山」（止而守）？ |
| 团队角色 | 乾为定方向，坤为抓落实，震为破局，艮为收口 |
| 冲突分析 | 离火（明争）vs 坎水（暗涌）——辨冲突形态再施策 |

**口诀**：一阴一阳之谓道，观象先于断吉凶；六爻自下而上，内卦主己、外卦主势。

## 二、乾坤之道 — 创始与承顺

<figure class="diagram-figure">
<img src="{{ '/assets/images/zhouyi/02-qian-kun.svg' | relative_url }}" alt="乾坤两卦示意图" />
<figcaption>图 2 — 乾健坤顺：自强不息与厚德载物</figcaption>
</figure>

### 语境要点

- **乾卦**：纯阳，「元亨利贞」——创始、亨通、适宜、守正。君子以自强不息。
- **坤卦**：纯阴，「利牝马之贞」——顺而不逆，厚而能载。君子以厚德载物。
- 乾坤不只是一男一女，更是**一切创造的两种力量**：发动与承载、愿景与执行。
- 乾过则亢（上九：亢龙有悔），坤过则疑（臣道不可独断）。

### 名句

| 原文 | 要义 |
|------|------|
| 天行健，君子以自强不息 | 持续精进，不怠于时 |
| 地势坤，君子以厚德载物 | 包容承载，以德服人 |
| 亢龙有悔 | 盛极必衰，知止为吉 |
| 黄裳，元吉 | 居中守谦，大吉 |

### 案例

**刘邦与萧何**：刘邦（乾）善断大势、敢作敢为；萧何（坤）守后方、镇关中、输粮饷。乾坤相配，非主仆简单分工，而是**创始之力必须有承载之德**，否则亢龙有悔。

### 用法

| 场景 | 乾坤术 |
|------|--------|
| 创业 | 创始人乾：定愿景；联合创始人坤：建系统、兜底线 |
| 个人成长 | 乾：学新技能；坤：把技能沉淀为习惯与作品 |
| 领导力 | 乾过则独断——学坤之倾听；坤过则依附——学乾之担当 |
| 职业瓶颈 | 问：我是该「健」起来，还是该「顺」下来？ |

**口诀**：乾健坤顺，相资而成；亢极为悔，黄裳元吉。

## 三、时与位 — 待时而动

<figure class="diagram-figure">
<img src="{{ '/assets/images/zhouyi/03-shi-wei.svg' | relative_url }}" alt="时位示意图" />
<figcaption>图 3 — 得时、得位、中正：君子藏器于身，待时而动</figcaption>
</figure>

### 语境要点

- **时**：时机。同样一件事，在初九可行，在上六可能凶。
- **位**：爻位。初、二、三、四、五、上——各有贵贱远近；二、五多为中正之位（见第一节「六爻」表）。
- 「君子藏器于身，待时而动」——有能力不急于表现，时机未到宜潜。
- 泰（天地交）与否（天地不交）——同一结构，时不同则吉凶迥异。

### 代表卦义

| 卦 | 要义 | 启示 |
|----|------|------|
| 屯 | 草创艰难 | 初创期宜建侯，不宜躁进 |
| 需 | 云上于天，有孚 | 等待时机，有信则吉 |
| 谦 | 地中有山 | 有实而谦，天道助之 |
| 豫 | 雷出地奋 | 和乐之时，亦防骄逸 |
| 泰 | 小往大来 | 通泰之时，思所以持 |
| 否 | 大人否亨 | 闭塞之时，守正待变 |

### 案例

**诸葛亮隆中对**：刘备三顾时，曹操已强、孙权已据——诸葛亮未劝立即北伐，而先取荆州、益州，联吴抗曹。正是**识时、守位、待变**的典范：不在错误时机做正确的事。

### 用法

| 场景 | 时位术 |
|------|--------|
| 职业跳槽 | 问：是「需」卦（宜等），还是「遁」卦（宜退）？ |
| 产品发布 | 市场未熟则屯；势已成则豫——辨阶段 |
| 谈判 | 二爻得中：不争锋，先找双方可接受之中点 |
| 投资 | 泰时防否；否时寻变——周期思维 |

**口诀**：藏器待时，中正为吉；泰否相推，循环有常。

## 四、变易之道 — 穷则变，变则通

<figure class="diagram-figure">
<img src="{{ '/assets/images/zhouyi/04-bian-yi.svg' | relative_url }}" alt="变易示意图" />
<figcaption>图 4 — 本卦、动爻、变卦：穷则变，变则通，通则久</figcaption>
</figure>

### 语境要点

- 《系辞》：「易穷则变，变则通，通则久。」——僵局时，变化是出路。
- **变卦**：六爻中动爻变阴阳，成另一卦——示「事态若变，趋势如何」。
- 「不易」：变化中有常道——仁义、中正、诚敬不因时势而弃。
- 吉凶悔吝四辞：吉（顺）、凶（逆）、悔（可改）、吝（难前）——悔优于吝，知悔则有机会。

### 名句

| 原文 | 要义 |
|------|------|
| 易穷则变，变则通，通则久 | 变革是破局之道 |
| 君子以思患而豫防之 | 忧患意识，未雨绸缪 |
| 无平不陂，无往不复 | 盛衰互见，循环往复 |
| 履霜，坚冰至 | 见微知著，防微杜渐 |

### 案例

**商鞅变法**：秦国积弱，变法则通，守旧则穷。周易之变易，在此是**结构性调整**而非表面修补。然变亦需「时」与「位」——秦孝公在位、商鞅得权，方有成；无此条件，变则凶。

### 用法

| 场景 | 变易术 |
|------|--------|
| 组织转型 | 辨：需变结构（变卦），还是需调节奏（守本卦）？ |
| 个人困境 | 问：何为「履霜」之微兆？何者可变、何者不可变？ |
| 危机管理 | 悔则改、吝则止——先止失血，再谋通途 |
| 战略复盘 | 画「本卦→变卦」：若坚持现状 vs 若调整策略，各走向何方？ |

**口诀**：穷则变，通则久；履霜知冰，思患豫防。

## 五、观象系辞 — 占断与决策

<figure class="diagram-figure">
<img src="{{ '/assets/images/zhouyi/05-juece.svg' | relative_url }}" alt="决策三问示意图" />
<figcaption>图 5 — 势在何处、时宜否、何者可变</figcaption>
</figure>

### 语境要点

- 《系辞》：「圣人设卦观象，系辞焉而明吉凶。」——占筮的本质是**借象明理**，非机械求签。
- 传统占法：大衍筮法、**铜钱卦（金钱卦）**等，得卦后读卦辞、爻辞、象传。
- 现代读法：可不拘于起卦形式，而以六十四卦为**六十四种情境模型**来对照当下。
- 孔子「述而不作」于易，重在教人修德：「善不积不足以成名，恶不积不足以灭身。」

### 铜钱用法（金钱卦）

最普及的起卦法：三枚铜钱摇六次，得六爻，自下而上叠成一卦。重在**专心所问、一次一事**，摇完再读辞省思。

#### 备物与约定

| 项目 | 说明 |
|------|------|
| **铜钱** | 三枚形制相同（古钱、硬币皆可） |
| **阳面** | 有花纹/满文/国徽一侧，记为「背」，属**阳** |
| **阴面** | 有汉字面值一侧，记为「字」，属**阴** |
| **坐姿** | 双掌合扣铜钱，默念所问之事，感而遂通后再摇 |

#### 一爻之四种结果

每摇一次，据三枚铜钱**背面（阳面）朝上**的枚数定爻：

| 背面数 | 名称 | 数值 | 爻象 | 是否动 |
|--------|------|------|------|--------|
| 3 | 老阳 | 9 | 阳爻 `—` | **动**（阳变阴） |
| 2 | 少阳 | 7 | 阳爻 `—` | 静 |
| 1 | 少阴 | 8 | 阴爻 `- -` | 静 |
| 0 | 老阴 | 6 | 阴爻 `- -` | **动**（阴变阳） |

- 动爻可在旁作记号：老阳旁标 `○`，老阴旁标 `×`（或改用变后之爻）。
- 连摇 **六次**，第一次为**初爻**，第六次为**上爻**——切勿颠倒次序。

#### 成卦与读卦步骤

1. **写本卦**：自下而上画出六爻，定卦名（下卦三爻 + 上卦三爻 → 查六十四卦表）。
2. **标动爻**：凡 6、9 者为动爻，记下位置（如「九三动」）。
3. **成变卦**（若有动爻）：动则阴阳互变，静爻不动，得另一卦，示趋势或后果。
4. **读辞**：
   - 无动爻 → 以**本卦卦辞**为主；
   - 一爻动 → 以该**爻辞**为主，卦辞为辅；
   - 多爻动 → 先读卦辞，再读各动爻辞（初学可取动爻最高位或最低位为主，宜择一说而贯）。
5. **参象传**：读该卦《彖传》《大象》《小象》，把占断落实为可行动的判断。

#### 简例

连摇六次得（自下而上）：7、8、9、8、7、8 → 下卦震、上卦坎，为**屯**卦；唯**九三**动（老阳）。

- **本卦屯**：草创艰难，宜建侯而不宜躁进。
- **九三爻辞**：「即鹿无虞，惟入于林中；君子几不如舍，往吝。」示冒进失导之戒。
- **变卦**：九三阳变阴，下卦由震变艮，上卦坎不变 → **山水蒙**卦，示宜启蒙审势、勿妄动。
- 读辞时对照：当下是否处「屯」之阶段？动爻是否正告「往吝」？

#### 占筮宜忌

| 宜 | 忌 |
|----|-----|
| 一事一问，心正意诚 | 同一件事反复占、直到满意为止 |
| 占后读辞、记省思 | 只问吉凶、不究义理 |
| 动爻变卦同看 | 颠倒初、上次序 |

**口诀**：三钱摇六次，自下初爻起；七九为阳、六八为阴，老则动而变。

#### 第三步：标动爻、成变卦

| 项目 | 内容 |
|------|------|
| **动爻** | 九三（老阳 9）→ 阳变阴 |
| **静爻** | 初九、六二、六四、九五、上六 — 不变 |
| **变卦下卦** | 震之九三变阴 → 阳·阴·阴 = **艮** ☶ |
| **变卦上卦** | 坎不变 → 仍为 **坎** ☵ |
| **变卦名** | 艮下坎上 = **蒙**（山水蒙） |

本卦示**当下之势**，变卦示**动而后之趋向**。

#### 第四步：读辞观象

| 文献 | 屯（本卦） | 九三（动爻） | 蒙（变卦） |
|------|------------|--------------|------------|
| **卦辞** | 元亨利贞，勿用有攸往，利建侯 | — | 亨。匪我求童蒙，童蒙求我 |
| **爻辞** | — | 即鹿无虞，惟入于林中；君子几不如舍，往吝 | 初六：发蒙，利用刑人 |
| **大象** | 云雷屯，君子以经纶 | 即鹿无虞，以纵禽也；君子舍之，往吝穷也 | 山下出泉，蒙；君子以果行育德 |
| **要义** | 初创期宜建制，不宜妄动 | **冒进失导，宜止** | 宜启蒙学习，慎于妄行 |

**读辞规则（本例）**：一爻动 → 以**九三爻辞**为主，**屯卦辞**为辅，**蒙卦**参变势。

|------|------|------|------|------|----------|----------|

#### 第六步：综合断语（模板）

按「势 → 时 → 变 → 行」写出可操作的判断：

| 维度 | 屯 · 九三动 → 蒙 | 现代对照 |
|------|------------------|----------|
| **势** | 云雷屯，草创维艰；内动外险 | 项目/事业处初创期，内有余力，外环境未明 |
| **时** | 九三动：不宜有攸往 | 当下**不宜强行推进**，非行动窗口 |
| **变** | 入蒙：须启蒙、学习、立规矩 | 若执意而动，易入「迷蒙」；宜先求明、后求进 |
| **行** | 利建侯；几不如舍 | 先组团队、定章程、找导师；**舍**掉急躁冒进 |

**断语示例**：事处初创，势有可为而时未至。九三动戒「往吝」，当缓进蓄势，先建侯、正名分、明路径；变而为蒙，示须启蒙审势，不宜独闯林中。宜守正待时，以学代躁。

#### 解析自检清单

| # | 自检项 | 本例 |
|---|--------|------|
| 1 | 六爻次序是否初→上？ | ✓ 7-8-9-8-7-8 |
| 2 | 内外卦是否定对？ | ✓ 震下坎上 = 屯 |
| 3 | 动爻是否标清？ | ✓ 仅九三 |
| 4 | 变卦是否推算？ | ✓ 艮下坎上 = 蒙 |
| 5 | 读辞主次是否明确？ | ✓ 九三爻辞为主 |
| 6 | 是否参大象、彖传？ | ✓ 经纶 / 果行育德 |
| 7 | 断语是否可行动？ | ✓ 建侯、勿往、启蒙 |

**口诀**：结构定名先分明，动爻变卦看趋势；卦辞爻辞象传会，势时变行四字成。

### 决策三问（现代用法）

| 问 | 对应易学思维 |
|----|--------------|
| **势在何处？** | 当前像哪一卦？主象是什么？ |
| **时宜否？** | 当动还是当守？得位吗？ |
| **何者可变？** | 动哪一「爻」？底线在哪里？ |

### 案例

**禹贡治水**：堵则溃，疏则通。大禹改「堵」为「疏」，正是变易思维。卦象中坎水本险，识其性而导之，则险可为用。

### 用法

| 场景 | 观象术 |
|------|--------|
| 重大决策 | 写下现状卦象（如「困」），再问变爻后趋势 |
| 每日省思 | 随机读一卦爻辞，对照今日一事 |
| 团队讨论 | 用卦名做隐喻：「我们现在更像『谦』还是『骄』？」 |
| 与塔罗对照 | 周易重**时势结构**；塔罗重**内在象征**——可互补 |

**口诀**：观象系辞，明吉凶；三问既明，再行动。

## 阅读边界

读《周易》宜：

1. **重理不泥占** — 卦爻辞是情境智慧，非命运判决书
2. **知变知常** — 变易中守仁义中正之「不易」
3. **勿以吉凶论人** — 凶卦多含警示与转机，吉卦多含持守之诫
4. **版本与注解** — 王弼注、朱熹《周易本义》、高亨注等，各有侧重，宜择一主读
