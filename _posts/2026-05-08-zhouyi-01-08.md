## 经传 1–8（乾至比）

### 1 · 乾 ䷀

**卦象**：乾上乾下

**卦辞**

> 元亨利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 潜龙，勿用。 |
| 九二 | 见龙在田，利见大人。 |
| 九三 | 君子终日乾乾，夕惕若，厉无咎。 |
| 九四 | 或跃在渊，无咎。 |
| 九五 | 飞龙在天，利见大人。 |
| 上九 | 亢龙有悔。 |
| 用九 | 见群龙无首，吉。 |

**要义**：自强不息，亢则有悔

---

### 2 · 坤 ䷁

**卦象**：坤上坤下

**卦辞**

> 元亨，利牝马之贞。君子有攸往，先迷，后得主，利。西南得朋，东北丧朋。安贞吉。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 履霜，坚冰至。 |
| 六二 | 直方大，不习无不利。 |
| 六三 | 含章可贞；或从王事，无成有终。 |
| 六四 | 括囊，无咎无誉。 |
| 六五 | 黄裳，元吉。 |
| 上六 | 龙战于野，其血玄黄。 |
| 用六 | 利永贞。 |

**要义**：厚德载物，顺而不迷

---

### 3 · 屯 ䷂

**卦象**：坎上震下

**卦辞**

> 元亨，利贞；勿用有攸往，利建侯。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 磐桓；利居贞，利建侯。 |
| 六二 | 屯如，邅如，乘马班如。匪寇婚媾。女子贞不字，十年乃字。 |
| 六三 | 即鹿无虞，惟入于林中，君子几，不如舍，往吝。 |
| 六四 | 乘马班如，求婚媾，往吉，无不利。 |
| 九五 | 屯其膏，小贞吉，大贞凶。 |
| 上六 | 乘马班如，泣血涟如。 |

**要义**：草创维艰，宜建侯

---

### 4 · 蒙 ䷃

**卦象**：艮上坎下

**卦辞**

> 亨。匪我求童蒙，童蒙求我。初筮告，再三渎，渎则不告。利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 发蒙，利用刑人，用说桎梏；以往吝。 |
| 九二 | 包蒙，吉。纳妇，吉；子克家。 |
| 六三 | 勿用取女，见金夫，不有躬，无攸利。 |
| 六四 | 困蒙，吝。 |
| 六五 | 童蒙，吉。 |
| 上九 | 击蒙；不利为寇，利御寇。 |

**要义**：启蒙教化，果行育德

---

### 5 · 需 ䷄

**卦象**：坎上乾下

**卦辞**

> 有孚，光亨，贞吉，利涉大川。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 需于郊，利用恒，无咎。 |
| 九二 | 需于沙，小有言；终吉。 |
| 九三 | 需于泥，致寇至。 |
| 六四 | 需于血，出自穴。 |
| 九五 | 需于酒食，贞吉。 |
| 上六 | 入于穴，有不速之客三人来；敬之，终吉。 |

**要义**：有孚光亨，利涉大川

---

### 6 · 讼 ䷅

**卦象**：乾上坎下

**卦辞**

> 有孚，窒惕，中吉；终凶。利见大人，不利涉大川。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 不永所事，小有言，终吉。 |
| 九二 | 不克讼，归而逋，其邑人三百户，无眚。 |
| 六三 | 食旧德，贞厉，终吉；或从王事，无成。 |
| 九四 | 不克讼；复即命，渝，安贞，吉。 |
| 九五 | 讼，元吉。 |
| 上九 | 或锡之鞶带，终朝三褫之。 |

**要义**：不可终讼，宜和

---

### 7 · 师 ䷆

**卦象**：坤上坎下

**卦辞**

> 贞，丈人吉，无咎。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 师出以律，否臧凶。 |
| 九二 | 在师中吉，无咎；王三锡命。 |
| 六三 | 师或舆尸，凶。 |
| 六四 | 师左次，无咎。 |
| 六五 | 田有禽，利执言，无咎；长子帅师，弟子舆尸，贞凶。 |
| 上六 | 大君有命，开国承家，小人勿用。 |

**要义**：容民畜众，贞丈人吉

---

### 8 · 比 ䷇

**卦象**：坎上坤下

**卦辞**

> 吉。原筮，元永贞，无咎。不宁方来，后夫凶。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 有孚比之，无咎；有孚盈缶，终来有它，吉。 |
| 六二 | 比之自内，贞吉。 |
| 六三 | 比之匪人。 |
| 六四 | 外比之，贞吉。 |
| 九五 | 显比；王用三驱，失前禽。邑人不诫，吉。 |
| 上六 | 比之无首，凶。 |

**要义**：亲比团结，先王以建万国
