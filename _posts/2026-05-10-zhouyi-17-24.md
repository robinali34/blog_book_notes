## 经传 17–24（随至复）

### 17 · 随 ䷐

**卦象**：兑上震下

**卦辞**

> 元亨，利贞，无咎。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 官有渝，贞吉；出门交有功。 |
| 六二 | 系小子，失丈夫。 |
| 六三 | 系丈夫，失小子；随有求得，利居贞。 |
| 九四 | 随有获，贞凶；有孚在道，以明，何咎？ |
| 九五 | 孚于嘉，吉。 |
| 上六 | 拘系之，乃从维之；王用亨于西山。 |

**要义**：随时而动，元亨

---

### 18 · 蛊 ䷑

**卦象**：艮上巽下

**卦辞**

> 元亨，利涉大川。先甲三日，后甲三日。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 干父之蛊，有子，考无咎，厉终吉。 |
| 九二 | 干母之蛊，不可贞。 |
| 九三 | 干父之蛊，小有悔，无大咎。 |
| 六四 | 裕父之蛊，往见吝。 |
| 六五 | 干父之蛊，用誉。 |
| 上九 | 不事王侯，高尚其事。 |

**要义**：整饬腐败，先甲后甲

---

### 19 · 临 ䷒

**卦象**：坤上兑下

**卦辞**

> 元亨，利贞；至于八月，有凶。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 咸临，贞吉。 |
| 九二 | 咸临，吉，无不利。 |
| 六三 | 甘临，无攸利；既忧之，无咎。 |
| 六四 | 至临，无咎。 |
| 六五 | 知临，大君之宜，吉。 |
| 上六 | 敦临，吉，无咎。 |

**要义**：教思无穷，容保民

---

### 20 · 观 ䷓

**卦象**：巽上坤下

**卦辞**

> 盥而不荐，有孚颙若。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 童观，小人无咎，君子吝。 |
| 六二 | 窥观，利女贞。 |
| 六三 | 观我生，进退。 |
| 六四 | 观国之光，利用宾于王。 |
| 九五 | 观我生，君子无咎。 |
| 上九 | 观其生，君子无咎。 |

**要义**：观国之光，省方观民

---

### 21 · 噬嗑 ䷔

**卦象**：离上震下

**卦辞**

> 亨，利用狱。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 屦校灭趾，无咎。 |
| 六二 | 噬肤灭鼻，无咎。 |
| 六三 | 噬腊肉，遇毒；小吝，无咎。 |
| 九四 | 噬干胏，得金矢；利艰贞，吉。 |
| 六五 | 噬干肉，得黄金，贞厉，无咎。 |
| 上九 | 何校灭耳，凶。 |

**要义**：明罚敕法，利用狱

---

### 22 · 贲 ䷕

**卦象**：艮上离下

**卦辞**

> 亨，小利有攸往。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 贲其趾，舍车而徒。 |
| 六二 | 贲其须。 |
| 九三 | 贲如，濡如，永贞吉。 |
| 六四 | 贲如，皤如，白马翰如，匪寇，婚媾。 |
| 六五 | 贲于丘园，束帛戋戋；吝，终吉。 |
| 上九 | 白贲，无咎。 |

**要义**：文饰有度，小利有攸往

---

### 23 · 剥 ䷖

**卦象**：艮上坤下

**卦辞**

> 不利有攸往。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 剥床以足，蔑贞；凶。 |
| 六二 | 剥床以辨，蔑贞凶。 |
| 六三 | 剥之，无咎。 |
| 六四 | 剥床以肤，凶。 |
| 六五 | 贯鱼以宫人宠，无不利。 |
| 上九 | 硕果不食，君子得舆，小人剥庐。 |

**要义**：剥落衰微，不利有攸往

---

### 24 · 复 ䷗

**卦象**：坤上震下

**卦辞**

> 亨。出入无疾，朋来无咎。反复其道，七日来复，利有攸往。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 不远复，无祗悔，元吉。 |
| 六二 | 休复，吉。 |
| 六三 | 频复，厉无咎。 |
| 六四 | 中行独复。 |
| 六五 | 敦复，无悔。 |
| 上六 | 迷复，凶，有灾眚。用行师，终有大败；以其国，君凶，至于十年不克征。 |

**要义**：反复其道，七日来复
