**Counselling for Toads**（《蛤蟆先生去看心理医生》）  
罗伯特·戴博德（Robert de Board）· 借《柳林风声》角色讲**交流分析（Transactional Analysis, TA）** · 核心：**识别自我状态，才能停止自动反应、建立成人边界**

## 全书概览

埃里克·伯恩（Eric Berne, 1910—1970）创立**交流分析（TA）**：用通俗语言描述人在沟通中的「自我状态」切换，以及关系中的「心理游戏」。戴博德以童话外壳写蛤蟆先生抑郁、求助、十次咨询、走向自主——是欧美世界**最畅销的心理自助入门之一**。

故事线：蛤蟆陷入深度抑郁，在朋友建议下见苍鹭咨询师。十次会谈中，他逐步理解：情绪不是外界「强加」的，而是自己对事件的**解读与状态选择**；童年形成的「人生剧本」可以被成人自我改写。

> 没有人能让我们感受什么，除非我们配合。  
> 情绪是自我状态切换的信号。

读本书，宜记三问：**此刻我是父母、成人还是儿童？我在玩什么心理游戏？我能否用 Adult 重新选择？**

## 一、三种自我状态（PAC 模型）

### 语境要点

人在任一时刻主要处于三种自我状态之一（或混合）：

| 状态 | 来源 | 典型表现 | 边界风险 |
|------|------|----------|----------|
| **父母 Parent** | 内化的父母/权威 | 批评、应该、说教、控制 | 越界管教他人 |
| **成人 Adult** | 当下信息处理 | 事实、选择、负责 | **健康边界的主体** |
| **儿童 Child** | 童年经验 | 讨好、叛逆、撒娇、恐惧 | 退缩或讨好，难说「不」 |

```text
刺激 → 常先进入 Child/Parent（自动）→ 若觉察 → 切换 Adult 回应
```

### 深度解析

PAC 不是「三种人格」，而是**此刻的功能模块**。父母状态也有 nurturing（养育型）与 critical（批判型）；儿童状态有 natural（自然）、adapted（适应）、rebellious（叛逆）等分支。

```text
Critical Parent：「你怎么又搞砸？」→ 内在批评，边界崩塌
Nurturing Parent：「没事我来帮你做」→ 过度代劳，越界
Adapted Child：「好的好的我都听」→ 讨好，难拒绝
Free Child：冲动、发泄 → 可能伤人
Adult：「事实是什么？我选择什么？」→ 边界与负责
```

与 [DBT 正念]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }})：觉察「此刻状态」是正念的人际版；与 [传习录]({{ '/2026/06/17/chuanxilu-wang-yangming.html' | relative_url }})「省察克治」：看见状态切换即省察入口。

### 名句或概念表

| 表述 | 要义 |
|------|------|
| 没有人能让我们感受什么，除非我们配合 | 强调解读与选择空间（非否认压迫） |
| 情绪是信号 | 提示状态切换，非最终真理 |
| 咨询目标是成人自主 | 内化 Adult，非依赖咨询师 |

### 典型案例（人物或情境）

**蛤蟆的抑郁开场**  
蛤蟆哭泣、无力、不想活——高度 Child 状态。苍鹭不急于给建议，先建立安全框架：**咨询是蛤蟆为自己的改变付费**，责任在来访者。这是 Adult 关系的底色。

**苍鹭的「我完了」游戏**  
当蛤蟆说「我完了」时，若对方进入拯救型 Parent，游戏继续。苍鹭常问：「谁的责任？你的感受从哪来？」——邀请 Adult 上线。

### 用法与场景表

| 场景 | 易入状态 | Adult 起手 |
|------|----------|------------|
| 被领导批评 | Adapted Child 或 Rebel Child | 核对事实、问清期待、定改进行动 |
| 孩子哭闹 | Critical Parent 升级 | 先稳定自己 Adult，再设定界限 |
| 伴侣指责 | Parent 对 Parent 大战 | 暂停，描述事实与需要 |
| 朋友倾诉 | Nurturing Parent 过度给方案 | 倾听 + 问「你需要建议还是陪伴？」 |
| 医生/办事窗口 | Adapted Child 不敢问 | 「请再说明一下选项」 |
| 网约车/客服争执 | Rebel Child 爆发 | 「描述问题 + 我要的解决」 |
| 邻居噪音 | Parent 说教或 Child 忍 | 「事实 + 请求 + 后果」 |
| 家长会老师反馈 | Child 羞耻 | 「具体行为是什么？我们怎么配合？」 |
| 甲方反复改需求 | 讨好 Child | 「变更入排期，请确认优先级」 |
| 长辈当众数落 | Child 僵住 | 「私下谈我可以听，当众我不回应」 |

### PAC 混合态识别

现实中常是 **混合态**，而非单一 P/A/C：

| 混合 | 表现 | 拆解 |
|------|------|------|
| CP + AC | 自责「你应该更努力」后乖乖加班 | 先停 CP 内在批评，再 Adult 排期 |
| NP + FC | 过度照顾他人后突然爆发 | 边界前置，防 FC 反弹 |
| CP + RC | 挑剔他人后叛逆甩锅 | 回到事实，分责任 |
| A 薄层 + C 底 | 表面理性，内里恐慌 | 命名 Child 恐惧，再续 Adult 对话 |

```text
觉察口诀：身体紧/想讨好 → 疑 Child；想教训人 → 疑 Parent；问「事实与选择」→ 邀 Adult
```

### 反模式

- PAC 当作贴标签羞辱：「你很 Child」——纵向关系，违背 TA 精神  
- 追求「永远 Adult」：情绪儿童需要被看见，非消灭  
- 用公式否认不公：「你配合了才被 PUA」——结构性伤害须识别

## 二、人生坐标 — 从「我不好」到「我好，你也好」

### 语境要点

伯恩用四象限描述人对自我与他人的基本信念：

| 坐标 | 信念 | 关系模式 |
|------|------|----------|
| 我不好，你好 | 自卑、讨好 | 难拒人，易受伤 |
| 我好，你不好 | 攻击、指责 | 用愤怒当边界，伤关系 |
| 我不好，你不好 | 绝望、抑郁 | 放弃沟通 |
| **我好，你也好** | 尊重自己与他人 | **边界 + 联结** |

蛤蟆早期多落在「我不好，你好」——与 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }}) 的认可欲求同构：用讨好换存在资格。

### 深度解析

坐标不是固定人格，而是**可滑动的默认倾向**。压力下从「我好你也好」滑向「我好你不好」（攻击）或「我不好你好」（讨好）极常见——**觉察滑动比维持完美更重要**。

```text
讨好 ←→ 攻击
  ↘     ↙
   抑郁（双不好）
        ↑
    成人目标：我好你也好
```

阿德勒的**共同体感觉**与「我好你也好」共振：不是赢过他人，而是在整体中贡献与联结。

### 典型案例（人物或情境）

**职场新人小周**  
被 senior 当众纠正，立刻「我不好你好」：连连道歉、加班补救。坐标滑动到 Adult：「感谢指出，我会改 X；Y 部分我的理解是……请确认。」

**父母指责「白眼狼」**  
子女瞬间「我不好」或反弹「你好我才不好」。Adult：「我听到你的失望。赡养我会承担我的部分，方式需要商量。」

### 用法与场景表

| 触发 | 常见滑向 | 拉回 Adult |
|------|----------|------------|
| 拒绝别人 | 我不好 | 重申：拒绝一事不否定我整个人 |
| 被冒犯 | 我好你不好 | 设边界句，非报复 |
| 项目失败 | 我不好你不好 | 复盘事实，区分一次失败与自我价值 |
| 亲密争吵 | 双向滑动 | 暂停 + 状态命名 |

### 反模式

- 「我好你也好」当作无原则妥协  
- 用坐标评判他人：「你是 narcissist」  
- 忽视「双不好」抑郁需专业帮助

## 三、情绪公式与人生剧本

### 语境要点

- **情绪公式**：情绪 = 事件 + 信念/解读（类似认知行为疗法 ABC）。
- **人生剧本**：童年为求生存与发展，无意识写下「人生会如何如何」；成人可觉察并改写。

### 深度解析

蛤蟆的剧本可能包含：「我必须逗大家笑才值得被爱」「强者不会脆弱」。抑郁是旧剧本崩裂的信号——**痛苦有时是改写前的必经**。

```text
事件：朋友批评蛤蟆
旧信念：「我不够好」
情绪：羞耻、愤怒
行为：讨好或爆发
新 Adult 链：事实 → 多种解释 → 选择回应
```

与 [四镜头分析]({{ '/2026/05/25/four-lenses-knowing-choosing.html' | relative_url }})：重大剧本改写（换城、换行、离婚）用多镜头；日常用情绪公式即可。

### 名句或概念表

| 概念 | 说明 |
|------|------|
| 脚本 Script | 无意识的人生计划 |
| 脚本重写 | Adult 意识下的新选择 |
| 安抚 Child | 不是纵容，是承认旧痛 |

### 典型案例

**「必须强大」剧本**  
高管蛤蟆式人物崩溃于一次公开失误——剧本不允许脆弱。咨询式自问：「谁规定我不能犯错？童年谁惩罚了脆弱？」然后练小步暴露：向信任同事承认一次小失误。

### 用法与场景表

| 剧本台词 | 来源猜测 | 改写方向 |
|----------|----------|----------|
| 「我不能让人失望」 | 讨好型家庭 | 有限承诺 + 清晰拒绝 |
| 「愤怒才安全」 | 被欺负史 | 边界句 + 离开现场 |
| 「我注定孤独」 | 早期忽视 | 小步社交实验 |
| 「只有成绩好才配」 | 成绩导向 | 价值多元清单 |

### 反模式

- 剧本论变成「全是父母的错」—— Adult 负责当下改写  
- 不行动只分析：剧本觉察须配行为实验  
- 用剧本为伤害他人开脱

## 四、咨询十讲 — 过程映射与自用

### 语境要点

全书约十次会谈，象征咨询阶段：建立联盟 → 探索童年与状态 → 处理愤怒与悲伤 → 走向自主告别。

| 阶段 | 主题 | 可带走 |
|------|------|--------|
| 初访 | 抑郁、无力、求助羞耻 | 允许求助不是软弱 |
| 早期 | 儿童状态、讨好、哭泣 | 看见「自动模式」 |
| 中期 | 愤怒、父母状态、父亲记忆 | 愤怒常是防御，非真边界 |
| 中后期 | 人生坐标、剧本觉察 | 从「我不好」松动 |
| 后期 | 成人抉择、计划生活 | 自己选人生剧本 |
| 结束 | 告别、独立 | 咨询师退出 = 内化 Adult |

### 深度解析

苍鹭的技法摘要（自学可模拟）：

| 技法 | 作用 |
|------|------|
| 澄清责任 | 「谁为你付费？谁要改变？」 |
| 状态命名 | 「此刻你像几岁？」 |
| 循环打断 | 不玩「我抓到你了」游戏 |
| _homework_ | 记录一周情绪与状态 |
| 告别仪式 | 标记成长节点 |

无咨询师时：用 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }}) 周记 + 信任友人作「见证人」，模拟 Adult 对话——**不能替代危机干预**。

### 十讲章回拆解（逐讲笔记）

| 讲次 | 蛤蟆状态 | 苍鹭焦点 | 读者可带走 |
|------|----------|----------|------------|
| **1** | 抑郁、羞耻求助 | 建立框架：谁付费、谁改变 | 求助是勇气，非软弱 |
| **2** | 哭泣、讲童年 | 引入儿童状态、情绪许可 | 情绪可命名，不必立刻「修好」 |
| **3** | 讨好、怕得罪 | 人生坐标初探：我不好—你好 | 看见自动讨好模式 |
| **4** | 愤怒萌芽 | 父母状态、父亲形象 | 愤怒有信息，表达可选 |
| **5** | 防御、借口 | 情绪公式：事件 + 解读 | 写下信念，非只怪事件 |
| **6** | 回忆创伤片段 | 人生剧本、早期决定 | 「谁写的剧本？」自问 |
| **7** | 与獾/权威冲突 | Critical Parent 识别 | 尊重 ≠ 放弃边界 |
| **8** | 悲伤、失落 | 安抚 Child、哀悼 | 脆弱可存在，非耻辱 |
| **9** | 计划生活、选路 | Adult 抉择、责任回归 | 自己选结局，非等拯救 |
| **10** | 告别、轻盈 | 内化咨询师 = 内化 Adult | 关系结束是成长节点 |

**阅读提示**：不必死记讲次，宜问「这一讲蛤蟆从哪个坐标滑向哪个？」——十讲是**抑郁 → 觉察 → 愤怒 → 剧本 → 自主**的压缩模型。

### 典型案例

**蛤蟆学愤怒**  
蛤蟆曾把愤怒当禁忌，后学表达：愤怒指向**边界被犯**，但表达方式可选 Adult。与「我好你不好」的攻击不同：**边界是陈述，攻击是贬低**。

### 反模式

- 期待咨询师/书籍「给答案」——改变在来访者行动  
- 把十次会谈当作唯一标准疗程——真实咨询长短因人而异  
- 自学咨询替代危机干预

## 五、心理游戏与沟通陷阱

### 语境要点

**心理游戏**：双方无意识配合的重复剧本，短期有「收获」（如道德优越感、被照顾感），长期损害关系。经典三步：**诱饵 → 混乱 → 结局（常带「我抓到你了」）**。

### 深度解析

| 游戏名（TA 经典） | 诱饵 | 结局 |
|-------------------|------|------|
| 是的，但是… | 求建议 | 否决一切，证明无解 |
| 我抓到你了 | 犯错 | 指责者获胜 |
| 可怜的我 | 诉苦 | 维持无助 |
| 笨蛋 | 挑衅 | 爆发证明对方对 |

职场版「是的但是」：会议里求方案又否决所有——可 Adult 问：「你倾向哪个约束下的方案？」

### 经典游戏深度拆解

| 游戏 | 阶段 1 诱饵 | 阶段 2 混乱 | 阶段 3 结局 | 你常扮演的角色 |
|------|-------------|-------------|-------------|----------------|
| **是的，但是…** | 「帮我想想怎么办」 | 否决每个建议 | 「没办法」→ 求助者赢「无解」 | Rescuer 或无助者 |
| **我抓到你了** | 故意犯错/试探 | 对方揪错 | 「我就知道你不靠谱」 | 检察官或笨蛋 |
| **可怜的我** | 诉苦、装无助 | 他人忙碌拯救 | 继续无助，证明世界残酷 | 受害者 |
| **笨蛋** | 挑衅、迟到 | 被挑衅者爆炸 | 「看，你脾气真差」 | 挑衅者或上钩者 |
| **踢我吧** | 自我贬低求肯定 | 对方安慰 | 短暂好感受，问题不解决 | 求安抚的 Child |
| **法庭** | 罗列对方罪状 | 反诉 | 无人赢，只有累 | Parent 对 Parent |

```text
游戏打断三步：① 命名「我们在玩 ___」② 问「你要什么结果」③ 转 Adult 交易（事实—选择—后果）
```

**案例：夫妻「谁先说话」**  
冷战三天，双方都在等对方低头——实为 **踢我吧 + 我抓到你了** 混合。一方 Adult：「我们在冷战游戏里，今晚 8 点谈 20 分钟，谈不完明天继续，但不冷战。」

### 中国语境游戏变体

| 变体 | 诱饵 | 结局 | Adult 打断 |
|------|------|------|------------|
| 「我都是为你好」 | 关心包装控制 | 子女内疚服从 | 「我听到关心，决定我自己做」 |
| 「别人都孝顺」 | 羞耻比较 | 子女过度补偿 | 「每家情况不同」 |
| 「就你特殊」 | 激将 | 证明给对方看 | 「我不按激将做决定」 |
| 饭局劝酒 | 面子 | 喝醉或伤感情 | 「我不喝，用茶敬你」 |
| 群聊道德绑架 | 不转不是人 | 表演善良 | 「不参与链式转发」 |

### 用法与场景表

| 场景 | 游戏 | 打断 |
|------|------|------|
| 夫妻冷战 | 谁先说话谁输 | 命名游戏，提议暂停规则 |
| 亲子作业 | 可怜的我 vs 暴躁父母 | 后果归还 + 固定作业时间 |
| 团队推责 | 笨蛋游戏 | 对事复盘，非抓罪人 |
| 社交抱怨 | 共生诉苦 | 听一次，问「你下一步要什么」 |

### 反模式

- 到处贴「你在玩游戏」——关系更炸  
- 只打断别人游戏，不看看自己配合了什么

## 六、愤怒、抑郁与「假边界」

### 语境要点

- **抑郁**常伴「我不好你不好」或深度 Child 无力。
- **愤怒**有时是 Adult 边界的信号，有时是 Child 的反弹或 Parent 的攻击。
- **假边界**：用攻击、冷战、道德绑架当边界——实为游戏。

### 深度解析

```text
真边界（Adult）：陈述事实 + 明确选择 + 承担后果
假边界（Parent/Child）：威胁、羞辱、冷战、报复
```

[不懂拒绝]({{ '/2026/05/21/art-of-saying-no-refusal.html' | relative_url }}) 练的是 Adult 拒绝；[DBT FAST]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }}) 的 **Fair / Truthful** 防假边界变攻击。

### 典型案例

**蛤蟆与獾的权威**  
獾代表 Critical Parent 压力；蛤蟆一度服从，后学在尊重中保持 Adult 选择——**尊重 ≠ 放弃边界**。

### 用法与场景表

| 表现 | 可能状态 | 调适 |
|------|----------|------|
| 闷声抑郁 | Child 无力 | 专业评估 + 小步行动 |
| 爆炸骂人 | Child/Rebel | 暂停 + TIP（见 DBT） |
| 冷暴力 | 被动攻击 | 命名 + 直接请求 |
| 讲道理说教 | Critical Parent | 问事实与需要 |

### 反模式

- 压抑一切愤怒 → 抑郁或突然爆发  
- 把攻击当「真性情」  
- 抑郁只靠自助书不就医

## 七、现代职场情境库

### 语境要点

职场触发 PAC 高频：上级像父母、绩效像成绩、同事竞争像兄弟姐妹脚本。

### 用法与场景表

| 情境 | 状态陷阱 | Adult 回应 |
|------|----------|------------|
| 不敢拒绝加活 | Adapted Child | 「本周排期已满，可下周接。」 |
| 领导发火 | Child 恐惧 / Parent 反击 | 「请具体指出问题，我记下了。」 |
| 抢功 | Rebel Child 报复 | 邮件留痕，陈述贡献 |
| 绩效 C | 我不好 | 要事实标准 + 改进计划 |
| 团建强迫 | 讨好 | 「那天我有私人安排，不参加。」 |
| 被 PUA | 双不好 | 记录、边界、必要时换环境 |

### 典型案例

**「好好先生」工程师**  
每次需求都说好，Child 怕「被抛弃」。十次会谈式自问：  
1. 此刻几岁？（像 8 岁怕老师）  
2. 事实：容量多少？  
3. 选择：协商优先级。  
配合 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }}) 课题分离。

### 反模式

- 用 TA  jargon 指责同事  
- Adult 当作无情机器人  
- 有毒环境只练心态

## 八、现代家庭与亲密关系情境库

### 语境要点

家庭是 Parent/Child 脚本的重演场：父母一句话回到童年，子女一句顶嘴触发 Parent。

### 用法与场景表

| 情境 | 常见模式 | Adult 方向 |
|------|----------|------------|
| 催婚催生 | Critical Parent | 「我听到了。人生节奏我自己负责。」 |
| 婆媳同盟/对立 | 三角游戏 | 夫妻 Adult 联盟对外 |
| 育儿焦虑 | Nurturing Parent 过度 | 后果给孩子，父母守界限 |
| 借钱亲戚 | 讨好 Child | 清晰拒绝 + 可选其他帮助 |
| 伴侣冷战 | 游戏：谁先低头 | 提议谈话规则 |

### 与 [亲密关系]({{ '/2026/06/10/intimate-relationships-moon.html' | relative_url }}) 联读

蜜月期易 Parent  nurturing 过度或 Child 融合；幻灭期宜识别「我把伴侣当父母/孩子」的脚本，重启 Adult 契约。

### 典型案例

**「为你好」母亲**  
女儿买贵包，母亲 Critical Parent 爆发。女儿 Adult：「我知道你在担心钱。这是我的消费选择，我会负责账单。」——非 Rebel Child 对吼。

### 反模式

- Adult 当作冷漠理性  
- 在虐待关系中单方面「修好自己」——安全优先  
- 用 TA 贴标签家庭全员

## 十、八周练习计划

| 周 | 主题 | 每日/每周动作 | 复盘问题 |
|----|------|---------------|----------|
| 1 | PAC 觉察 | 每天 1 次事后标 P/A/C | 哪个状态最常自动上线？ |
| 2 | 内在父母 | 记录一句 Critical Parent 自语 | 这是谁的声音？ |
| 3 | 内在儿童 | 记录一次讨好/恐惧 | 像几岁的我？ |
| 4 | Adult 重写 | 选 1 句边界话用 Adult 语气说 | 身体感受如何？ |
| 5 | 人生坐标 | 画本周滑动曲线 | 何时从「都好」滑走？ |
| 6 | 情绪公式 | 填 3 次「事件—信念—情绪」 | 信念有替代吗？ |
| 7 | 游戏打断 | 识别 1 个重复困境游戏 | 我配合了什么诱饵？ |
| 8 | 剧本一页 | 写「新剧本」一页纸 | 下月第一个小实验？ |

### 练习册模板（精简）

```text
【日期 / 情境】
【事件】
【主要状态 P / A / C】
【信念 / 坐标（我好？你好？）】
【实际行为 · 若 Adult 会如何做】
【结果 · 下周试什么】
```

## FAQ 与常见误读

| 问题 | 简答 |
|------|------|
| 「配合才受伤」是否责怪受害者？ | 原意是保留选择空间；压迫与 PUA 仍须外部支持 |
| 没有咨询师怎么练？ | 日记 + 见证人 + 必要时短程咨询；危机须求助 |
| PAC 会贴标签吗？ | 宜描述「状态」非评判「人格」 |
| 和 CBT 关系？ | 情绪公式与 ABC 相近；TA 更重关系与脚本 |
| 和弗洛伊德？ | TA 更通俗、偏此时此地；不深挖无意识数年 |
| 十次咨询够吗？ | 书是象征；真实疗程因人而异 |
| 儿童读者？ | 故事可读，概念需成人引导 |

**Q8：PAC 和 MBTI 冲突吗？**  
不属同一体系。PAC 是**此刻沟通功能**；MBTI 是倾向描述。宜在冲突现场用 PAC，不用类型贴人。

**Q9：一直 Parent 批评自己怎么办？**  
记一句内在 CP 台词 → 问「这是谁的声音」→ Adult 重写 → 配 [DBT]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }}) 自我慈悲一句。

**Q10：怎么知自己在玩游戏？**  
重复困境 + 熟悉结局 + 短期「赢感」（道德优越、被照顾）——三者齐，疑游戏。

**Q11：十讲读完还是抑郁？**  
书是入门；持续抑郁须评估与治疗。TA 不替代药物与危机干预。

**Q12：续作《蛤蟆先生情绪指南》关系？**  
情绪命名延伸；PAC 与人生坐标仍是核心框架，可并行阅读。

**Q13：职场能用 TA 吗？**  
能，宜对内觉察、对外少 jargon；用 Adult 行为，非贴标签。

**Q14：父母也是「Child」？**  
父母亦有儿童状态；你不必当父母的 Parent，除非你是监护人且对未成年负责。

### 阅读边界

1. **不替代治疗** — 自杀意念、重度抑郁、创伤须专业帮助  
2. **不神话咨询** — 改变发生在会谈之外的行动  
3. **要练不要囤** — 识别状态 ≠ 能维持 Adult  
4. **文化适配** — 英式人物与咨询伦理，宜本土化理解  
5. **与阿德勒/DBT 互补** — 状态觉察 + 课题分离 + 情绪技能
