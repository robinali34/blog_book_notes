## 经传 57–64（巽至未济）

### 57 · 巽 ䷸

**卦象**：巽上巽下

**卦辞**

> 小亨，利有攸往，利见大人。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 进退，利武人之贞。 |
| 九二 | 巽在床下，用史巫纷若，吉，无咎。 |
| 九三 | 频巽，吝。 |
| 六四 | 悔亡，田获三品。 |
| 九五 | 贞吉，悔亡，无不利。无初有终。先庚三日，后庚三日，吉。 |
| 上九 | 巽在床下，丧其资斧，贞凶。 |

**要义**：巽入，申命行事

---

### 58 · 兑 ䷹

**卦象**：兑上兑下

**卦辞**

> 亨，利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 和兑，吉。 |
| 九二 | 孚兑，吉，悔亡。 |
| 六三 | 来兑，凶。 |
| 九四 | 商兑未宁，介疾有喜。 |
| 九五 | 孚于剥，有厉。 |
| 上六 | 引兑。 |

**要义**：和悦，朋友讲习

---

### 59 · 涣 ䷺

**卦象**：巽上坎下

**卦辞**

> 亨，王假有庙，利涉大川，利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 用拯马壮，吉。 |
| 九二 | 涣奔其机，悔亡。 |
| 六三 | 涣其躬，无悔。 |
| 六四 | 涣其群，元吉；涣有丘，匪夷所思。 |
| 九五 | 涣汗其大号，涣王居，无咎。 |
| 上九 | 涣其血去逖出，无咎。 |

**要义**：涣散群险，王假有庙

---

### 60 · 节 ䷻

**卦象**：坎上兑下

**卦辞**

> 亨，苦节，不可贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 不出户庭，无咎。 |
| 九二 | 不出门庭，凶。 |
| 六三 | 不节若，则嗟若，无咎。 |
| 六四 | 安节，亨。 |
| 九五 | 甘节，吉。往有尚。 |
| 上六 | 苦节，贞凶，悔亡。 |

**要义**：节制有度，苦节不可贞

---

### 61 · 中孚 ䷼

**卦象**：巽上兑下

**卦辞**

> 豚鱼吉，利涉大川，利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 虞吉，有他不燕。 |
| 九二 | 鸣鹤在阴，其子和之；我有好爵，吾与尔靡之。 |
| 六三 | 得敌，或鼓或罢，或泣或歌。 |
| 六四 | 月几望，马匹亡，无咎。 |
| 九五 | 有孚挛如，无咎。 |
| 上九 | 翰音登于天，贞凶。 |

**要义**：信发于中，豚鱼吉

---

### 62 · 小过 ䷽

**卦象**：震上艮下

**卦辞**

> 亨，利贞；可小事，不可大事；飞鸟遗之音，不宜上，宜下，大吉。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 飞鸟以凶。 |
| 六二 | 过其祖，遇其妣；不及其君，遇其臣，无咎。 |
| 九三 | 弗过防之，从或戕之，凶。 |
| 九四 | 无咎，弗过遇之；往厉必戒，勿用永贞。 |
| 六五 | 密云不雨，自我西郊；公弋取彼在穴。 |
| 上六 | 弗遇过之；飞鸟离之，凶，是谓灾眚。 |

**要义**：小事可过，不可大事

---

### 63 · 既济 ䷾

**卦象**：坎上离下

**卦辞**

> 亨小，利贞。初吉终乱。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 曳其轮，濡其尾，无咎。 |
| 六二 | 妇丧其茀，勿逐，七日得。 |
| 九三 | 高宗伐鬼方，三年克之；小人勿用。 |
| 六四 | 繻有衣袽，终日戒。 |
| 九五 | 东邻杀牛，不如西邻之禴祭，实受其福。 |
| 上六 | 濡其首，厉。 |

**要义**：初吉终乱，慎终如始

---

### 64 · 未济 ䷿

**卦象**：离上坎下

**卦辞**

> 亨，小狐汔济，濡其尾，无攸利。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 濡其尾，吝。 |
| 九二 | 曳其轮，贞吉。 |
| 六三 | 未济，征凶，利涉大川。 |
| 九四 | 贞吉，悔亡；震用伐鬼方，三年有赏于大国。 |
| 六五 | 贞吉，无悔；君子之光，有孚，吉。 |
| 上九 | 有孚于饮酒；无咎，濡其首，有孚失是。 |

**要义**：事未终，慎辨物居方
