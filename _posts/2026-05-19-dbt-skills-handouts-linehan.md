**DBT Skills Training Handouts and Worksheets**  
*《DBT 技能训练讲义与练习册》*（第二版）  
Marsha M. Linehan · 与《DBT Skills Training Manual》配套 · 核心：**在接纳与改变之间，用可练技能稳住情绪、关系与行动**

## 全书概览

Marsha M. Linehan（1943—）创立**辩证行为疗法（Dialectical Behavior Therapy, DBT）**，最初用于自杀风险高、情绪失调显著的群体，后扩展到抑郁、焦虑、进食障碍、创伤后应激等广泛场景。

本书是**讲义（Handouts）+ 练习册（Worksheets）**合集：不是理论专著，而是**拿来就练**的技能库。与《技能训练手册》（Manual）相对——Manual 给治疗师/教师结构与教法；Handouts & Worksheets 给学员**复印、填写、反复练**。

> People cannot change what they do not acknowledge.  
> 人无法改变自己不愿承认的部分——接纳与改变必须同时存在。

读 DBT，宜记三问：**此刻情绪是什么？技能该用哪一模块？练了有没有真的做（而不只是知道）？**

## DBT 是什么：接纳与改变同在

### 辩证（Dialectics）

| 常见陷阱 | DBT 辩证 |
|----------|----------|
| 只接纳 → 停滞 | 接纳 **且** 改变 |
| 只改变 → 自责、否认 | 承认此刻如此 **且** 学新行为 |
| 非黑即白 | 两极中找「合题」 |

**Wise Mind（智慧心）**：情绪心（Emotion Mind）+ 理性心（Reasonable Mind）的整合——不是压抑情绪，也不是被情绪淹没。

### 生物社会理论（简版）

```text
情绪系统天生敏感（生物）
        +
成长中情绪未获有效回应/被惩罚（社会）
        =
情绪升级快、降得慢、冲动行为多（技能缺口）
```

去羞耻读法：不是「性格坏」，而是**系统 + 技能**问题——技能可练。

## 一、正念 — What 与 How 技能

### 语境要点

正念不是「放空」，而是**有意图地参与当下**。

**What 技能（做什么）**

| 技能 | 做法 | 一句记 |
|------|------|--------|
| **观察 Observe** | 注意体验，不加词 | 只看 |
| **描述 Describe** | 用词语贴标签 | 只述 |
| **参与 Participate** | 投入当下活动 | 只做 |

**How 技能（怎么做）**

| 技能 | 做法 |
|------|------|
| **不评判 Nonjudgmentally** | 不说「不该如此」 |
| **一心 One-mindfully** | 一次一事 |
| **有效 Effectively** | 目标导向，非争对错 |

### 典型练习（讲义中常见）

- **正念呼吸**：注意力在呼吸进出，走神则带回  
- **身体扫描**：从头到脚观察感觉  
- **正念进食**：慢、观察味道质地  

### 与站内对照

[菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})「念头才萌」是**觉察**；DBT 正念是**结构化训练**。[阳明]({{ '/2026/06/22/yangming-xinxue.html' | relative_url }})「事上磨练」重践行；正念是事前的**当下锚定**。

### 用法

| 场景 | 正念术 |
|------|--------|
| 开会前 2 分钟 | 呼吸 + 描述当下身体感觉 |
| 愤怒上头 | 先观察 30 秒，再说话 |
| 多任务焦虑 | 一心：只开当前这一件事 |

**口诀**：观察、描述、参与；不评、一心、有效。

## 二、痛苦耐受 — 危机时不让情况更糟

### 语境要点

目标不是「立刻舒服」，而是**熬过冲动波峰而不做破坏性行动**（自伤、爆发、报复性邮件、酗酒等）。

### 危机生存技能（常用缩写）

| 技能 | 含义 | 示例 |
|------|------|------|
| **TIP** | 体温 · 激烈运动 · paced呼吸 · 放松肌肉 | 冷水洗脸、快走 15 分钟 |
| **ACCEPTS** | 分心、贡献、比较、情绪、推掉、感觉、数、加任务 | 危机时转移注意 |
| **自我安抚** | 五感安抚 | 香氛、毯子、音乐 |
| **改善瞬间 IMPROVE** | 意象、意义、祈祷、放松、一件事、假期、鼓励 | 想象安全地 |
| **转念一念** | 痛苦可承受、生命有意义 | 认知重评 |
| **冲浪 Urge Surfing** | 冲动如浪，观察其起灭 | 不立刻行动 |

### 典型场景

**想发愤怒长文给老板**  
→ TIP（快走）+ Urge Surfing（等 20 分钟）+ 痛苦耐受目标：**此刻目标是「不恶化」**，非「赢得争论」。

### 用法

| 场景 | 痛苦耐受 |
|------|----------|
| 分手当晚 | ACCEPTS + 自我安抚 |
| 财务坏消息 | 转念 + 冲浪 |
| 亲子冲突爆发点 | 先离开现场 10 分钟（若安全） |

## 三、情绪调节 — 理解情绪、改变情绪

### 语境要点

情绪有功能：传达信息、促进行动。DBT 不是消灭情绪，而是**减少情绪失控带来的代价**。

### 核心技能

| 技能 | 内容 |
|------|------|
| **认识情绪** | 名称、诱因、身体感觉、冲动、表达 |
| **ABC PLEASE** | 减少脆弱：生病、饮食、睡眠、运动、物质等 |
| **相反行动 Opposite Action** | 情绪冲动若**不 fit 事实**，做相反行为 |
| **积累正向情绪** | 短期愉悦 + 长期价值目标 |
| **掌握情绪** | 早期介入，非等爆发 |

### 相反行动示例

| 情绪 | 冲动 | 若不合事实则 |
|------|------|--------------|
| 恐惧 | 回避 | 逐步接近 |
| 羞耻 | 躲藏 | 抬头、适度自我暴露 |
| 愤怒 | 攻击 | 温和坚定、或先离开 |
| 悲伤 | 躺平 | 小步活动 |

### 练习册典型页

- 情绪日记：事件 → 情绪强度 0–10 → 行为 → 后果  
- 识别**首要情绪**与**次要情绪**（如愤怒下是受伤）

[Gazzaniga]({{ '/2026/05/12/gazzaniga-cognitive-neuroscience.html' | relative_url }})：情绪标记辅助调节；DBT 提供**操作步骤**。[有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }})：事后 ABC 分析可并入周复盘。

## 四、人际效能 — DEAR MAN · GIVE · FAST

### 语境要点

既要**得到所求**，又要**维护关系**，还要**自尊**——三者常拉扯，DBT 分技能应对。

### DEAR MAN（主张 — 得到所求）

| 字母 | 技能 |
|------|------|
| **D** Describe | 描述事实 |
| **E** Express | 表达感受与主张 |
| **A** Assert | 明确提出要求 |
| **R** Reinforce | 说明对方好处 |
| **M** Mindful | 专注目标，不被带偏 |
| **A** Appear confident | 表现自信 |
| **N** Negotiate | 愿意协商 |

### GIVE（关系 — 维护关系）

**G**entle 温和 · **I**nterested 兴趣 · **V**alidate 验证 · **E**asy manner 轻松

### FAST（自尊 — 维护自尊）

**F**air 公平 · **A**pologies 少过度道歉 · **S**tick to values 守价值 · **T**ruthful 诚实

### 典型场景：请求加薪

- **DEAR MAN**：事实（职责扩大）→ 感受 → 主张（调整薪酬）→ 对公司留存好处  
- **GIVE**：听老板约束，验证预算压力  
- **FAST**：不贬低自己「我不配」，不说谎威胁  

与 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})「交换利益」可对读：DEAR MAN 是**个人版**利益与需求表达。

### 与鬼谷子之别

[鬼谷子]({{ '/2026/04/27/guiguzi.html' | relative_url }})重揣摩与博弈；DBT 人际效能重**真诚、验证、边界**——宜危机与亲密关系用 DBT，复杂谈判可补鬼谷子视角。

## 讲义与练习册怎么用

### 阅读顺序（自学）

| 周 | 模块 | 动作 |
|----|------|------|
| 1–2 | 正念 | 每日 5–10 分钟 + 填观察日志 |
| 3–4 | 痛苦耐受 | 列个人危机清单 + 预选技能 |
| 5–6 | 情绪调节 | 情绪日记 + PLEASE 自查 |
| 7–8 | 人际效能 | 写 3 个 DEAR MAN 脚本并演练 |

### 练习册填写模板（精简）

```text
【日期 / 情境】
【情绪名称 + 强度 0–10】
【身体感觉 · 冲动行为 · 实际行为】
【使用的 DBT 技能】
【结果 · 下周试什么】
```

### 与复盘合并（周课）

```text
本周情绪高峰 1 次：用何技能？有效否？
人际困难 1 次：DEAR MAN / GIVE / FAST 哪步卡住？
正念练习天数：___ / 7
PLEASE 最弱一环：睡眠 / 饮食 / 运动 / …
```

见 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }}) 三行日记 / KPT。

## 技能选型速查

| 你的状态 | 优先模块 | 起手技能 |
|----------|----------|----------|
| 冲动极强、想立刻做坏事 | 痛苦耐受 | TIP + Urge Surfing |
| 情绪淹没、说不清 | 正念 | 观察 + 描述 |
| 反复焦虑/低落 | 情绪调节 | ABC + 相反行动 |

| 关系僵但还要合作 | 人际效能 | GIVE + FAST |
| 事后后悔爆发 | 复盘 + 情绪调节 | 填练习册 + 相反行动预习 |

## 阅读边界

1. **不替代治疗** — 自杀意念、自伤、严重抑郁须求助专业人士  
2. **要练不要囤** — 讲义看懂 ≠ 技能习得；须重复填写练习册  
3. **辩证非和稀泥** — 接纳不是认同伤害；改变不是自我攻击  
4. **文化适配** — 部分例子偏美国临床语境，宜改写成自己的情境  
5. **与经典并读** — 哲学养心与临床技能互补，非互相取代
