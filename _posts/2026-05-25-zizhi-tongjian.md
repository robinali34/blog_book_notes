**《资治通鉴》**  
北宋司马光主编 · 历时十九年 · 294 卷 · 上起周威烈王二十三年（前 403），下迄后周世宗显德六年（959）· 十六朝一千三百六十二年 · 核心：**以史为鉴，察治国用人之得失**

## 全书概览

《资治通鉴》不是帝王八卦集，而是一套**编年体因果链**：一事起，百事后知之。司马光删削诸史，按年系事，使读者见**制度何以立、人心何以离、功业何以成、祸乱何以生**。

与兵书、纵横术不同，通鉴回答的不是「如何一战取胜」或「如何说服一人」，而是：

> **在数十年至数百年的时间尺度上，何种决策与结构组合导致兴、何种导致亡？**

> 鉴前世之兴衰，考当时之得失，劳于治道，使为君者可观。

读通鉴，宜记三问：**谁在做决策？结构如何约束他？人心为何随之而动？**

司马光不取炎黄传说开篇，而取**周威烈王承认三家为诸侯**——因在他看来，天子既无力诛，又无力赏，名分一坏，战国之乱无解。此史观直接影响全书读法：

| 通鉴关切 | 现代组织镜 |
|----------|------------|
| 名分与纲纪 | 章程、董事会权威 |
| 变法与反弹 | 组织变革、利益重切 |
| 开国与守成 | 创业期 vs 成熟期逻辑切换 |
| 纳谏与信息 | 真话是否划算 |
| 藩镇与权重 | 大区、渠道、关键岗位制衡 |

## 一、礼崩序乱 — 名分是秩序的第一根桩

### 语境要点

- 通鉴**开篇即三家分晋**（前 403），非从黄帝起笔——天子不能制大夫，礼崩之始，战国之乱由此开。
- 「天子非惟无力于诛，又无力于赏」——最高权威失能，诸侯大夫各行其是。
- 读此主题：看**制度合法性**如何先于武力合法性。
- 与 [韩非子]({{ '/2026/05/02/hanfeizi.html' | relative_url }})法术势：名分崩则法难施于上。

### 名句

| 原文/史论 | 要义 |
|-----------|------|
| 天子非惟无力于诛，又无力于赏 | 最高权威空心化 |
| 礼乐征伐自天子出（失效） | 名分源头断裂 |
| 臣弑其君，子弑其父 | 纲纪坏则乱不可止 |

### 典型史案

**三家分晋**  
晋六卿中，韩、赵、魏三家瓜分晋国，周威烈王被迫承认其为诸侯。表面是权力再分配，实质是**周礼所定「臣不得弑君、大夫不得专国」**已失效。此后变法、合纵连横、兼并战争皆在「名分已坏」的棋盘上进行。

**田氏代齐**  
陈国公子完（田氏）后裔在齐国渐夺政权，最终姜齐让位于田齐。与三家分晋同构：**长期侵蚀合法性的过程**，比一夜政变更值得警惕——组织里「架空」往往早于「夺权」。

**反例：周初封建**  
名分清晰、赏罚有据时，诸侯虽强仍守朝贡——礼未崩则乱有上限。

### 用法与场景

| 场景 | 通鉴镜 | 具体操作 |
|------|--------|----------|
| 公司治理 | 章程权威 | 董事会决议是否被 CEO 架空？写权力清单年检 |
| 合伙创业 | 名分先行 | 股权、决策权、退出机制书面化，忌口头君子协定 |
| 合规体系 | 强人约束 | 规则若只约束基层、不约束高管，等同周天子无力 |
| 平台治理 | 规则信用 | 算法/条款朝令夕改即「无力于赏」 |
| 家族企业 | 继承名分 | 接班人授权是否经正式程序 |

## 二、变法图强 — 制度换国力，亦换代价

### 语境要点

- 战国变法共性：**集中君权、奖励耕战、削弱贵族、统一度量**。
- 通鉴对商鞅、王安石着墨甚多——变法是**用短期阵痛换长期国力**，亦易积怨。
- 读变法案：问三件事——**谁受益、谁受损、补偿与退出机制有无**。
- 与 [商君书]({{ '/2026/05/13/shangjunshu.html' | relative_url }})对照读变法理论，与通鉴读变法后果。

### 名句

| 论断 | 要义 |
|------|------|
| 治世不一道，便国不法古 | 变法须有理论勇气 |
| 法行必信，民安其赏 | 信赏则法行 |
| 民不畏死，奈何以死惧之 | 苛酷积怨之戒 |

### 典型史案

**商鞅变法（秦孝公）**  
废井田、开阡陌，奖励军功，实行连坐。秦国由弱转强，但「刑过不避大臣，赏善不遗匹夫」背后，是**贵族利益被整体重切**。商鞅本人死于惠文王即位后的反弹——变法者未必能保全于变法之后。

**王安石变法（北宋）**  
青苗法、市易法、保甲法等，意在富国强兵，却引发新旧党争。司马光与王安石政见相左，最终《资治通鉴》成书与变法失败几乎同期——典型**改革未获共识、执行走样、利益集团反扑**的三重困局。

**反例：王莽改制**  
理想大、落地乱、名分伪——变法无信无共识则速败。

### 用法与场景

| 场景 | 变法镜 | 具体操作 |
|------|--------|----------|
| 组织重组 | 补偿先行 | 动编制前先算谁损、如何补、过渡期多久 |
| 流程数字化 | 减负为本 | 减旧负担再加新系统，否则只增负不改效 |
| 绩效改革 | 透明申诉 | 标准公开、过渡期、申诉通道——缺一即生怨 |
| 薪酬改革 | 试点 | 单部门验证再推广，忌一夜全员切换 |
| 文化变革 | 共识会 | 关键意见领袖先对齐，再全员宣布 |

## 三、一统与暴亡 — 秦之镜

### 语境要点

- 秦统一是制度突破极大；秦速亡是**模式不切换**之戒。
- 读秦案：问「得天下手段」与「守天下手段」是否同一套。
- [孙子]({{ '/2026/06/08/sunzi.html' | relative_url }})「非利不动」于秦二世前完全失效——怒而兴师、苛而固国。

### 名句

| 论断 | 要义 |
|------|------|
| 仁义不施而攻守之势异也 | 得天下与守天下手段须变 |
| 族秦者秦也，非天下也 | 败因在内，非仅外敌 |
| 天下苦秦久矣 | 人心交换破裂 |

### 典型史案

**秦始皇一统**  
书同文、车同轨、郡县制——制度突破极大。然严刑峻法、焚书坑儒、徭役过重（修长城、阿房宫、骊陵），**以力聚国，未以仁固国**。

**秦二世而亡**  
赵高指鹿为马，李斯被诛，章邯反水，项羽破咸阳。通鉴载其速亡链：**继承失控 → 信息垄断 → 忠臣失声 → 基层崩解**。非一战之败，而是**合法性+人心**同时归零。

### 用法与场景

| 场景 | 秦镜 | 具体操作 |
|------|------|----------|
| 高速增长后 | 模式切换 | 创业期压榨不可直接用于成熟期 |
| 并购后整合 | 人心优先 | 约法三章式清晰规则，忌文化清洗过激 |
| 信息治理 | 防指鹿为马 | 一线反馈渠道、独立审计 |
| 继承交接 | 制度续 | CEO 更替预案，忌一人独断后真空 |
| 团队扩张 | 负荷线 | 问：基层是否已「苦秦」？ |

## 四、开国用人 — 刘邦何以胜项羽

### 语境要点

- 楚汉是通鉴**人心 + 谋断 + 用人**的综合试卷。
- 开国逻辑：敢用奇才、能忍小辱、先安民心。
- 守成逻辑：功臣安置、交换结构重谈——见 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})。

### 名句

| 原文 | 要义 |
|------|------|
| 运筹帷幄中，决胜千里外 | 用人所长 |
| 吾宁斗智，不能斗力 | 开国重谋与组织 |
| 大方趋士，至无所用 | 谦抑容人 |

### 典型史案

**约法三章**  
刘邦入咸阳，与父老约「杀人者死，伤人及盗抵罪」，余法尽除。对比项羽屠城、劫掠——**人心交换**：百姓用拥护换安全。

**鸿门宴**  
项羽势大却犹豫；范增举玦再三，沛公脱身。项羽失在**谋断不果、听谗离间**；刘邦胜在**能忍、能用、能听张良**。可与 [鬼谷子决篇]({{ '/2026/04/27/guiguzi.html' | relative_url }})并读。

**萧何追韩信**  
刘邦欲止，萧何曰「诸将易得，信国士也」——开国期识别**不可替代人才**比占一城一地重要。

**韩信之死**  
天下定后，鸟尽弓藏。开国逻辑与守成逻辑冲突——交换结构须随阶段重谈。

### 用法与场景

| 场景 | 开国镜 | 具体操作 |
|------|--------|----------|
| 市场进入 | 先人心 | 信任、口碑、合规先于份额 |
| 关键招聘 | 萧何追韩信 | 慢决策、快行动；非标人才单独流程 |
| 公司上市后 | 功臣重设 | 创业期权/权力结构再谈判 |
| 竞争 | 忍小辱 | 非核心战场退让，换核心胜机 |
| 团队 | 容人所长 | 不苛求全能，像刘邦用三杰 |

## 五、守成纳谏 — 文景与贞观

### 语境要点

- 盛世常由**减法**起：少折腾、少争功、让社会修复。
- 纳谏不是性格好，是**制度让真话划算**。
- [菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})「耳中常闻反对声」与贞观精神相通。

### 名句

| 原文 | 要义 |
|------|------|
| 以铜为镜，可以正衣冠；以古为镜，可以知兴替 | 纳谏与读史 |
| 水所以载舟，亦所以覆舟 | 人心即政权基础 |
| 陛下开臣言路，则臣敢尽言 | 真话需安全通道 |

### 典型史案

**文景之治**  
文帝废肉刑、减轻徭役、与民休息；景帝平七国之乱后仍守轻徭薄赋。通鉴显：**盛世常由减法起**。

**贞观纳谏**  
唐太宗自谦「见机不如房玄龄，决断不如尉迟敬德」，任魏征犯颜直谏。**纳谏不是性格好，是制度让真话划算**。

**玄武门之变（背景案）**  
兄弟相争，太宗即位。通鉴不回避权力更迭之残酷，亦记其后**任贤、纳谏、省刑**以补合法性。

### 用法与场景

| 场景 | 守成镜 | 具体操作 |
|------|--------|----------|
| 成熟业务 | 做减法 | 减 KPI、减会议，优于再加杠杆 |
| 高管团队 | 魏征角色 | 反对票、红队复盘、无惩罚复盘 |
| 1:1 文化 | 真话划算 | 谎话才危，非真话才危 |
| 产品 | 技术债偿还 | 盛世期修基础，非只堆功能 |
| 领导 | 以古为镜 | 季度读一史案对照当下决策 |

## 六、扩张与反思 — 汉武帝前后

### 语境要点

- 扩张须设**止损与反思点**——轮台诏是通鉴极珍贵的「胜者悔悟」。
- [孙子]({{ '/2026/06/08/sunzi.html' | relative_url }})「久暴师则国用不足」于武帝晚年应验。

### 名句

| 原文 | 要义 |
|------|------|
| 朕即位以来，所为狂悖，使天下愁苦 | 胜者亦须悔 |
| 兵者凶器，不得已而用之 | 扩张边界 |
| 方士之言，多不验 | 先知勿靠鬼神 |

### 典型史案

**汉武帝前期**  
推恩制削藩、独尊儒术、北击匈奴——国力与威望达顶峰。

**轮台诏（悔悟）**  
晚年征伐过度、财政枯竭、民力疲弊，下诏自责——**战略扩张须设止损与反思点**。通鉴价值：记功亦记过，不造神。

### 用法与场景

| 场景 | 扩张镜 | 具体操作 |
|------|--------|----------|
| 业务多元化 | 轮台时刻 | 每条战线设复盘日：继续还是收缩？ |
| 融资烧钱 | 民力账 | 团队 burnout、现金流是否已竭？ |
| 国际扩张 | 多线风险 | 同时开五国不如一国做透 |
| 个人 | 野心上限 | 问：赢的下一场，输得起吗？ |

## 七、中兴与党争 — 光武与东汉末

### 语境要点

- 中兴要诀：**核心团队稳定 + 战争红利分配清晰**。
- 党争是**身份政治取代事实政治**——国力未亡于外敌，先亡于内耗。

### 名句

| 论断 | 要义 |
|------|------|
| 云台二十八将，各得其所 | 中兴在分配 |
| 党人相讥，天下清流浊流 | 标签化之始 |
| 宦官专权，士人结党 | 两败俱伤 |

### 典型史案

**光武中兴**  
刘秀定天下，「云台二十八将」各得其所。中兴要诀：**核心团队稳定 + 战争红利分配清晰**。

**党锢之祸**  
宦官专权，士人结党，互相攻讦，终致杀戮。国力未亡于外敌，先亡于**内部身份政治**。

### 用法与场景

| 场景 | 党争镜 | 具体操作 |
|------|--------|----------|
| 部门墙 | 标签战 | 「我们是实线他们是虚线」即党锢微澜 |
| 舆论场 | 站队先于证据 | 设事实复盘，禁先站队 |
| 招聘 | 派系 | 忌只招「自己人」 |
| 绩效 | 清流浊流 | 用结果与行为，非阵营 |
| 并购 | 文化党争 | 早定共同敌人是市场，非彼此 |

## 八、谋断与人心 — 鸿门、官渡、赤壁

### 语境要点

- 谋断主题：信息、环境、联盟、执行——强于兵力亦可败。
- 与 [孙子庙算]({{ '/2026/06/08/sunzi.html' | relative_url }})、[鬼谷子揣权决]({{ '/2026/04/27/guiguzi.html' | relative_url }})并读。

### 名句

| 原文 | 要义 |
|------|------|
| 竖子不足与谋 | 不决之代价 |
| 士卒不尽食，将军不能食 | 人心即战力 |
| 鞠躬尽瘁，死而后已 | 忠臣不能代替国力 |

### 典型史案

**官渡之战**  
曹操兵寡，袁绍兵众。曹操袭乌巢焚粮，袁绍不纳许攸——**信息、决断、执行力**三差，兵力悬殊亦可逆转。

**赤壁之战**  
曹操军水土不服、疫疾、链船之误；孙刘联盟、火攻。势强者若**轻敌、忽视环境、低估联盟**，亦可一败涂地。

**诸葛亮出师**  
《出师表》背后，是蜀汉国力已衰仍须北伐——**战略企图与资源现实错位**，忠臣亦难挽颓势。

### 用法与场景

| 场景 | 谋断镜 | 具体操作 |
|------|--------|----------|
| 竞品战 | 忌链船 | 优势变僵化负担时主动拆 |
| 战略坚持 | 国力账 | 季度算资源与现实，区分信念与执念 |
| 危机 | 鸿门检查 | 窗口期 48h 内是否决断？ |
| 联盟 | 赤壁镜 | 环境、伙伴、轻敌三查 |
| 项目 | 乌巢原则 | 保护关键补给线（现金流、核心人才） |

## 九、藩镇与崩坏 — 盛唐转衰

### 语境要点

- 藩镇是**权重失衡**之长周期病：边功坐大、中央难制、财政军事双失控。
- 黄巢是**基层交换破裂**之果：民变非一日。

### 名句

| 论断 | 要义 |
|------|------|
| 方镇之患，始于开元 | 盛世已埋种子 |
| 黄巢起义，天下纷然 | 基层破裂 |
| 骄兵难制 | 权重失衡 |

### 典型史案

**安史之乱**  
节度使权重边将，安禄山反叛，盛唐由盛转衰。根因链：**边功坐大 → 中央难制 → 藩镇世袭 → 财政军事双失控**。

**黄巢起义**  
晚唐赋役繁重、灾荒频仍，黄巢起事，长安陷落。民变非一日：通鉴见**基层交换破裂**。

### 用法与场景

| 场景 | 藩镇镜 | 具体操作 |
|------|--------|----------|
| 大区负责人 | 制衡 | 权、财、人勿长期一人垄断；轮岗与审计 |
| 平台依赖 | 渠道风险 | 单一渠道即节度使——多元化 |
| 关键供应商 | 依赖度 | 第二供应商预案 |
| 技术骨干 | 单点 | 知识文档化、梯队建设 |
| 基层体验 | 黄巢镜 | 一线薪酬、负荷、公平感季度测 |

## 十、五代速替 — 武人政治的上限

### 语境要点

- 武力可夺权，**文治与制度交接**不能接续则必速替。
- 陈桥兵变与「杯酒释兵权」是通鉴给出的**正面交接**样板。

### 名句

| 论断 | 要义 |
|------|------|
| 天子者，兵强马壮者为之 | 五代逻辑 |
| 杯酒释兵权 | 以制度换和平 |
| 割燕云，贻百年患 | 屈辱交换长患 |

### 典型史案

**朱温篡唐**  
藩镇武人掌权，朱温灭唐建梁。五代五十三年更迭如走马——**武力可夺权，文治不能接续则必速替**。

**儿皇帝石敬瑭**  
割燕云十六州予契丹，称臣求援。短利换长患——**屈辱性交换**贻害后世百年。

**陈桥兵变**  
赵匡胤黄袍加身，却「杯酒释兵权」——知武人政治之弊，反以**制度换和平交接**，开宋初不同于五代之路。

### 用法与场景

| 场景 | 五代镜 | 具体操作 |
|------|--------|----------|
| 并购整合 | 制度交接 | 不只靠强人压场，写清权财人移交 |
| 短期对赌 | 燕云戒 | 赢一季条款是否输十年？ |
| 创始人退休 | 释兵权 | 渐进交权，非一日悬空 |
| 战时 CEO | 文治接续 | 打江山 CEO 与守江山 CEO 分工 |
| 合作 | 屈辱条款 | 拒签损害核心资产的短利约 |

## 读通鉴四步法

| 步骤 | 做法 |
|------|------|
| 1 | **锚定一年** — 选一年读透，不追剧情 |
| 2 | **画因果链** — 谁决策 → 结构约束 → 人心反应 → 次年后果 |
| 3 | **对照主题** — 归入十大专题之一 |
| 4 | **写一句鉴** — 「此事若置于今日某组织，像什么？」 |

### 复盘模板（读史用）

```text
【年份 / 卷次】
【决策者与约束】谁有权？谁制衡？
【制度变动】变了什么规则？
【人心得失】谁拥护、谁怨恨、为何？
【次年验证】预言是否应验？
【今日映射】一场景、一教训
```

可与 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }}) 四步法并用：通鉴提供**案例库**，复盘提供**结构**。

## 阅读边界

读《资治通鉴》宜：

1. **取结构，不泥权术** — 帝王术非现代管理全部
2. **辨史观** — 司马光有儒家名分史观，宜与出土文献、专史对照
3. **忌故事会** — 记名不记因果，则白读
4. **古今异制** — 郡县、藩镇无今日一一对应，取**机制相似**即可
5. **与道德互补** — 知兴亡亦需知何为正当目标
6. **忌鉴戒口号化** — 未画因果链的「以史为鉴」只是装饰
