**Nonviolent Communication: A Language of Life**  
马歇尔·卢森堡（Marshall B. Rosenberg, 1934—2015）· **非暴力沟通（NVC, Nonviolent Communication）** · 核心：**观察、感受、需要、请求 — 用需要取代指责，用请求取代要求**

> 暴力的根源在于人们忽视彼此的感受与需要。  
> 沟通的目标不是赢，而是联结。

卢森堡是临床心理学家，NVC 源于对人本主义、甘地非暴力思想的实践提炼。它不是「永远不生气」的道德教条，而是一套**可练的语言结构**：在冲突中仍能说真话、听真话，而不必用羞辱、威胁或讨好换取和平。

读 NVC，宜记三问：**我说的是事实还是评判？我说的是感受还是想法？我提出的是请求还是要求？**

## 全书概览：NVC 解决什么问题？

日常语言里，我们常把**观察**与**评判**、**感受**与**想法**、**需要**与**策略**、**请求**与**要求**混在一起。混得越厉害，对方越像被攻击，自己也越委屈——于是要么爆发，要么冷战，要么讨好。

NVC 做的不是「把真话咽下去」，而是**换一套更不易触发防御的说法**，让对方有机会听见你的需要；同时，用**同理倾听**听懂对方需要，即使最后仍要拒绝。

```text
暴力语言：评判 → 辩护/反击 → 联结断裂
NVC 语言：观察+感受+需要+请求 → 对方可能听见 → 联结或清晰边界
```

## 四要素总表

| 要素 | 是 | 不是 | 常见错误 |
|------|-----|------|----------|
| **观察 Observation** | 具体、可录像的事实 | 评判（「你总是…」「你太自私」） | 用「总是/从不」代替一次事件 |
| **感受 Feeling** | 情绪词（难过、害怕、疲惫） | 想法（「我觉得你不关心」） | 用「我觉得你…」开头 |
| **需要 Need** | 人类共通需要（尊重、安全、休息） | 策略（「我需要你今晚洗碗」里的「洗碗」） | 把「要对方做什么」当需要 |
| **请求 Request** | 具体、可行、此刻可答、可被拒绝 | 要求（「你必须…」「不然…」） | 用威胁、内疚绑架 |

**标准句式**：

```text
当我看到 / 听到 [观察]
我感到 [感受]
因为我需要 [需要]
你愿意 [请求] 吗？
```

**自检口诀**：观事实、说感受、连需要、发请求。

## 一、观察 — 与评判分开

### 为什么要练观察？

大脑爱概括。「你又迟到」往往是评判；「这次会议 9:15 开始，你 9:28 进门」才是观察。评判会引爆辩护；观察给对方留下讨论空间。

| 评判（Jackal） | 观察（Giraffe） |
|----------------|-----------------|
| 「你从来不听我的！」 | 「刚才我说到第三句时，你看了两次手机。」 |
| 「你太不负责任了！」 | 「答应周三交的稿，周五还没收到。」 |
| 「我妈就是控制狂。」 | 「这周她打了四次电话问周末安排。」 |

### 练习：去评判化改写

| 原句 | 观察版 |
|------|--------|
| 「老公根本不在乎家。」 | 「这周三个晚上他十点后才回家。」 |
| 「老板压榨人。」 | 「这个月第四次周五临下班派新任务。」 |
| 「孩子故意气我。」 | 「我叫他收玩具时，他继续玩积木约五分钟。」 |

### 与正念、诚意的连接

[DBT 正念]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }})的**描述 Describe**：只述所见。[传习录]({{ '/2026/06/17/chuanxilu-wang-yangming.html' | relative_url }})「诚意」：不欺人亦不欺己——观察是诚意的语言版。

### 观察练习：24 小时挑战

| 时段 | 任务 |
|------|------|
| 晨 | 记录 1 句内心评判 → 改观察 |
| 午 | 会议/聊天中抓 1 个「总是/从不」 |
| 晚 | 复盘：哪句观察最难写？ |

## 二、感受 — 与想法分开

### 感受词 vs 伪感受

「我觉得被忽视」「我觉得你不尊重我」——**「我觉得」后面常接的是想法**，不是感受。感受是身体与情绪的状态词。

| 伪感受（想法） | 真感受 |
|----------------|--------|
| 我觉得你不爱我 | 孤单、伤心、害怕 |
| 我觉得被利用 | 愤怒、疲惫、委屈 |
| 我觉得很失败 | 羞愧、沮丧、焦虑 |

### 感受词库（节选 — 宜抄进笔记本扩充）

**不适类**：生气、烦躁、恼火、挫败、担心、焦虑、害怕、紧张、不知所措、孤单、伤心、难过、失望、疲惫、厌倦、麻木

**舒适类**：平静、放松、满足、开心、感激、希望、好奇、温暖、踏实、轻松、振奋

[高敏感]({{ '/2026/05/26/highly-sensitive-person-aron.html' | relative_url }})读者：情绪粒度细不是「矫情」，而是**更早识别过载**的信号——先命名感受，再决定要不要说话。

### 场景：说不出口感受

很多人只会说「我没事」「随便」。可练**微表达**：

```text
不是：「没事。」
而是：「我现在有点累，需要半小时安静。」
```

配合 [蛤蟆先生]({{ '/2026/05/15/counselling-for-toads-de-board.html' | relative_url }})：儿童状态常怕表达感受会招惩罚；Adult 状态练习「感受可说出来」。

### 身体线索表（辅助命名感受）

| 身体信号 | 可能感受 |
|----------|----------|
| 肩颈紧 | 烦躁、压力 |
| 胃沉 | 焦虑、害怕 |
| 脸热 | 羞愧、愤怒 |
| 胸空 | 孤单、失落 |
| 麻木 | 过载、解离 |

## 三、需要 — 人类共通语言

### 需要 vs 策略

| 需要（Universal Needs） | 策略（满足需要的具体方式） |
|-------------------------|----------------------------|
| 休息、支持、清晰、尊重 | 「你必须今晚洗碗」 |
| 联结、自主、安全、意义 | 「我们必须每周见你父母」 |
| 诚实、理解、贡献、玩耍 | 「你要换工作」 |

**关键**：两人可以争策略，却可能在**需要层**发现共同点。例：「周末见父母」之争，底层可能都是「联结」与「自主」——可协商形式，不必二选一羞辱。

### 需要词库（节选）

| 类别 | 需要词 |
|------|--------|
| 自主 | 选择、自由、空间、独立 |
| 联结 | 亲密、陪伴、归属、被爱、理解 |
| 身心 | 休息、安全、健康、舒适、滋养 |
| 意义 | 贡献、成长、学习、创造、目的 |
| 尊重 | 认可、诚实、平等、信任 |
| 玩乐 | 乐趣、轻松、幽默、美 |

### 需要层对话示例

**伴侣**：「你总加班！」（评判）  
→「这周四个晚上你十点后才回家（观察），我感到孤单（感受），因为我需要陪伴和联结（需要）。你愿意周三晚上留给我们吗？（请求）」

### 策略—需要拆解练习

| 原句 | 策略 | 底层需要 |
|------|------|----------|
| 你必须洗碗 | 洗碗 | 公平、支持、休息 |
| 我要你秒回消息 | 秒回 | 安全、联结、可预期 |
| 别再见那个人 | 断绝往来 | 安全、信任 |

## 四、请求 — 可拒绝的邀请

### 好请求的四条件

| 条件 | 说明 |
|------|------|
| **具体** | 对方知道做什么 |
| **可行** | 此刻做得到 |
| **积极** | 说「要什么」，少只说「不要什么」 |
| **可拒绝** | 真允许对方说「不」 |

| 要求（Jackal） | 请求（Giraffe） |
|----------------|-----------------|
| 「你必须今晚洗碗，不然别睡了。」 | 「你愿意今晚洗碗吗？若不行，我们商量别的分工。」 |
| 「以后都听我的。」 | 「这件事我愿意先听你的想法，再一起定。」 |
| 「好吧随便你。」（被动攻击） | 「我没办法同意这个方案；我需要明天前再答复你。」 |

### 倾听 ≠ 同意

[不懂拒绝]({{ '/2026/05/21/art-of-saying-no-refusal.html' | relative_url }})强调：可先同理，再清晰拒绝。

```text
「我听到你很希望这周完成（倾听观察/需要）。
我没办法加班做（清晰边界）。
我愿意下周一把初稿给你（替代/协商）。」
```

[被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }})：对方因拒绝而生气，多半是**对方的课题**；你仍可用 NVC 表达关心，但不牺牲边界。

## 长颈鹿语言 vs 豺狼语言

卢森堡用**长颈鹿**（心大、高处视野）与**豺狼**（评判、威胁）比喻两套语言。

| 豺狼 Jackal | 长颈鹿 Giraffe |
|-------------|----------------|
| 诊断、标签 | 观察 |
| 责备、羞辱 | 感受 + 需要 |
| 惩罚、威胁 | 请求 |
| 应该、必须 | 选择、邀请 |
| 讨好、否认自我 | 诚实 + 尊重对方 |

```text
豺狼：「你怎么这么笨！」
长颈鹿：「这份报告有三处数据错误（观察），我担心（感受），因为我需要准确和可信（需要）。你愿意核对后再发吗？（请求）」
```

**注意**：长颈鹿不是「甜腻讨好」。坚定拒绝也可以是长颈鹿：「我需要拒绝；我重视我们的合作。」

## 倾听 — 同理心四步

表达只是 NVC 一半；另一半是**带着同理听**——听见对方感受与需要，即使不同意其策略。

| 步 | 动作 | 示例回应 |
|----|------|----------|
| 1 | **复述观察** | 「你是说会议里被打断了三次？」 |
| 2 | **猜测感受** | 「听起来很沮丧？」 |
| 3 | **连接需要** | 「你是不是需要被尊重、被听完？」 |
| 4 | **确认请求** | 「你希望我下次先让你说完？」 |

### 倾听的边界

| 误区 | 正解 |
|------|------|
| 倾听 = 同意 | 可同理后仍拒绝 |
| 倾听 = 解决问题 | 有时对方只要被听见 |
| 倾听 = 无限吸收 | HSP 需设时间/能量上限（见 [高敏感]({{ '/2026/05/26/highly-sensitive-person-aron.html' | relative_url }})） |

### 倾听脚本模板

```text
「当你 [观察]……
你是不是感到 [感受]……
因为你需要 [需要]……
你是不是希望 [请求/策略]？」
```

对方激动时，**先倾听几轮，再表达自己**——与 DBT GIVE 的 Validate（验证）一致。

### 案例小传：只听不说

**阿杰**每次伴侣哭诉都立刻给方案，对方更怒：「你根本不懂！」

**改法**：三轮倾听（感受+需要）后再问：「你需要我给建议，还是只要陪着？」——常与 [亲密关系]({{ '/2026/06/10/intimate-relationships-moon.html' | relative_url }})幻灭期「追—逃」相关：追的人要**联结**，不是方案。

## 愤怒与内疚 — NVC 怎么看

### 愤怒

NVC **不否认愤怒**，但区分：

| 层次 | 内容 |
|------|------|
| 刺激 | 对方行为（观察） |
| 愤怒 | 常夹杂评判（「他不该…」） |
| 底层 | 未满足的需要 |

**练法**：愤怒上头时，[DBT STOP]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }}) 暂停 → 问「我需要什么？」→ 再用四要素表达。

| Jackal 愤怒 | Giraffe 转化 |
|-------------|--------------|
| 「这混蛋气死我了！」 | 「当我看到…我感到愤怒，因为我需要公平/尊重。」 |

愤怒可以是**边界信号**：告诉你某需要长期被忽视——但表达仍可选择暴力或 NVC。

### 内疚

| 类型 | 说明 | 做法 |
|------|------|------|
| 道德内疚 | 确实伤害了他人 | 承认观察、补救、改进行为 |
| 操纵性内疚 | 别人用「你让我失望」绑架 | 区分：对方失望是对方感受；你的选择是你的课题 |
| 讨好者内疚 | 拒绝后自责 | [不懂拒绝]({{ '/2026/05/21/art-of-saying-no-refusal.html' | relative_url }}) 四步自处 |

```text
内疚时自问：
1. 我是否真的伤害了谁？（观察）
2. 我当时的需要是什么？
3. 下次如何既守需要又减少伤害？
```

## 与 DBT 对照 — DEAR MAN · GIVE · FAST 映射

| DBT 技能 | 字母 | NVC 对应 | 联用提示 |
|----------|------|----------|----------|
| **DEAR MAN** | D Describe | 观察 | 先事实，再情绪 |
| | E Express | 感受 | 真情绪词 |
| | A Assert | 需要 + 请求 | 明确主张 |
| | R Reinforce | 可嵌入「若你愿意…对双方…」 | 非操纵，是说明正向结果 |
| | M Mindful | 一心目标 | 不被羞辱带偏 |
| | A Appear confident | 语气平稳 | 不必吼 |
| | N Negotiate | 请求可协商 | 策略可换，需要可谈 |
| **GIVE** | G Gentle | 长颈鹿语气 | 无评判 |
| | I Interested | 倾听 | 好奇对方需要 |
| | V Validate | 同理四步 | 「听起来你…」 |
| | E Easy manner | 轻松 | 非审判庭 |
| **FAST** | F Fair | 公平 | 对自己与他人公平 |
| | A Apologies | 少过度道歉 | 拒绝不必道歉十遍 |
| | S Stick to values | 需要层价值 | 与 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }}) 价值一致 |
| | T Truthful | 观察诚实 | 不撒谎威胁 |

### 完整场景：职场请求调整分工

**情境**：同事第三次把本职外工作推给你。

| 步骤 | DEAR MAN + NVC 草稿 |
|------|---------------------|
| D + 观察 | 「这周周一、周三、周五下班前，你各发来一项本该你完成的报表核对。」 |
| E + 感受 | 「我感到焦虑、疲惫。」 |
| A + 需要 + 请求 | 「因为我需要可预期的负荷和休息。你愿意以后先自己完成，或提前一天协商吗？」 |
| R | 「这样我们组截止日期更稳，我也能在急活时真正帮上你。」 |
| GIVE | 听对方是否压力大，验证：「听起来你也在赶工？」 |
| FAST | 不贬低：「不是我小气」，诚实：「我没办法再默默接。」 |

若对方拒绝：进入 [不懂拒绝]({{ '/2026/05/21/art-of-saying-no-refusal.html' | relative_url }}) 清晰「不」+ 上级正式渠道。

## 典型场景详解

### 职场边界

| 子场景 | Jackal | Giraffe |
|--------|--------|---------|
| 临时加活 | 「领导就会压榨！」 | 「这周第三次临下班派新任务（观察）。我焦虑（感受），需要可预期节奏和休息（需要）。愿意以后提前一天说吗？（请求）」 |
| 会议被打断 | 「你根本不尊重人！」 | 「我说到一半时被打断两次（观察）。挫败（感受）。需要被听完（需要）。愿意我先讲完再讨论吗？」 |
| 抢功 | 「你太卑鄙！」 | 「会上呈现的是我上周发的方案（观察）。困惑、失望（感受）。需要认可与诚实（需要）。愿意会上说明来源吗？」 |

**HSP 补充**：会后需恢复时间；书面 NVC 草稿常比当场吵更稳（[高敏感]({{ '/2026/05/26/highly-sensitive-person-aron.html' | relative_url }})）。

### 对父母

与 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }})课题分离联用：

| 纠缠 | NVC + 分离 |
|------|------------|
| 催婚 | 「听到你们希望我今年结婚（倾听）。我感到压力（感受）。我需要按自己节奏决定人生（需要+课题）。我愿意每月和你们聊一次近况，婚期由我决定（请求+边界）。」 |
| 干涉育儿 | 「你们建议少穿一件（观察）。我担心（感受）。我需要做父母的自主与信任（需要）。愿意先让我们试一周吗？（请求）」 |
| 经济绑定 | 「你们资助买房（观察）。感激也内疚（感受）。需要自主与清晰约定（需要）。愿意把还款写进协议吗？」 |

**倾听父母**：Often 底层需要是**安心、联结、不被抛弃**——可说「我需要你们放心，我也需要你们信任我的选择」。

### 伴侣与亲密关系

联结 [亲密关系]({{ '/2026/06/10/intimate-relationships-moon.html' | relative_url }})：幻灭期指责 → 改述需要。

| 阶段 | 常见 Jackal | Giraffe 转向 |
|------|--------------|--------------|
| 绚丽 | （较少指责，多理想化） | 仍可练诚实需要，避免把对方当完美策略 |
| 幻灭 | 「你变了！」「你从不…」 | 「这周三次约会你迟到（观察）。失望（感受）。需要可靠与尊重（需要）。愿意提前告知吗？」 |
| 内省 | 冷战、翻旧账 | 先自问需要，再对话 |
| 启示 | — | 请求可协商，需要可共享 |

**争吵暂停协议**（可与伴侣约定）：

```text
1. 任一方说「暂停」→ 停止评判句
2. 各自 20 分钟：写观察-感受-需要-请求
3. 回来先倾听，再表达
```

### 亲子（简版）

| 对孩子表达 | 对孩子倾听 |
|------------|------------|
| 「玩具还在地上（观察）。我担心绊倒（感受）。需要安全（需要）。愿意现在收吗？」 | 「你不想收？是不是想继续玩、需要自主？」 |
| 避免：「你怎么这么不听话！」 | 避免：立刻讲道理到孩子shutdown |

### 线上沟通（现代扩展）

| 子场景 | Jackal | Giraffe |
|--------|--------|---------|
| 已读不回 | 「你故意冷落我！」 | 「消息发出六小时未回（观察）。焦虑（感受）。需要可预期回应（需要）。愿意忙时回一句『晚点到』吗？」 |
| 群聊争执 | 「你就是杠精！」 | 「刚才三次打断我发言（观察）。挫败（感受）。需要被听完（需要）。愿意私聊细谈吗？」 |
| 邮件催办 | 「你们都不负责！」 | 「截止日已过两天（观察）。担心（感受）。需要清晰进度（需要）。愿意今天下班前给更新吗？」 |

[高敏感]({{ '/2026/05/26/highly-sensitive-person-aron.html' | relative_url }})：线上冲突刺激更久——书面 NVC + 延迟发送。

### 案例小传：拒绝后的 Giraffe

**小周**被拉进志愿者群，不想参加。讨好版：已读乱答应。  
**Giraffe 拒绝**：「看到群里的招募（观察）。为难（感受）。需要休息和专注主业（需要）。我这次没办法参加，祝活动顺利（边界+善意）。」

## 自我同理 — 对自己也用四要素

```text
当我看到 我又答应了一件不想做的事
我感到 疲惫、怨恨
因为我需要 休息、自主、诚实
我愿意 下次先说「我想一下」吗？
```

### 自我同理日记

```text
【触发】
【Jackal 自责句】
【感受】
【需要】
【对自己的请求】
```

## 常见卡点与纠偏

| 卡点 | 纠偏 |
|------|------|
| 像背模板、不真诚 | 先写草稿，再口语化；诚意比格式重要 |
| 对方不吃 NVC | 你只管自己的语言；对方暴力仍可用 STOP + 边界 |
| 变成道德优越 | 长颈鹿不是「我比你文明」 |
| 请求被拒就崩溃 | 被拒是请求的正常结果；换策略或守边界 |
| 把需要藏进威胁 | 「我需要尊重」≠「不尊重就分手」——后者是要求 |
| 中文「给面子」 | 观察+需要可含蓄，但避免假同意 |
| 当众 NVC | 私下一对一更有效；公开易表演 |

## 练习计划

### 21 天渐进

| 周 | 每日任务 | 记录 |
|----|----------|------|
| **1 观察** | 把 1 句评判改观察 | 原句 / 改写 |
| **2 感受** | 积累感受词 + 每日命名 1 情绪 | 情绪强度 0–10 |
| **3 需要** | 积累需要词 + 冲突下问「我需要什么」 | 需要 vs 策略 |
| **4 整合** | 写 3 封四要素草稿（可不发） | 倾听轮数 |

### 练习册模板

```text
【日期 / 情境】
【Jackal 原句】
【观察】
【感受】
【需要】
【请求】
【若倾听对方：猜测其感受与需要】
【实际结果 · 下次调整】
```

```text
成功 Giraffe 1 次：哪一步最有效？
倾听：是否急于给建议而跳过验证？
```

## 暴力沟通 vs 非暴力沟通 — 扩展对照

| 暴力 | NVC 转向 |
|------|----------|
| 「你从来不听我的！」 | 「刚才我说到一半被打断，我感到沮丧，因为我需要被听见。你愿意听完再说吗？」 |
| 「你太自私了！」 | 「周末计划没和我商量，我感到失落，因为我需要参与感。我们一起定下周安排好吗？」 |
| 「好吧随便你。」（被动攻击） | 「我没办法同意这个方案，我需要时间考虑，明天回复你。」 |
| 「我牺牲这么多你还…」（内疚绑架） | 「我为家做了很多（观察）。疲惫（感受）。需要被看见与分担（需要）。愿意一起排家务吗？」 |
| 「爱我就该…」（要求） | 「我需要联结（需要）。你愿意今晚聊半小时吗？若不行我们另约时间。」 |
| 「你怎么这么玻璃心」（羞辱） | _不继续争辩人格；STOP + 边界或离开对话_ |
| 「算了不说了」（撤退） | 「我现在过载，需要暂停。愿意一小时后再谈吗？」 |

## FAQ 与常见误读

| 问题 | 简答 |
|------|------|
| NVC 是不是讨好？ | 否；诚实表达需要，可坚定拒绝 |
| 生气还能 NVC 吗？ | 能；先 STOP，愤怒下少评判 |
| 对方暴力怎么办？ | 安全优先；NVC 不替代安全计划 |
| 一定要四句式吗？ | 结构是脚手架，诚意为本 |
| 中国人太直接会冒犯？ | 观察+需要可温和，不必背英文腔 |
| 和 DBT 哪个先学？ | 危机多先 DBT；日常表达 NVC |
| 倾听多久算够？ | 视情境；至少验证感受一轮 |
| 请求被拒算失败吗？ | 否；清晰已是成功 |
| 能用于职场向上吗？ | 能；配合 DEAR MAN 与书面留痕 |
| 亲子几岁可教？ | 父母先练；孩子通过建模学习 |

### 阅读边界

1. **不替代治疗与法律保护** — 虐待、危机须专业介入  
2. **不道德绑架自己** — 允许生气、允许做不到完美 Giraffe  
3. **要练不要囤** — 草稿本比通读三遍有用  
4. **与课题分离并读** — 边界在内核，NVC 在表达  
5. **文化适配** — 面子、含蓄语境下调整语气，不调整结构
