**《孙子兵法》**  
春秋孙武撰 · 十三篇五千余言 · 世界最早兵学经典之一 · 核心：**先计后战、全胜为上、以实击虚**

## 全书概览

《孙子兵法》不是「勇猛冲锋手册」，而是一套**以最小代价取胜**的决策系统。孙子把战争拆成可计算的要素：庙算、形势、虚实、地形、情报——在刀兵相见之前，胜负往往已在庙堂分晓。

> 兵者，国之大事，死生之地，存亡之道，不可不察也。

> 上兵伐谋，其次伐交，其次伐兵，其下攻城。

## 一、始计 · 庙算先知

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/01-miaosuan.svg' | relative_url }}" alt="庙算五事七计示意图" />
<figcaption>图 1 — 始计篇：五事七计，多算胜</figcaption>
</figure>

### 语境要点

- **始计**：战前先算，非临阵才谋。「多算胜，少算不胜，而况于无算乎！」
- **五事**：道、天、地、将、法——民心、天时、地利、将领、法度。
- **七计**：比较双方君主、将领、天时地利、法令、兵力、训练、赏罚孰优孰劣。
- **知彼知己**：「知彼知己，百战不殆；不知彼而知己，一胜一负；不知彼不知己，每战必殆。」

### 名句

| 原文 | 要义 |
|------|------|
| 兵者，国之大事 | 战略级决策，不可轻启 |
| 未战而庙算胜者，得算多也 | 胜负在战前已见端倪 |
| 知彼知己，百战不殆 | 信息对称是底线 |
| 兵者，诡道也 | 示形用诈，非无道之诈 |

### 古代案例

**郭嘉十胜十败**：官渡之战前，曹操惧袁绍兵众，郭嘉以七计框架比较：绍道（政治）不如公、谋（将）不如公、法（制度）不如公……连言「十胜」。曹操遂决意迎战。此即**庙算多者胜**——未战而心胜。

**勾践灭吴**：勾践败于夫差后，十年生聚、十年教训，修「道」（民心）与「法」（组织），待吴国内耗、天时已变，方起伐。非凭一时之怒，而是**算足而后动**。

### 现代应用

| 场景 | 庙算法 |
|------|--------|
| 商业竞争 | 对标分析：品牌、团队、渠道、现金流、合规、激励——七项打分 |
| 创业立项 | 五事自检：用户真需求（道）、窗口期（天）、赛道空间（地）、核心班子（将）、运营体系（法） |
| 并购决策 | 尽职调查即「知彼」；自知现金流与整合能力即「知己」 |
| 个人择业 | 比较平台、直属上级、成长空间、薪酬制度、训练机会——勿无算而入 |

**口诀**：未战先庙算，五事七计明；知彼又知己，不动则已动则胜。

## 二、作战 · 谋攻 — 全胜与速决

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/02-mougong.svg' | relative_url }}" alt="谋攻全胜层次示意图" />
<figcaption>图 2 — 谋攻篇：伐谋为上，攻城为下</figcaption>
</figure>

### 语境要点

- **作战篇**：「国之大事，不可不察」的延伸——久战耗国：「久暴师则国用不足」「兵闻拙速，未睹巧之久也」。
- **谋攻篇**：取胜有层次——**全国为上，破国次之；全旅为上，破旅次之**。
- **伐谋**：破敌之谋于庙堂，不战而屈人之兵，为上上策。
- **攻城**：「其下攻城」——消耗最大，非不得已勿用。

### 名句

| 原文 | 要义 |
|------|------|
| 上兵伐谋，其次伐交，其次伐兵，其下攻城 | 战略优先级 |
| 不战而屈人之兵，善之善者也 | 最高胜利形态 |
| 以十围之，以五攻之，以分击之 | 兵力对比与打法 |
| 知胜有五：知可以战与不可以战者胜 | 知止是胜机之一 |

### 古代案例

**墨子止楚攻宋**：公输班为楚作云梯攻宋，墨子行十日十夜至郢，**伐谋**于殿上：以攻防推演示楚攻必败，又令弟子守宋备械。楚王乃止。未费一兵，而全宋之国——典型「不战而屈人」。

**孙膑马陵道**：孙膑减灶诱庞涓，不与其主力硬撼，而以谋设伏全歼魏军，虏太子申。胜在**知可以战与不可以战**——避其锋芒，击其必救之虚。

### 现代应用

| 场景 | 谋攻术 |
|------|--------|
| 市场竞争 | 优先标准制定、生态联盟、专利布局（伐谋/伐交），慎打价格战（伐兵） |
| 产品策略 | 用体验与网络效应取胜，而非烧钱硬砸渠道（戒「攻城」式消耗战） |
| 谈判 | 先破对方最优方案（伐谋），再谈条款；勿一上来就撕破脸 |
| 诉讼纠纷 | 调解、仲裁优于漫长诉讼——「久暴师则国用不足」 |

**口诀**：善战者伐谋，其次伐交兵；攻城不得已，速决莫久营。

## 三、军形 · 兵势 — 先为不可胜

### 语境要点

- **军形篇**：「胜兵先胜而后求战，败兵先战而后求胜。」先立于不败之地，再待敌之可胜。
- **不可胜在己，可胜在敌**：我能守，敌有隙，方能攻。
- **兵势篇**：「激水之疾，至于漂石者，势也」——组织**势能**与**节奏**，使「如转圆石于千仞之山」。
- **奇正相生**：以正合，以奇胜——常规部队牵制，奇兵制胜。

### 名句

| 原文 | 要义 |
|------|------|
| 胜兵先胜而后求战 | 先备后战 |
| 守而不失，攻而必取 | 形全则守，胜有余力则攻 |
| 以正合，以奇胜 | 正兵接战，奇兵出奇 |
| 善战者，求之于势，不责于人 | 重系统与势能，非苛责个人 |

### 古代案例

**韩信背水一战**：韩信背水列阵，士卒「知亡无死，勿生之兵」，势如激水；又遣奇兵袭赵营，两面夹击。非蛮勇，而是**造势**使人无退、使敌失措。

**李世民虎牢关**：窦建德援王世充，世民待其师疲食尽方出战，以少击众。先据虎牢之**形胜**，再借敌之怠——「先为不可胜」而后一击。

### 现代应用

| 场景 | 形势术 |
|------|--------|
| 现金流管理 | 先保生存底线（不可胜在己），再谈扩张 |
| 产品发布 | 正兵：主功能稳态迭代；奇兵：颠覆特性打差异 |
| 团队管理 | 流程与工具造「势」，减少对个人英雄主义依赖 |
| 投资 | 安全边际即「先胜」；不赌小概率翻盘 |

**口诀**：先为不可胜，以待敌之可胜；正合奇胜出，势至如转石。

## 四、虚实 · 军争 — 形人而我无形

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/03-xushi.svg' | relative_url }}" alt="虚实篇示意图" />
<figcaption>图 3 — 虚实篇：致人而不致于人，击虚避实</figcaption>
</figure>

### 语境要点

- **虚实篇**：「致人而不致于人」——调动敌人，而不被敌人调动。
- **形人而我无形**：使敌备于处处，我攻于不备——「吾所与战之地不可知，不可知则敌所备者多」。
- **军争篇**：「以迂为直，以患为利」——争利之道，争的是**先机**与**要道**，非意气之争。
- **军争之难**：「先处战地而待敌者佚，后至战地而趋战者劳。」

### 名句

| 原文 | 要义 |
|------|------|
| 致人而不致于人 | 掌握主动权 |
| 攻其无备，出其不意 | 击虚原则 |
| 避实而击虚 | 不在其强项硬撼 |
| 以迂为直，以患为利 | 表面迂回，实则占先 |

### 古代案例

**白起长平**：赵军坚壁不出，秦使间言换赵括为将；赵括出击，秦断粮道，击其不能之虚——赵军四十万溃。虚实转换在**诱敌离险**。

**围魏救赵**（孙膑）：魏攻赵，齐不直救邯郸而攻大梁，魏军回援，中途于桂陵伏击。**致人而不致于人**——打其必救，掌握节奏。

### 现代应用

| 场景 | 虚实术 |
|------|--------|
| 竞品策略 | 避其旗舰产品锋芒，攻其薄弱品类或区域 |
| 营销 | 声东击西：话题在 A，转化在 B |
| 项目管理 | 关键路径上抢先占资源（先处战地而待敌） |
| 谈判 | 表面让步于非核心条款，换核心利益 |

**口诀**：形人而我无形，致人不受人制；避实而击虚，以迂为直争先。

## 五、九变 · 地形 · 九地 — 知地因粮

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/04-dixing.svg' | relative_url }}" alt="地形九地示意图" />
<figcaption>图 4 — 地形六类、九地态势：因地制宜</figcaption>
</figure>

### 语境要点

- **九变篇**：将能「通变」——有所不争（城不争、地不争、主命不争），有所必为。
- **地形篇**：通、挂、支、隘、险、远六地，战法各异；「地形者，兵之助也。」
- **九地篇**：散地、轻地、争地、交地、衢地、重地、圮地、死地、亡地——按深入程度与退路定军心。
- **因粮于敌**：「智将务食于敌，食敌一钟，当吾二十钟。」——取资于敌，减己方消耗。

### 名句

| 原文 | 要义 |
|------|------|
| 将通于九变之利者，知用兵矣 | 权变非无原则 |
| 投之亡地然后存，陷之死地然后生 | 绝境激势（慎用） |
| 知地形，得地者之利 | 地利须匹配打法 |
| 主不可以怒而兴师，将不可以愠而致战 | 情绪不可启动战争 |

### 古代案例

**马谡失街亭**：诸葛亮伐魏，命马谡守街亭。谡据山不下寨，为张郃断水围击——**不知地形：隘险之地却自绝粮道**。一失要冲，全盘受挫。

**曹操乌林之前**：曹操军染疫、士卒不服水土，却强行连船东进；孙刘联军借风火与江形破之。不知**九地**（重地、交地）之险，又违「主不可以怒而兴师」。

### 现代应用

| 场景 | 地利术 |
|------|--------|
| 市场进入 | 辨「通地」（易进易退）与「死地」（只能赢不能输） |
| 远程办公 | 团队分散则加强制度与节奏，防「散地」无志 |
| 供应链 | 因粮于敌：本地化采购、合作伙伴生态，减长途消耗 |
| 危机决策 | 九变：什么必须坚持，什么可以放弃——勿争不该争之城 |

**口诀**：九变知取舍，地形六类分；九地定军心，因粮减己损。

## 六、火攻 · 用间 — 先知与慎战

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/05-yongjian.svg' | relative_url }}" alt="用间五间示意图" />
<figcaption>图 5 — 用间篇：五间并用，慎战明主</figcaption>
</figure>

### 语境要点

- **火攻篇**：火攻有五：火人、火积、火辎、火库、火队——**因五火之变而应之**，亦戒「主慎之」。
- **用间篇**：「明君贤将，所以动而胜人，成功出于众者，先知也。」
- **五间**：因间、内间、反间、死间、生间——情报体系，非单点间谍。
- **慎战总结**：「非利不动，非得不用，非危不战」；「主不可以怒而兴师，将不可以愠而致战。」

### 名句

| 原文 | 要义 |
|------|------|
| 知彼知己，胜乃不殆 | 全书反复之核 |
| 先知者，不可取于鬼神 | 靠人谋与系统，非玄学 |
| 非利不动，非得不用，非危不战 | 出手三条件 |
| 亡国不可以复存，死者不可以复生 | 慎战终极告诫 |

### 古代案例

**赤壁之战**：周瑜、黄盖用火攻，前提却是**用间**（蒋干、蔡瑁张允）与**先知**（知曹军不习水战、连船之弊）。火攻是「五火之变」的应法，非孤注一掷。

**郑国烛之武退秦师**：秦晋围郑，烛之武夜见秦穆公，陈亡郑无益于秦而益于晋，秦遂退。**伐谋伐交**之极——一言退敌，胜在情报与外交判断。

### 现代应用

| 场景 | 用间术（正当情报） |
|------|---------------------|
| 竞争分析 | 公开信息、行业报告、用户访谈、离职员工合规访谈——建「先知」系统 |
| 反间意识 | 防范对手误导性定价、假新闻、钓鱼情报 |
| 重大投资 | 非利不动：无明确 ROI 不投入；非得不用：无必要不烧资源 |
| 组织决策 | 领导愤怒时不拍板重大战争级项目 |

**口诀**：先知非鬼神，五间建体系；火攻应五变，慎战止怒愠。

## 阅读边界

读《孙子兵法》宜：

1. **明胜道，不泥诈道** — 「诡道」是兵法，不是无底线人生哲学
2. **慎战为本** — 非利不动、主不可怒；现代映射到：不冲动 All-in
3. **古今异器** — 车战、攻城与今日商业不同，宜取**结构原则**（庙算、虚实、先知）
4. **与道德互补** — 孙子教「如何胜」，《论语》教「为何战」；胜而不义，终不可久
