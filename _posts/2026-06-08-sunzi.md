## 全书概览

《孙子兵法》不是「勇猛冲锋手册」，而是一套**以最小代价取胜**的决策系统。孙子把战争拆成可计算的要素：庙算、形势、虚实、地形、情报——在刀兵相见之前，胜负往往已在庙堂分晓。

与纵横术、处世格言不同，孙子回答的不是「如何说服」或「如何养心」，而是：

> **在资源有限、信息不全、对手理性的条件下，如何以最小代价取得可持久的胜？**

> 兵者，国之大事，死生之地，存亡之道，不可不察也。

> 上兵伐谋，其次伐交，其次伐兵，其下攻城。

读孙子，宜记三问：**算足了吗？代价可承受吗？胜后守得住吗？**

### 春秋兵学语境：为何需要「庙算」？

春秋战国之交，战争频率与规模陡增：贵族车战渐让位于步卒合战，兼并取代会盟。旧礼乐秩序崩解，**算不清就开战**的代价空前沉重。孙子之异于《吴子》《六韬》者，在于把战争**前置为决策问题**：

| 压力 | 孙子式回应 |
|------|------------|
| 君主好战、将领好功 | 始计：五事七计，多算胜 |
| 久战耗国、攻城惨烈 | 谋攻：伐谋为上，攻城为下 |
| 以少敌众、以弱击强 | 形势：先为不可胜，借势击虚 |
| 敌强我弱、正面不敌 | 虚实：致人而不致于人 |
| 地形复杂、军心不稳 | 九变地形九地：因地制宜 |
| 情报不明、怒而兴师 | 用间慎战：先知非鬼神 |

可与 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange/' | relative_url }})「交换结构」并读：孙子算的是**国与国、军与军**的利害账，管理算的是**组织内**的激励与约束。

## 一、始计 · 庙算先知

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/01-miaosuan.svg' | relative_url }}" alt="庙算五事七计示意图" />
<figcaption>图 1 — 始计篇：五事七计，多算胜</figcaption>
</figure>

### 语境要点

- **始计**：战前先算，非临阵才谋。「多算胜，少算不胜，而况于无算乎！」
- **五事**：道、天、地、将、法——民心、天时、地利、将领、法度。
- **七计**：比较双方君主、将领、天时地利、法令、兵力、训练、赏罚孰优孰劣。
- **知彼知己**：「知彼知己，百战不殆；不知彼而知己，一胜一负；不知彼不知己，每战必殆。」
- **诡道**：示形用诈，是**战术层**信息不对称，非无底线欺诈——与 [鬼谷子]({{ '/2026/04/27/guiguzi/' | relative_url }})捭阖可对照读。

### 深度解析

庙算是把战争还原为**可比较变量**：

```text
五事（结构性条件）→ 七计（双方对比）→ 多算/少算/无算（决策质量）
```

「道」在孙子处首先是**上下同欲**——组织愿为同一目标承担代价。现代映射：员工是否真信战略，还是口头对齐？[韩非子]({{ '/2026/05/02/hanfeizi/' | relative_url }})以法术势固「道」，孙子以庙算验「道」是否成立。

### 名句

| 原文 | 要义 |
|------|------|
| 兵者，国之大事 | 战略级决策，不可轻启 |
| 未战而庙算胜者，得算多也 | 胜负在战前已见端倪 |
| 知彼知己，百战不殆 | 信息对称是底线 |
| 兵者，诡道也 | 示形用诈，非无道之诈 |
| 计者，校也 | 庙算本质是校对、比较 |
| 主孰有道 | 七计之首在「道」，非唯兵力 |

### 典型史案

**郭嘉十胜十败**  
官渡之战前，曹操惧袁绍兵众，郭嘉以七计框架比较：绍道（政治）不如公、谋（将）不如公、法（制度）不如公……连言「十胜」。曹操遂决意迎战。此即**庙算多者胜**——未战而心胜。可与 [资治通鉴]({{ '/2026/05/25/zizhi-tongjian/' | relative_url }})官渡纪事并读。

**勾践灭吴**  
勾践败于夫差后，十年生聚、十年教训，修「道」（民心）与「法」（组织），待吴国内耗、天时已变，方起伐。非凭一时之怒，而是**算足而后动**。

**反例：袁绍官渡**  
兵多粮足却庙算失：不听许攸、不救乌巢、将帅离心——七计中「将」「法」皆输，兵力反成负担。

### 用法与场景

| 场景 | 庙算法 | 具体操作 |
|------|--------|----------|
| 商业竞争 | 对标七计 | 列双方：品牌/团队/渠道/现金流/合规/训练/激励，逐项 1–5 分 |
| 创业立项 | 五事自检 | 用户真需求（道）、窗口期（天）、赛道（地）、班子（将）、运营（法） |
| 并购决策 | 知彼知己 | 尽调=知彼；现金流与整合能力=知己；无算不签 |
| 个人择业 | 七计择业 | 平台、上级、成长、薪酬、训练、文化、赏罚——勿无算而入 |
| 产品立项 | 庙算会 | 会前每人独立打分，会中对齐假设，再定做/不做 |
| 危机应对 | 三问 | 我们掌握什么？对手掌握什么？情绪是否在替我们「无算」？ |

## 二、作战 · 谋攻 — 全胜与速决

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/02-mougong.svg' | relative_url }}" alt="谋攻全胜层次示意图" />
<figcaption>图 2 — 谋攻篇：伐谋为上，攻城为下</figcaption>
</figure>

### 语境要点

- **作战篇**：「国之大事，不可不察」的延伸——久战耗国：「久暴师则国用不足」「兵闻拙速，未睹巧之久也」。
- **谋攻篇**：取胜有层次——**全国为上，破国次之；全旅为上，破旅次之**。
- **伐谋**：破敌之谋于庙堂，不战而屈人之兵，为上上策。
- **攻城**：「其下攻城」——消耗最大，非不得已勿用。
- **知胜有五**：知可战不可战、识众寡之用、上下同欲、以御待怠、将能而君不御——五条皆组织问题。

### 深度解析

谋攻篇定义**胜利的质量**，而不只是胜负：

| 层次 | 含义 | 现代镜 |
|------|------|--------|
| 伐谋 | 破对方战略意图 | 标准、专利、生态、监管叙事 |
| 伐交 | 破对方联盟 | 供应链、渠道、合作伙伴 |
| 伐兵 | 正面消耗 | 价格战、人才战 |
| 攻城 | 摧毁对方根基 | 烧钱抢市场、恶意诉讼拖垮 |

「全国为上」在现代即：**尽量保留对方与自己可续用的资产**（人才、品牌、客户关系），而非焦土。与 [商君书]({{ '/2026/05/13/shangjunshu/' | relative_url }})「农战」之刚性可对照：秦以破国之法兴，亦以苛急速亡——见通鉴秦镜。

### 名句

| 原文 | 要义 |
|------|------|
| 上兵伐谋，其次伐交，其次伐兵，其下攻城 | 战略优先级 |
| 不战而屈人之兵，善之善者也 | 最高胜利形态 |
| 以十围之，以五攻之，以分击之 | 兵力对比与打法 |
| 知胜有五：知可以战与不可以战者胜 | 知止是胜机之一 |
| 兵闻拙速，未睹巧之久也 | 久战不利，宜速决 |
| 凡用兵之法，全国为上 | 全胜优于破敌 |

### 典型史案

**墨子止楚攻宋**  
公输班为楚作云梯攻宋，墨子行十日十夜至郢，**伐谋**于殿上：以攻防推演示楚攻必败，又令弟子守宋备械。楚王乃止。未费一兵，而全宋之国——典型「不战而屈人」。

**孙膑马陵道**  
孙膑减灶诱庞涓，不与其主力硬撼，而以谋设伏全歼魏军，虏太子申。胜在**知可以战与不可以战**——避其锋芒，击其必救之虚。

**反例：战国攻城战**  
屡攻坚城、尸积如山，国力耗尽而为他人作嫁——谋攻篇所戒之下策。

### 用法与场景

| 场景 | 谋攻术 | 具体操作 |
|------|--------|----------|
| 市场竞争 | 伐谋优先 | 先争标准、专利、开发者生态，慎打价格战 |
| 产品策略 | 全胜思维 | 用体验与网络效应取胜，非烧钱硬砸渠道 |
| 谈判 | 破谋再谈 | 先瓦解对方最优 BATNA，再谈条款 |
| 诉讼纠纷 | 速决与调解 | 调解、仲裁优于三年诉讼——「久暴师则国用不足」 |
| 组织冲突 | 伐交 | 对齐共同上级与利益相关方，再处理当事人 |
| 裁员收缩 | 全国为上 | 保留核心能力与声誉，非羞辱式离场 |

## 三、军形 · 兵势 — 先为不可胜

### 语境要点

- **军形篇**：「胜兵先胜而后求战，败兵先战而后求胜。」先立于不败之地，再待敌之可胜。
- **不可胜在己，可胜在敌**：我能守，敌有隙，方能攻。
- **兵势篇**：「激水之疾，至于漂石者，势也」——组织**势能**与**节奏**，使「如转圆石于千仞之山」。
- **奇正相生**：以正合，以奇胜——常规部队牵制，奇兵制胜。
- **善战者，求之于势，不责于人**：系统优于苛责个人——与 [The Algorithm]({{ '/2026/06/14/the-algorithm-jon-mcneill/' | relative_url }})「删繁就简、造系统」同路。

### 深度解析

形与势是孙子**组织论**核心：

```text
形 = 可防守的结构与资源底线（不可胜在己）
势 = 释放能量的坡度与节奏（如转圆石于千仞之山）
正 = 明面、稳态、牵制
奇 = 变异、奇袭、差异化
```

投资中的「安全边际」、产品中的「核心功能稳 + 创新特性奇」、管理中的「流程造势」皆出于此。与 [菜根谭]({{ '/2026/06/03/caigentan/' | relative_url }})「鹰立如睡」互参：形全不必张扬，势至则一击。

### 名句

| 原文 | 要义 |
|------|------|
| 胜兵先胜而后求战 | 先备后战 |
| 守而不失，攻而必取 | 形全则守，胜有余力则攻 |
| 以正合，以奇胜 | 正兵接战，奇兵出奇 |
| 善战者，求之于势，不责于人 | 重系统与势能，非苛责个人 |
| 激水之疾，至于漂石者，势也 | 势能来自坡度与积累 |
| 乱生于治，怯生于勇，弱生于强 | 奇正相生，不可偏废 |

### 典型史案

**韩信背水一战**  
韩信背水列阵，士卒「知亡无死，勿生之兵」，势如激水；又遣奇兵袭赵营，两面夹击。非蛮勇，而是**造势**使人无退、使敌失措。

**李世民虎牢关**  
窦建德援王世充，世民待其师疲食尽方出战，以少击众。先据虎牢之**形胜**，再借敌之怠——「先为不可胜」而后一击。

**反例：项羽垓下**  
前期势如破竹，后期无稳固后方、无联盟、无退路设计——形未全而屡攻，终至四面楚歌。见 [资治通鉴]({{ '/2026/05/25/zizhi-tongjian/' | relative_url }})楚汉纪。

### 用法与场景

| 场景 | 形势术 | 具体操作 |
|------|--------|----------|
| 现金流管理 | 先胜底线 | 先保 18 个月 runway，再谈扩张 |
| 产品发布 | 正奇组合 | 正兵：主功能稳态；奇兵：颠覆特性打差异 |
| 团队管理 | 造势 | 流程、工具、信息透明，减英雄主义依赖 |
| 投资 | 安全边际 | 先问「最坏能否活」，再谈 upside |
| 个人职业 | 不可胜在己 | 技能组合 + 财务缓冲，再择进攻时机 |
| 增长 | 坡度思维 | 先积累用户/数据/品牌坡度，再释放活动 |

## 四、虚实 · 军争 — 形人而我无形

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/03-xushi.svg' | relative_url }}" alt="虚实篇示意图" />
<figcaption>图 3 — 虚实篇：致人而不致于人，击虚避实</figcaption>
</figure>

### 语境要点

- **虚实篇**：「致人而不致于人」——调动敌人，而不被敌人调动。
- **形人而我无形**：使敌备于处处，我攻于不备——「吾所与战之地不可知，不可知则敌所备者多」。
- **军争篇**：「以迂为直，以患为利」——争利之道，争的是**先机**与**要道**，非意气之争。
- **军争之难**：「先处战地而待敌者佚，后至战地而趋战者劳。」
- **避实击虚**：不在对手最强处硬撼——与 [鬼谷子]({{ '/2026/04/27/guiguzi/' | relative_url }})抵巇「乘隙」相通，孙子更重**节奏与主动权**。

### 深度解析

虚实是**信息战 + 节奏战**：

| 原则 | 操作含义 |
|------|----------|
| 致人 | 设议题、设战场、设时间表 |
| 不致于人 | 不跟对手节奏跑 |
| 形人 | 让对手暴露部署、分散资源 |
| 我无形 | 意图与主攻点不明 |
| 以迂为直 | 表面绕远，实则占先 |

### 名句

| 原文 | 要义 |
|------|------|
| 致人而不致于人 | 掌握主动权 |
| 攻其无备，出其不意 | 击虚原则 |
| 避实而击虚 | 不在其强项硬撼 |
| 以迂为直，以患为利 | 表面迂回，实则占先 |
| 先处战地而待敌者佚 | 抢先布局则从容 |
| 吾所与战之地不可知 | 主攻点保密则敌备分散 |

### 典型史案

**白起长平**  
赵军坚壁不出，秦使间言换赵括为将；赵括出击，秦断粮道，击其不能之虚——赵军四十万溃。虚实转换在**诱敌离险**。（读史宜与 [通鉴]({{ '/2026/05/25/zizhi-tongjian/' | relative_url }})辨因果，忌故事会。）

**围魏救赵**（孙膑）  
魏攻赵，齐不直救邯郸而攻大梁，魏军回援，中途于桂陵伏击。**致人而不致于人**——打其必救，掌握节奏。

**反例：曹操赤壁**  
强势水师却链船、轻疫患、轻联盟——实变虚，被火攻一击。见通鉴谋断篇。

### 用法与场景

| 场景 | 虚实术 | 具体操作 |
|------|--------|----------|
| 竞品策略 | 错位 | 避旗舰锋芒，攻薄弱品类/区域/人群 |
| 营销 | 声东击西 | 话题在 A，转化在 B；测试主攻点保密 |
| 项目管理 | 抢先 | 关键路径资源先占，别等项目救火 |
| 谈判 | 迂直 | 表面让步非核心，换核心条款；定议程 |
| 求职 | 致人 | 让多家并行 offer，掌握节奏 |
| 诉讼 | 管辖与时机 | 选有利战场与时间，非被动应诉 |

## 五、九变 · 地形 · 九地 — 知地因粮

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/04-dixing.svg' | relative_url }}" alt="地形九地示意图" />
<figcaption>图 4 — 地形六类、九地态势：因地制宜</figcaption>
</figure>

### 语境要点

- **九变篇**：将能「通变」——有所不争（城不争、地不争、主命不争），有所必为。
- **地形篇**：通、挂、支、隘、险、远六地，战法各异；「地形者，兵之助也。」
- **九地篇**：散地、轻地、争地、交地、衢地、重地、圮地、死地、亡地——按深入程度与退路定军心。
- **因粮于敌**：「智将务食于敌，食敌一钟，当吾二十钟。」——取资于敌，减己方消耗。
- **主不可以怒而兴师**：[菜根谭]({{ '/2026/06/03/caigentan/' | relative_url }})「慎怒」与孙子「慎战」在此汇合。

### 深度解析

地利篇是孙子的**情境智能**：

| 九地类型 | 军心特征 | 组织镜 |
|----------|----------|--------|
| 散地 | 志不专 | 远程团队各自为战 |
| 重地 | 深入敌境 | all-in 新市场，无退路 |
| 死地 | 亡则存 | 创业生死线项目 |
| 衢地 | 交争之地 | 平台型市场多方博弈 |

「九变」回答：**哪些仗不打**——有时最大的胜是止损与撤退。

### 名句

| 原文 | 要义 |
|------|------|
| 将通于九变之利者，知用兵矣 | 权变非无原则 |
| 投之亡地然后存，陷之死地然后生 | 绝境激势（慎用） |
| 知地形，得地者之利 | 地利须匹配打法 |
| 主不可以怒而兴师，将不可以愠而致战 | 情绪不可启动战争 |
| 智将务食于敌 | 借力、用生态减消耗 |
| 无所不备则无所不寡 | 处处防守则处处薄弱 |

### 典型史案

**马谡失街亭**  
诸葛亮伐魏，命马谡守街亭。谡据山不下寨，为张郃断水围击——**不知地形：隘险之地却自绝粮道**。一失要冲，全盘受挫。

**曹操乌林之前**  
曹操军染疫、士卒不服水土，却强行连船东进；孙刘联军借风火与江形破之。不知**九地**（重地、交地）之险，又违「主不可以怒而兴师」。

**反例：诸葛亮北伐**  
国力不济仍屡出祁山——战略企图与资源地形错位，忠臣亦难挽。见通鉴出师表镜。

### 用法与场景

| 场景 | 地利术 | 具体操作 |
|------|--------|----------|
| 市场进入 | 辨九地 | 易进易退=轻地；只能赢不能输=死地——匹配投入 |
| 远程办公 | 防散地 | 加强节奏、共同目标、同步仪式 |
| 供应链 | 因粮于敌 | 本地化、伙伴生态，减长途与单一依赖 |
| 危机决策 | 九变取舍 | 列「必争/不争」清单，勿争面子之战 |
| 区域扩张 | 地形六类 | 通地快打、险地慎入、远地重后勤 |
| 情绪决策 | 怒愠检查 | 重大决策 24 小时冷却期 |

## 六、火攻 · 用间 — 先知与慎战

<figure class="diagram-figure">
<img src="{{ '/assets/images/sunzi/05-yongjian.svg' | relative_url }}" alt="用间五间示意图" />
<figcaption>图 5 — 用间篇：五间并用，慎战明主</figcaption>
</figure>

### 语境要点

- **火攻篇**：火攻有五：火人、火积、火辎、火库、火队——**因五火之变而应之**，亦戒「主慎之」。
- **用间篇**：「明君贤将，所以动而胜人，成功出于众者，先知也。」
- **五间**：因间、内间、反间、死间、生间——情报体系，非单点间谍。
- **慎战总结**：「非利不动，非得不用，非危不战」；「主不可以怒而兴师，将不可以愠而致战。」
- **先知不可取于鬼神**：靠人谋与系统——与 [周易]({{ '/2026/05/07/zhouyi/' | relative_url }})占筮互补：易观势，孙子重实证情报。

### 深度解析

全书收束于**信息 + 克制**：

```text
五间（情报网络）→ 先知（决策优势）→ 火攻（杠杆工具，慎用）
→ 非利不动/非得不用/非危不战（出手纪律）
```

现代「火攻」可以是：一次性资源集中、公关战役、产品大版本——前提是**先知**确认时机与对手脆弱点，而非赌运气。

### 名句

| 原文 | 要义 |
|------|------|
| 知彼知己，胜乃不殆 | 全书反复之核 |
| 先知者，不可取于鬼神 | 靠人谋与系统，非玄学 |
| 非利不动，非得不用，非危不战 | 出手三条件 |
| 亡国不可以复存，死者不可以复生 | 慎战终极告诫 |
| 因间者，因其官人而用之 | 情报要嵌入结构 |
| 主不可怒而兴师 | 情绪与战争绝缘 |

### 典型史案

**赤壁之战**  
周瑜、黄盖用火攻，前提却是**用间**（蒋干、蔡瑁张允）与**先知**（知曹军不习水战、连船之弊）。火攻是「五火之变」的应法，非孤注一掷。

**郑国烛之武退秦师**  
秦晋围郑，烛之武夜见秦穆公，陈亡郑无益于秦而益于晋，秦遂退。**伐谋伐交**之极——一言退敌，胜在情报与外交判断。

**反例：秦二世**  
信息垄断、指鹿为马、忠臣失声——先知系统崩溃，火攻般速亡。见通鉴。

### 用法与场景

| 场景 | 用间术（正当情报） | 具体操作 |
|------|---------------------|----------|
| 竞争分析 | 先知系统 | 公开信息、行业报告、用户访谈、合规渠道——建仪表盘 |
| 反间意识 | 防误导 | 警惕对手假定价、假新闻、钓鱼问卷 |
| 重大投资 | 三不原则 | 非利不动、非得不用、非危不战——写进 IC 备忘录 |
| 组织决策 | 怒愠禁令 | 领导愤怒时不拍板战争级项目；设冷却期 |
| 产品大促 | 火攻时机 | 确认竞品空窗、供应链就绪、团队能承受一击 |
| 个人决策 | 鬼神检查 | 重大选择是否靠「感觉/占卜」替代尽职调查？ |

## 阅读边界

读《孙子兵法》宜：

1. **明胜道，不泥诈道** — 「诡道」是兵法，不是无底线人生哲学
2. **慎战为本** — 非利不动、主不可怒；现代映射到：不冲动 All-in
3. **古今异器** — 车战、攻城与今日商业不同，宜取**结构原则**（庙算、虚实、先知）
4. **与道德互补** — 孙子教「如何胜」，《论语》教「为何战」；胜而不义，终不可久
5. **忌兵法口号化** — 未填七计表就言「知彼知己」，易成表演

## 常见误读与纠正

| 误读 | 纠正 |
|------|------|
| 「兵不厌诈」= 什么都可以骗 | 战术示形 ≠ 违约欺诈 |
