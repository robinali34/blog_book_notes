## 经传 49–56（革至旅）

### 49 · 革 ䷰

**卦象**：兑上离下

**卦辞**

> 巳日乃孚，元亨利贞，悔亡。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 巩用黄牛之革。 |
| 六二 | 巳日乃革之，征吉，无咎。 |
| 九三 | 征凶，贞厉；革言三就，有孚。 |
| 九四 | 悔亡，有孚改命，吉。 |
| 九五 | 大人虎变，未占有孚。 |
| 上六 | 君子豹变，小人革面；征凶，居贞吉。 |

**要义**：革故鼎新，己日乃孚

---

### 50 · 鼎 ䷱

**卦象**：离上巽下

**卦辞**

> 元吉，亨。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 鼎颠趾，利出否；得妾以其子，无咎。 |
| 九二 | 鼎有实，我仇有疾，不我能即，吉。 |
| 九三 | 鼎耳革，其行塞。雉膏不食；方雨亏悔，终吉。 |
| 九四 | 鼎折足，覆公餗，其形渥，凶。 |
| 六五 | 鼎黄耳，金铉，利贞。 |
| 上九 | 鼎玉铉，大吉，无不利。 |

**要义**：正位凝命，大亨以正

---

### 51 · 震 ䷲

**卦象**：震上震下

**卦辞**

> 亨。震来虩虩，笑言哑哑。震惊百里，不丧匕鬯。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 震来虩虩，后笑言哑哑，吉。 |
| 六二 | 震来厉，亿丧贝，跻于九陵，勿逐，七日得。 |
| 六三 | 震苏苏，震行无眚。 |
| 九四 | 震遂泥。 |
| 六五 | 震往来，厉；亿无丧，有事。 |
| 上六 | 震索索，视矍矍，征凶。震不于其躬，于其邻，无咎。婚媾有言。 |

**要义**：恐惧修省，震不丧匕鬯

---

### 52 · 艮 ䷳

**卦象**：艮上艮下

**卦辞**

> 艮其背，不获其身；行其庭，不见其人，无咎。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 艮其趾，无咎，利永贞。 |
| 六二 | 艮其腓，不拯其随，其心不快。 |
| 九三 | 艮其限，列其夤，厉熏心。 |
| 六四 | 艮其身，无咎。 |
| 六五 | 艮其辅，言有序，悔亡。 |
| 上九 | 敦艮，吉。 |

**要义**：止其所当止，思不出其位

---

### 53 · 渐 ䷴

**卦象**：巽上艮下

**卦辞**

> 女归吉，利贞。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 鸿渐于干；小子厉，有言，无咎。 |
| 六二 | 鸿渐于磐，饮食衎衎，吉。 |
| 九三 | 鸿渐于陆，夫征不复，妇孕不育，凶；利御寇。 |
| 六四 | 鸿渐于木，或得其桷，无咎。 |
| 九五 | 鸿渐于陵，妇三岁不孕；终莫之胜，吉。 |
| 上九 | 鸿渐于陆，其羽可用为仪，吉。 |

**要义**：女归吉，渐进之道

---

### 54 · 归妹 ䷵

**卦象**：震上兑下

**卦辞**

> 征凶，无攸利。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 归妹以娣，跛能履，征吉。 |
| 九二 | 眇能视，利幽人之贞。 |
| 六三 | 归妹以须，反归以娣。 |
| 九四 | 归妹愆期，迟归有时。 |
| 六五 | 帝乙归妹，其君之袂，不如其娣之袂良；月几望，吉。 |
| 上六 | 女承筐，无实；士刲羊，无血。无攸利。 |

**要义**：征凶，无攸利

---

### 55 · 丰 ䷶

**卦象**：震上离下

**卦辞**

> 亨，王假之；勿忧，宜日中。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初九 | 遇其配主，虽旬无咎，往有尚。 |
| 六二 | 丰其蔀，日中见斗。往得疑疾，有孚发若，吉。 |
| 九三 | 丰其沛，日中见沬；折其右肱，无咎。 |
| 九四 | 丰其蔀，日中见斗，遇其夷主，吉。 |
| 六五 | 来章，有庆誉，吉。 |
| 上六 | 丰其屋，蔀其家，窥其户，阒其无人，三岁不觌，凶。 |

**要义**：丰大，日中则昃

---

### 56 · 旅 ䷷

**卦象**：离上艮下

**卦辞**

> 小亨，旅贞吉。

**爻辞**

| 爻 | 原文 |
|----|------|
| 初六 | 旅琐琐，斯其所取灾。 |
| 六二 | 旅即次，怀其资，得童仆，贞。 |
| 九三 | 旅焚其次，丧其童仆，贞厉。 |
| 九四 | 旅于处，得其资斧，我心不快。 |
| 六五 | 射雉，一矢亡；终以誉命。 |
| 上九 | 鸟焚其巢，旅人先笑后号啕；丧牛于易，凶。 |

**要义**：羁旅慎行，小亨
