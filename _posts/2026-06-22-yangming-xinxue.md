## 全书概览

阳明心学不是空谈心性，而是一套**在事上成德**的实践哲学：良知人人本有，却被私欲遮蔽；须在具体事务中**体认、扩充、践行**，方称真知。与程朱理学向外穷理不同，阳明把功夫收束到**此心**——理不在心外，善恶之几不在事后。

> 无善无恶心之体，有善有恶意之动。知善知恶是良知，为善去恶是格物。

> 破山中贼易，破心中贼难。

> 知是行之始，行是知之成。

读阳明，宜记三问：**良知此刻是否清明？知与行是否脱节？最难的一件事上是否在磨练？**

### 龙场语境：困厄何以成学？

正德初年，刘瑾专权，阳明因忤旨被杖、贬贵州龙场驿丞。瘴疠、石棺、仆从病死——外在荣辱与身家俱绝。正是在此绝境，他悟出「圣人之道，吾性自足」：

| 压力 | 阳明式回应 |
|------|------------|
| 外求理而不得 | 心即理：理在良知，不在外物训诂 |
| 知而不行 | 知行合一：真知本含行意 |
| 静坐空想 | 事上磨练：在事务中验良知 |
| 善恶纷杂 | 致良知：扩充本体，为善去恶 |
| 志气短促 | 立志：志圣贤则近圣贤 |

它不是逃避现实，而是在现实之中**把功夫落在当下这一念、这一事**——与 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }})「日课校正」、[管理的本质]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})「可持续交换」可并读：内修诚意，外立结构。

## 一、心即理 — 龙场悟道

### 语境要点

- 贬谪龙场，石棺旁悟：「圣人之道，吾性自足，向之求理于事物者误也。」
- **心外无理，心外无物**——不是否定事物，而是理须在此心良知上体认；物接于心而有意。
- 反对程朱「格物穷理」流于训诂、支离；「心之本体，原自不动」——不被外境牵夺主宰。

### 深度解析

心即理解决**价值源头**：程朱一路「事物 → 穷理 → 存天理」；阳明一路「良知 → 即理 → 事上体认」。不是取消外物，而是反对**把心丢在外面找**。与 [周易]({{ '/2026/05/07/zhouyi.html' | relative_url }})「察势」互补：易看时机变局，阳明问**此心当如何应**。与 [荀子]({{ '/2026/06/18/xunzi.html' | relative_url }})对照：性恶给制度设计，性善给方向与诚意。

### 名句

| 原文 | 要义 |
|------|------|
| 心外无理，心外无物 | 理在良知，不在外求 |
| 圣人之道，吾性自足 | 成德资源本具，困厄可发明 |
| 心之本体，原自不动 | 定力非无情，是不被境转 |
| 意之所在，即物之所在 | 物接于心，在事上见理 |

### 典型史案

**龙场悟道（正德三年）**  
阳明于贵州龙场驿，身处绝境，忽悟「圣人之道，吾性自足」。——**逆境常是向内转的契机**；与 [菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})「咬得菜根」异曲同工，路向更哲学化。

**格竹之困（早年）**  
阳明早年按朱熹说格庭前之竹，七日不得要领而病。——外求理之弊，亲身经验；后转心学，格物义亦随之翻新（见第五节）。

### 用法与场景

| 场景 | 心即理术 | 具体操作 |
|------|----------|----------|
| 职业迷茫 | 内省志 | 问「何为正当之志」，非先问头衔 |
| 道德两难 | 良知一问 | 此事若人尽知，可坦然否？ |
| 被毁誉牵动 | 不动体 | 誉毁过心，问「动的是事还是我执」 |
| 重大决策 | 势心并修 | [周易]({{ '/2026/05/07/zhouyi.html' | relative_url }})察势 + 良知定夺 |

### 反模式（误用）

| 误用 | 后果 | 纠正 |
|------|------|------|
| 以心即理护短 | 私欲当良知 | 须「知善知恶」验 |
| 否定外物 | 不理事 | 在事上见心，非离事 |

**口诀**：心外无理，性自足；困中能悟，事上见真。

## 二、致良知 — 成德之本

### 语境要点

- 良知：不虑而知，不学而能——孟子四端（恻隐、羞恶、辞让、是非）之心学展开。
- 「致」：扩充、实现、贯彻到底；私欲如蔽日之云，致良知即扫障复明。
- 「此心不动，随机而动」——良知清明则应变不滞；致良知是功夫总名，贯穿格物、立志、知行合一。

### 深度解析

致良知是**操作系统内核**：本体（无善无恶心之体）→ 意动（有善有恶）→ 良知（知善知恶）→ 格物（为善去恶）。与 [认知神经科学]({{ '/2026/05/12/gazzaniga-cognitive-neuroscience.html' | relative_url }})对读：脑会事后为行为编叙事；致良知是在**行动前**加「知善知恶」检校。与 [DBT 技能训练手册]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }})并用：DBT 稳情绪，良知定方向。

### 名句

| 原文 | 要义 |
|------|------|
| 致良知 | 成德功夫总名 |
| 此心不动，随机而动 | 定力与应变 |
| 知善知恶是良知 | 善恶之几，心自知之 |
| 满街都是圣人 | 良知本具，蔽则不显 |

### 典型史案

**天泉论道**  
钱德洪、王畿论四句教理解不一，阳明令各说所得，同证「无善无恶心之体」——**同一教义，须在事上各证**，非死背口号。

**南赣教化**  
平盗之余立乡约、兴学校——致良知不是闭门冥想，而是**化民成俗**的政教实践。

### 用法与场景

| 场景 | 致良知术 |
|------|----------|
| 伦理抉择 | 底线不在规章才想起，而在日常习惯 |
| 冲突前 | 问：我在欺人还是欺己？（见 [传习录]({{ '/2026/06/17/chuanxilu-wang-yangming.html' | relative_url }})） |
| 情绪上头 | 命名私欲：贪、惧、怒、名——再为善去恶 |
| 人际沟通 | 与 [非暴力沟通]({{ '/2026/06/01-nonviolent-communication-rosenberg.html' | relative_url }})「观察实情」同路 |

### 反模式

| 误用 | 后果 |
|------|------|
| 良知=自我感觉 | 把私欲当天理 |
| 只致不行 | 仍是知而不行 |

**口诀**：知善知恶是良知，扩充到底在事上。

## 三、知行合一 — 真知即所以为行

### 语境要点

- 批评「知行分离」：知而不行，只是未知。「知是行之始，行是知之成。」——行是知的完成形态。
- **真知本含行意**；见好色已好，见恶臭已恶，便是知行合一。
- 「一念发动处，便即是行」——恶念之萌亦是行之一端，故须在念起时格物。

### 深度解析

| 状态 | 特征 | 阳明判词 |
|------|------|----------|
| 真知 | 见善即行，见恶即止 | 知是行之始 |
| 伪知 | 口说仁义，身不践行 | 只是未知 |
| 盲行 | 无省察的冲动 | 须良知先导 |

与 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }})：复盘验「以为知」与「实为行」之差距。与 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }})：阿德勒重课题分离；阳明重**知与行本一体**。

### 名句

| 原文 | 要义 |
|------|------|
| 知是行之始，行是知之成 | 行是知的完成 |
| 知而不行，只是未知 | 拆穿伪知 |
| 一念发动处，便即是行 | 微念处下功夫 |
| 圣学只一功夫 | 知行不可分作两事 |

### 典型史案

**龙场教俗**  
阳明于龙场立约、教民、平讼——悟道后不闭关，即**在事务中展开知**。知行合一非口号，是治理实践。

**平定宁王（正德十四年）**  
宁王朱宸濠起兵，阳明赴任途中募兵、假檄、奇袭——**知到深处不得不行**；事功是知的当场展开。

### 用法与场景

| 场景 | 知行合一术 |
|------|------------|
| 学习 | 读完即练，练不成则未真懂 |
| 目标管理 | 指标若与日常行为无关，即「未知」 |
| 习惯养成 | 小善即行，不等「准备好」 |
| 周复盘 | 列「以为知 / 实际行」对照表 |

### 反模式

| 误用 | 后果 |
|------|------|
| 以行代知 | 盲动无省察 |
| 等「想明白」再动 | 永不行 |

**口诀**：不行不足以为知，知到深处即是行。

## 四、事上磨练 — 在事上用功

### 语境要点

- 「人须在事上磨练做功夫，方有益。」静坐非无用，但易流于空想；**事**是检验良知之镜。
- 军旅、吏事、平盗、平叛，皆是心学战场——阳明一生「破贼」与「讲学」并行。
- 「在事上用功，方是格物」——困厄、冲突、责任重压，皆是磨练良机。

### 深度解析

| 类型 | 特征 | 风险 |
|------|------|------|
| **事修** | 在真实任务中验良知 | 正路 |
| **坐修** | 仅静坐观想 | 易离事 |
| **言修** | 讲学自嗨 | 易伪知 |

[传习录]({{ '/2026/06/17/chuanxilu-wang-yangming.html' | relative_url }})大量问答皆从「某友问某事」起——心学格式本是**案例驱动**。与 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})：管理在交换结构中磨练识人与担责；阳明磨练**此心是否真诚**。

### 名句

| 原文 | 要义 |
|------|------|
| 人须在事上磨练做功夫 | 事是道场 |
| 在事上用功，方是格物 | 格物在应事 |
| 尔那一点良知，是尔自家的准则 | 事上照此准则 |
| 毁誉不动心，然后能应事 | 磨练目标是应事不乱 |

### 典型史案

**南赣平盗**  
阳明于南赣设十家牌法、申明亭，既治盗又教化——**硬事与软教并行**，事上磨练之政事实录。

**平定宁王**  
军情危急，随机而动——最高压之事，正是知行合一与致良知的考场。

### 用法与场景

| 场景 | 事上磨练术 |
|------|------------|
| 管理 | 在项目中修领导力，非等「有空进修」 |
| 冲突 | 最难对话正是磨练处 |
| 逆境 | 贬谪、失败、边缘化——问「性自足」是否真 |
| 技能学习 | 在真实任务里练，非只模拟 |

### 反模式

| 误用 | 后果 |
|------|------|
| 以忙代修 | 事多而不省 |
| 逃避难事 | 失磨练场 |

**口诀**：静坐亦功，事上为要；应事不乱，方见真知。

## 五、格物新解 — 为善去恶

### 语境要点

- 阳明释《大学》格物：「为善去恶是格物。」非格竹子的外在穷理，而是**每一念、每一事上为善去恶**。
- 「格」者，正也——正其不正，即格物。与朱熹争点：朱重即物穷理；王重**正心诚意在念动处**。

### 深度解析

外格一路：即物 → 穷理 → 积累知识。内格一路：意动 → 知善知恶 → 为善去恶（即格物）。每日微修：见一念嫉妒、一语推诿——**知恶即去，知善即行**，便是一日格物。与 [菜根谭]({{ '/2026/06/03/caigentan.html' | relative_url }})「念头才萌，鬼神已听」同频。

### 名句

| 原文 | 要义 |
|------|------|
| 为善去恶是格物 | 格物即道德抉择 |
| 格者，正也 | 正其不正 |
| 意之所在，即物之所在 | 无离心外之物 |
| 实有格物之功 | 在事上验，非在纸上 |

### 典型史案

**格竹之困与转向**  
早年格竹不得，后悟格物不在竹而在**为善去恶**——思想史公案，示外求之限。

**南赣乡约**  
以乡约规范民俗——格物从一身扩展到**化俗成俗**，非仅内心戏。

### 用法与场景

| 场景 | 格物术 |
|------|--------|
| 微习惯 | 小恶不纵，小善即行 |
| 每日收工 | 今日一恶念、一善行 |
| 会议 | 欲推诿时问：为善去恶何在？ |
| 言语 | 恶语将出未出，即格 |

### 反模式

| 误用 | 后果 |
|------|------|
| 格物=读书累篇 | 重蹈程朱支离 |
| 只格人不格己 | 道德审判 |

**口诀**：为善去恶是格物，一念发动处下功。

## 六、破心中贼 — 破山中贼易

### 语境要点

- 「破山中贼易，破心中贼难。」——外在敌人可见，私欲、恐惧、功名心难察。
- 山中贼：宁王、盗贼；心中贼：骄傲、贪欲、惧毁、巧诈。平宁王后功高遭忌——**破贼之后，更难破名**。
- 心学最难不在战场，在**胜后、闲后、宠后**。

### 深度解析

| 贼名 | 表现 | 破法 |
|------|------|------|
| **名** | 好誉、畏毁 | 毁誉不动 |
| **利** | 贪取、计较 | 义利之辨 |
| **惧** | 畏难、避责 | 事上磨练 |
| **怒** | 一时之忿 | 此心不动 |

与 [说「不」的艺术]({{ '/2026/05/21-art-of-saying-no-refusal.html' | relative_url }})：拒绝对外是边界；破心中贼是对内——**惧得罪、贪认可**常是说不出口的根。与 [鬼谷子]({{ '/2026/04/27/guiguzi.html' | relative_url }})：纵横术可破外局；阳明补**破内贼**。

### 名句

| 原文 | 要义 |
|------|------|
| 破山中贼易，破心中贼难 | 内敌更险 |
| 此心不动 | 破贼之要 |
| 去人欲，存天理 | 去蔽非禁欲 |
| 功高遭忌 | 名贼之果 |

### 典型史案

**平定宁王朱宸濠**  
宁王起兵，阳明奇袭平定——「山中贼」破之速；事后功高遭忌、谗言四起，**心中贼**之考更长。

**龙场前後**  
贬谪前争名于官场；龙场后转向向内——绝境破一层心中贼（外求名禄）。

### 用法与场景

| 场景 | 破心中贼术 |
|------|------------|
| 成功后 | 问：骄傲是否起？名贼在否？ |
| 被批评 | 问：惧毁还是真改？ |
| 拒绝请求 | 察惧得罪之贼 |
| 团队胜利 | 分功、让名，破独揽功之贼 |

### 反模式

| 误用 | 后果 |
|------|------|
| 只破外敌 | 内贼依旧 |
| 以破贼为名行凶 | 私欲当正义 |

**口诀**：破山中贼易，破心中贼难；此心不动，贼无所栖。

## 七、立志 — 学做圣人亦可

### 语境要点

- 「立志而圣，犹志粮夫。」——志于圣贤，终近圣贤；志于庸常，终近庸常。
- 少年阳明问「何为第一等事」，不满足于「读书登第」——**志先于术**。
- 「志不立，天下无可成之事」——立志是功夫起点与价值锚点。

### 深度解析

粮夫之志终日为糊口；圣贤之志则**事上亦为成德**。志高不等于职位高；扫地、理事、带兵皆可同一路。与 [被讨厌的勇气]({{ '/2026/05/08/courage-to-be-disliked-kishimi.html' | relative_url }})：阿德勒问关系边界；阳明问**此生所志**。

### 名句

| 原文 | 要义 |
|------|------|
| 立志而圣，犹志粮夫 | 志高行高 |
| 志不立，天下无可成之事 | 志为功夫根 |
| 夫学贵得之心 | 志须真，非饰 |
| 圣人只是常人而能致良知 | 志圣非绝远 |

### 典型史案

**少年问第一等事**  
幼时问师「何为第一等事」，师曰「读书登第」，阳明曰「恐未是第一等事」——**立志先于技巧**。

**龙场后讲学**  
立志不在庙堂，在**人人可为圣贤**之教——平民亦可致良知。

### 用法与场景

| 场景 | 立志术 |
|------|--------|
| 职业选择 | 问：此事合我志否？ |
| 日课开篇 | 今日一事，志于圣贤还是敷衍？ |
| 逆境 | 贬谪仍不坠志——龙场模型 |
| 中年倦怠 | 重审志，非换噱头 |

### 反模式

| 误用 | 后果 |
|------|------|
| 立志成口号 | 日课无志 |
| 鄙视常业 | 圣学在事上，非避事 |

**口诀**：立志而圣，犹志粮夫；志高一寸，行远一程。

## 八、四句教 — 晚年总纲

### 语境要点

- 晚年总结：**无善无恶心之体，有善有恶意之动。知善知恶是良知，为善去恶是格物。**
- 「无善无恶」指心体未发之中，非谓行事无善恶——误解最盛处，宜慎读。
- 天泉论道：钱德洪、王畿各执一端，阳明令互补——**教义须活**，不可执半句。

### 深度解析

| 句 | 指涉 | 常见误读 |
|----|------|----------|
| 无善无恶心之体 | 未发之中 | ≠ 无道德 |
| 有善有恶意之动 | 念起处 | ≠ 善恶相对主义 |
| 知善知恶是良知 | 判断几 | ≠ 我个人感觉 |
| 为善去恶是格物 | 实践 | ≠ 只内心不行动 |

王畿「四无」与钱德洪「四有」之争——阳明示：**上根下根不同，须各尽所能**。与 [四镜头分析]({{ '/2026/05/25/four-lenses-knowing-choosing.html' | relative_url }})联用，可作总复习框架。

### 名句

| 原文 | 要义 |
|------|------|
| 无善无恶心之体 | 体之未动，非虚无 |
| 有善有恶意之动 | 念起即见善恶 |
| 知善知恶是良知 | 道德自觉 |
| 为善去恶是格物 | 落在行上 |

### 典型史案

**天泉论道（嘉靖六年）**  
阳明将逝，门人问教法传承。钱德洪主「四有」，王畿主「四无」，阳明曰「汝等须补对，各认一边」——**教法须契人**，执一失全。

**后学分歧**  
明末泰州学派等发展四无，被批评流荡——示四句教**宜深不宜滑**。

### 用法与场景

| 场景 | 四句教术 |
|------|----------|
| 总复习 | 以四句串八主题 |
| 争论「无善无恶」 | 回体用知行四段，防断章 |
| 日课 | 晨起念四句，晚间验第四句 |
| 对读程朱 | 辨格物致知异同 |

### 反模式

| 误用 | 后果 |
|------|------|
| 执「无善无恶」护恶 | 道德虚无 |
| 只讲前三句不行 | 缺格物 |

**口诀**：体用知行四句尽，为善去恶是实功。

## 读阳明四步法

| 步骤 | 做法 |
|------|------|
| 1 | **选一事** — 今日最难或最重要的一件事 |
| 2 | **致良知** — 问：知善知恶否？私欲何在？ |
| 3 | **定一行** — 为善去恶的一小步，今日即做 |
| 4 | **周复盘** — 何处知而不行？何处破心中贼？ |

### 日课模板

```text
【立志】今日一事，志于圣贤还是志于敷衍？
【致良知】何处私欲起？如何为善去恶？
【知行合一】昨日以为知者，今日是否行了？
【事上磨练】最难的一件事是什么？如何在其中用功？
【收工三行】一善 / 一失 / 明日一戒
```

### 周课模板

```text
【本周主题】八题轮换（心即理 → … → 四句教）
【公案对照】本周一事像哪则公案？
【知行之差】列三条「以为知 / 实际行」
【破贼检视】名、利、惧、怒、惰，哪贼最盛？
【下周一行】只选一条功夫深耕
```

可与 [有效复盘]({{ '/2026/05/24/effective-retrospective.html' | relative_url }}) 三行日记、KPT 并用；与 [DBT]({{ '/2026/05/19/dbt-skills-handouts-linehan.html' | relative_url }}) 情绪日记互补——DBT 稳态，阳明定向。

## 阅读边界

读阳明心学，宜知以下边界：

1. **良知非借口** — 勿以「心即理」护短，实为私欲；须「知善知恶」验  
2. **非反制度** — 心学重内，组织仍须 [韩非子]({{ '/2026/05/02/hanfeizi.html' | relative_url }})式底线与 [管理]({{ '/2026/06/02/management-essence-interest-exchange.html' | relative_url }})式结构  
3. **四句教宜深读** — 「无善无恶」指体之未动，非道德虚无  
4. **与程朱差异** — 争点在外逐理 vs 内致良知，可并存互补，非必水火  
5. **事功与修养** — 阳明非纯隐士，勿以「悟」逃避责务  
6. **格言易成口号** — 须日课践行，否则只是讲学谈资  
7. **常见误读** — 「无善无恶」非什么都行；「知行合一」非想到即做到；「立志圣」非好高骛远
