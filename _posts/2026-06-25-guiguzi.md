**《鬼谷子》**  
战国纵横家经典 · 传为鬼谷子（王诩）所著 · 核心弟子：苏秦、张仪、孙膑、庞涓

## 全书概览

《鬼谷子》是中国古代游说、谋略与识人之学的源头之一，与《战国策》互为表里——策士之言，多本于此书。全书十二篇，从**开口闭口**（捭阖）到**揣摩决断**（揣、摩、权、谋、决），构成一套完整的**人际博弈与说服操作系统**。

> 智用于众人之所不能知，而能用于众人之所不能，潜谋于无形，常胜于不争不费。

## 一、捭阖篇 — 开口与闭口

<figure>
<img src="{{ '/assets/images/guiguzi/01-baihe.svg' | relative_url }}" alt="捭阖篇示意图" />
<figcaption>图 1 — 捭阖：一开一闭，辨虚实而定言说之道</figcaption>
</figure>

### 语境要点

- **捭**（开）：张扬、引导、让对方吐露真情
- **阖**（闭）：收敛、守密、让对方自行显露
- 核心：「审定有无与其实虚，同异与其能否」——先辨虚实，再定开阖
- 圣人之道：「一阴一阳之谓道」——言说亦分阴阳，不可一味强攻

### 案例

**张仪戏楚**：张仪以六百里商於之地诱楚怀王绝齐，实为虚辞。先**捭**（开大利益，引楚王心动），待楚绝齐后**阖**（闭口不认，反受秦攻）。楚王因贪开而不能阖，终至丧师失地。

### 用法

| 场景 | 捭（开） | 阖（闭） |
|------|----------|----------|
| 客户访谈 | 开放式提问，引导对方说出真实痛点 | 沉默倾听，不急于表态，让对方补充细节 |
| 谈判开局 | 先抛大框架、愿景，观察对方反应 | 关键底价闭口不谈，以沉默制造压力 |
| 向上汇报 | 主动展开成果与选项 | 风险、未定事项先阖而不说，待领导追问 |
| 面试识人 | 「请谈谈你最大的失败」 | 听完不评价，看候选人是否继续自曝 |

**口诀**：实者捭之使言，虚者阖之使谋。

## 二、反应篇 — 反观往复

<figure>
<img src="{{ '/assets/images/guiguzi/02-fanying.svg' | relative_url }}" alt="反应篇示意图" />
<figcaption>图 2 — 反应：反以观往，覆以察来，象比钓语</figcaption>
</figure>

### 语境要点

- 「反以观往，覆以察来」——用过去推未来，用对方之言反推其意
- 核心手法：**象比**（以类比诱导）、**钓语**（以饵引话）
- 欲闻其声，先默；欲张之，先阖——反向操作

### 案例

**苏秦说秦王**：苏秦初说秦惠王，秦王不纳。苏秦**反其道**：转投六国合纵，待秦惧而合纵成势，再令天下知秦之强不可独抗。以「往之不纳」观「来之必惧」，终使秦国重视连横之策。

### 用法

| 场景 | 反应术 |
|------|--------|
| 需求澄清 | 用户说「我要更快」→ 反问「您目前最慢的一环在哪？」以彼之言导彼之实 |
| 冲突调解 | 不直接否定甲方，先复述甲方立场（覆），再引乙方观点对比（反） |
| 产品调研 | 不说「你需要 X 功能」，问「你现在怎么解决这个问题？」让其自曝流程 |
| 绩效面谈 | 先让员工自述不足（钓语），再针对性给建议，抵触最小 |

**口诀**：言有不合者，反而求之，其应必出。

## 三、内揵篇 — 内合外应

<figure>
<img src="{{ '/assets/images/guiguzi/03-neijian.svg' | relative_url }}" alt="内揵篇示意图" />
<figcaption>图 3 — 内揵：以信任楔子打入，上中下三层递进</figcaption>
</figure>

### 语境要点

- **揵**：楔子，打入对方内部建立信任与影响力
- 三类人：上揵（君主级）、中揵（权臣级）、下揵（执行级）
- 「事皆有内揵，素结本始」——一切游说必先有内应或信任基础
- 远而亲者，有阴德也；近而疏者，志不合也

### 案例

**张仪内揵于秦**：张仪落魄入秦，先结纳掾史、司马错等中层（中揵），再献连横之策于秦惠王（上揵）。若无内廷信任，纵有良策亦难施行。

### 用法

| 场景 | 内揵策略 |
|------|----------|
| 大客户销售 | 先搞定内部 champion（使用者/技术负责人），再触达决策层 |
| 跨部门项目 | 在各相关部门找到利益一致者，建立「内应」联盟 |
| 组织变革 | 先争取关键意见领袖支持，再公开推行 |
| 海外拓展 | 找本地合伙人（下揵/中揵）再触达政府或总部资源 |

**口诀**：欲说者务稳度，说大人必为之来；言往者，先顺辞也。

## 四、抵巇篇 — 乘隙击虚

<figure>
<img src="{{ '/assets/images/guiguzi/04-dixi.svg' | relative_url }}" alt="抵巇篇示意图" />
<figcaption>图 4 — 抵巇：察缝隙、乘虚击，五类巇各有其用</figcaption>
</figure>

### 语境要点

- **巇**：缝隙、裂痕——组织、人心、盟约中的薄弱环节
- 「自天地之弥合，终复归其巇」——事物盛极必衰，隙必生
- 五类巇：明巇、暗巇、贼巇、内巇、间巇
- 圣人见巇而填之、或因而用之

### 案例

**孙膑减灶诱庞涓**：魏军强势，正面不敌。孙膑察庞涓骄矜之**巇**（轻敌），减灶示弱，引其深入马陵——以敌之隙，成致命一击。

### 用法

| 场景 | 抵巇术 |
|------|--------|
| 竞争分析 | 找对手产品/组织/供应链的薄弱环节，不正面硬刚 |
| 危机干预 | 在矛盾初现（小巇）时介入，避免演变成大裂 |
| 团队管理 | 发现成员间的信任裂痕，早沟通弥合，防被人利用 |
| 市场切入 | 大厂忽视的下沉市场、长尾需求，即是巇 |

**口诀**：巇始其末也，可引而塞，可引而反。

## 五、飞钳篇 — 飞而钳之

<figure>
<img src="{{ '/assets/images/guiguzi/05-feiqian.svg' | relative_url }}" alt="飞钳篇示意图" />
<figcaption>图 5 — 飞钳：先扬辞令其心服，再钳势使其不脱</figcaption>
</figure>

### 语境要点

- **飞**：飞扬其辞，令对方飘飘然
- **钳**：钳制其势，使其不得脱
- 用 praise、利益、名声笼络，再以利害钳住
- 「立势而制事」——先造势，再制人

### 案例

**范雎说秦王**：范雎入秦，先谦抑自贬（飞其谦德），秦王再三恳请，方献远交近攻之策。秦王既敬其才，又赖其策，范雎遂钳持秦相之位。

### 用法

| 场景 | 飞钳术 |
|------|--------|
| 人才招募 | 先真诚认可对方过往成就（飞），再以使命与平台绑定（钳） |
| 合作谈判 | 先抬举对方行业地位，再提出独家/排他条款 |
| 演讲说服 | 开场赞美听众，建立好感后再提挑战性问题与方案 |
| 注意边界 | 飞而不实则为谄，钳而过甚则为胁——需把握分寸 |

**口诀**：飞钳之语，钩钳之语，其说辞也，乍同乍异。

## 六、忤合篇 — 合纵连横之机

<figure>
<img src="{{ '/assets/images/guiguzi/06-wuhe.svg' | relative_url }}" alt="忤合篇示意图" />
<figcaption>图 6 — 忤合：因势而合，因势而忤，苏秦张仪各得其时</figcaption>
</figure>

### 语境要点

- **忤**：背逆、不合；**合**：契合、联合
- 「世无常贵，事无常师」——没有永远的对立面，只有当下的势
- 择主而事，择时而动；知其开阖，说之以其类
- 谋臣之职：因时因势，忤而合之，合而忤之

### 案例

**苏秦张仪交替**：苏秦合纵抗秦，张仪连横破纵。二人同窗不同路，各择其主、各合其时。苏秦得势于六国衰危之际，张仪得势于秦强六国惧之时——**忤合**于势，非忠于一人。

### 用法

| 场景 | 忤合术 |
|------|--------|
| 职业规划 | 行业下行时忤（离开），风口来时合（加入） |
| 战略合作 | 今日竞合，明日联盟——以利益与时势为准，非意识形态 |
| 产品 pivot | 原方向忤（放弃），新方向合（市场信号） |
| 政治智慧 | 不在逆势时强谏，待可谏之窗再进言 |

**口诀**：凡趋合倍反，计有适合。

## 七、揣篇 — 揣情察意

<figure>
<img src="{{ '/assets/images/guiguzi/07-chuai.svg' | relative_url }}" alt="揣篇示意图" />
<figcaption>图 7 — 揣：以耳目观外在言行，推度内隐情志</figcaption>
</figure>

### 语境要点

- **揣**：度量、揣测——从外在言行推内在情志
- 「其变在外者，常有馀；其变在内者，常有不足」
- 观其所好，以知其志；观其所恶，以知其情
- 不直接问，而观其应激反应

### 案例

**烛之武退秦师**：郑国将亡，烛之武夜见秦穆公。不直接求援，而**揣**秦之虑：灭郑无益于秦，反壮晋国之东境。以秦之私心（惧晋）破秦郑之隙，秦遂退兵。

### 用法

| 场景 | 揣术 |
|------|------|
| 用户研究 | 不看用户说什么，看用户做什么、反复抱怨什么 |
| 招聘面试 | 观察候选人谈及何事时眼神发亮、何时回避 |
| 商务宴请 | 从点菜、敬酒、谈伴选择揣其地位与关系网 |
| 投资尽调 | 从创始人对失败的描述方式揣其诚实度与韧性 |

**口诀**：揣情者，必以其甚喜甚恶，先观其所好恶，乃知其欲。

## 八、摩篇 — 摩意试探

<figure>
<img src="{{ '/assets/images/guiguzi/08-mo.svg' | relative_url }}" alt="摩篇示意图" />
<figcaption>图 8 — 摩：微触试探，如操钩临渊，饵投而察其应</figcaption>
</figure>

### 语境要点

- **摩**：摩擦、试探——如以羽毛摩物，微触而察其应
- 与揣篇相连：揣是静观，摩是主动轻触
- 「古之善摩者，如操钩而临深渊，饵而投之，必得鱼焉」
- 欲使对方动，先以小激之

### 案例

**毛遂自荐**：平原君欲合纵赴楚，门客不足。毛遂**摩**其急：主动请行，至楚廷以三寸舌摩楚王——先以弱国之急轻触，再层层递进，终定盟约。

### 用法

| 场景 | 摩术 |
|------|------|
| A/B 测试 | 小流量试探用户反应，再放大有效策略 |
| 价格探测 | 先报高价看反应，再逐步让步观察底线 |
| 内部提案 | 会前先与关键人非正式聊，摩其态度再定正式议程 |
| 社交破冰 | 从低风险话题（天气、行业动态）摩起，再深入 |

**口诀**：摩者，求其情也。微而摩之，高而摩之。

## 九、权篇 — 权衡利害

<figure>
<img src="{{ '/assets/images/guiguzi/09-quan.svg' | relative_url }}" alt="权篇示意图" />
<figcaption>图 9 — 权：权衡利害轻重，计寡者智，计多者乱</figcaption>
</figure>

### 语境要点

- **权**：秤锤，权衡——比较轻重、利害、远近、虚实
- 「计多者乱，计寡者智，计不信者诚，计拙者诚，计利者从，计害者避」
- 说人者：务在博其说，信其辞，杂而合之，乱而取之
- 因事为制，权衡于心

### 案例

**蔺相如完璧归赵**：秦王以十五城换和氏璧，蔺相如**权**之：秦强赵弱，直拒则危；遂许献璧至秦，察秦王无诚意（权其虚），乃持璧以撞柱相胁，终完璧归赵。

### 用法

| 场景 | 权术 |
|------|------|
| 方案评审 | 列清各选项的利弊权重，不陷入「计多者乱」 |
| 资源分配 | 权投入产出比，寡计而精行 |
| 谈判让步 | 每次让步必有对价，权其利从、害避 |
| 技术选型 | 权长期维护成本 vs 短期交付速度 |

**口诀**：凡谋有道，必得其情，审得其此，成其事而后可。

## 十、谋篇 — 阴谋阳略

<figure>
<img src="{{ '/assets/images/guiguzi/10-mou.svg' | relative_url }}" alt="谋篇示意图" />
<figcaption>图 10 — 谋：阴中策划，阳中成事，商鞅变法为例</figcaption>
</figure>

### 语境要点

- 「谋之于阴，成之于阳」——策划在暗，成事在明
- 为人臣者，莫难在於进说，莫妙在於得君之旨
- 同利相导，同害相救——找共同利益点串联多方
- 不见其类而为之者，见逆；不得其情而说之者，见非

### 案例

**商鞅变法**：谋之于孝公之前（阴）：三次说以帝道、王道、霸道，终以强国之术合其旨。成之于秦国法令（阳）：徙木立信，公开颁法。阴谋得君心，阳略行天下。

### 用法

| 场景 | 谋术 |
|------|------|
| 产品发布 | 内测收集反馈（阴谋），发布会高调亮相（阳略） |
| 组织改革 | 小范围试点验证（阴），成功后全面推广（阳） |
| 联盟构建 | 私下对齐各方利益（阴），再签公开协议（阳） |
| 个人影响力 | 默默积累专业口碑（阴），关键场合一次亮剑（阳） |

**口诀**：故为强者，积于弱；为直者，积于曲。

## 十一、决篇 — 决断之机

<figure>
<img src="{{ '/assets/images/guiguzi/11-jue.svg' | relative_url }}" alt="决篇示意图" />
<figcaption>图 11 — 决：五机之中速断疑虑，机不可失</figcaption>
</figure>

### 语境要点

- **决**：决断——在五机（危、安、死、生、誉、辱）中速断
- 「决断以决疑」——犹豫是最大的成本
- 善决者：先察是非之语，别成败之纪，察同异之党，权当势之宜
- 度远而近，审古而察今

### 案例

**鸿门宴范增举玦**：项羽宴刘邦，范增数举玉玦示杀意。项王**不决**，沛公脱险。范增叹曰：「竖子不足与谋！」——机不可失，决而不断，反受其乱。

### 用法

| 场景 | 决术 |
|------|------|
| 创业 pivot | 数据已明，勿因感情拖延关停亏损业务 |
| 人事决策 | 察其已久，去留当断，勿拖成团队毒瘤 |
| 投资退出 | 设好止损线，触发即决，不存侥幸 |
| 日常管理 | 二选一时，权势之宜后速决，再迭代修正 |

**口诀**：夫决情定疑，万事之基也。

## 十二、符言篇 — 持符守信

<figure>
<img src="{{ '/assets/images/guiguzi/12-fuyan.svg' | relative_url }}" alt="符言篇示意图" />
<figcaption>图 12 — 符言：耳目心协调，君臣各守其符</figcaption>
</figure>

### 语境要点

- **符**：信符、符节——君主与谋士之间的信任契约
- 安徐正静，柔节先定；善与而不静，先意从欲
- 目者可以观色，耳者可以听声，心者可以知情
- 总纲：耳目心三者协调，言听计从而不妄动

### 案例

**管仲相桓公**：桓公以鲍叔牙荐，授管仲全权。管仲**持符**于桓：「国有政于仲，无得而问焉。」君臣各守其位，管仲谋、桓公信，遂成霸业。

### 用法

| 场景 | 符言术 |
|------|--------|
| 顾问角色 | 谋臣不越位代决，主公不微服乱谋——各守其符 |

## 阅读边界

《鬼谷子》教的是**术**，不是**德**。读此书宜：

1. **明其理，不泥其诈** — 识破套路，亦守己之诚
2. **识势而不逐奸** — 势可借，道不可弃
3. **古今异制** — 战国策士以口舌换封地，今人宜用于正当商业与职业场景
4. **与儒家互补** — 鬼谷子教「如何成」，孟子教「为何成」；术德并修，方为上乘
