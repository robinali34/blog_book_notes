**The Power of When: Discover Your Chronotype and Learn to Do Everything at the Right Time**  
Michael Breus · 临床睡眠心理学家 · 核心：**人与人节律相位不同，同一任务放在不同时段，效率与损耗差很多**

> 不是人人都该六点起床；先弄清你是狮子、熊、狼还是海豚。  
> 但「类型」是地图，不是借口——锚点起床仍要。

## 全书概览

Michael Breus 用通俗四型（**狮子、熊、狼、海豚**）把昼夜节律差异翻译成日程语言：何时深度工作、何时开会、何时运动、何时创意、何时性亲密。分型依据主要是**睡眠中间点**与问卷，本质是简化版相位分类，方便办公室人与团队排程，而非精密医学分型。

价值在三处：**(1)** 承认晚型不是道德问题；**(2)** 把任务与时钟对齐，减硬拧；**(3)** 海豚型提醒失眠倾向者别只靠「早起文化」自责。局限亦明：四型边界模糊、样本营销感强、不能替代 OSA/CBT-I。

读本书，宜记三问：**我的睡眠中间点在哪？最难的任务是否放在清醒峰？我是否用「我是狼」逃避固定起床？**

## 一、四型模型 — 狮、熊、狼、海豚

### 语境要点

| 类型 | 约占比 | 睡眠中间点倾向 | 清醒峰 | 典型困境 |
|------|--------|----------------|--------|----------|
| **狮子** | ~15–20% | 较早（约 2–4am） | 清晨–上午 | 下午社交性下降 |
| **熊** | ~50% | 中段（约 3–4am） | 上午中段 | 随社会钟，午后低谷 |
| **狼** | ~15–20% | 较晚（约 4–6am） | 下午–傍晚 | 早会痛苦、被贴懒 |
| **海豚** | ~10% | 不规律 | 碎片 | 失眠、浅睡、焦虑 |

**睡眠中间点** =（入睡时刻 + 起床时刻）÷ 2（跨午夜须换算 24h 制）。

### 深度解析

```text
社会默认日程 → 为「熊」设计 → 狼在早会损耗、狮在晚局强撑
Breus 目标：在固定锚点下，把高认知任务移到个人峰
```

与 [生命时光]({{ '/2026/06/26/life-time-russell-foster/' | relative_url }})：青少年狼型多，政策与个人排程应分流。

### 粗分流程

| 步骤 | 动作 |
|------|------|
| 1 | 假期或无闹钟 3 天，记自然睡醒 |
| 2 | 算中间点 |
| 3 | 书内问卷交叉 |
| 4 | 2 周后复测，取稳定型 |

### 典型案例

**案例 A：狼型产品经理**  
强制 9 点站会 + 晨间深度写文档 → 全年低效。改：站会 10:30，深度块 15–18 点，产出升——**排程即合理便利**。

**案例 B：狮子型家长**  
晚陪作业崩溃。配偶轮值晚间，狮子 21 点后低刺激恢复——见 [HSP]({{ '/2026/05/26/highly-sensitive-person-aron/' | relative_url }}) 剂量分劳。

### 反模式

| 误用 | 后果 | 纠正 |
|------|------|------|
| 类型当命运 | 不优化 | 在型内排程 |
| 狼型拒绝固定起床 | 相位更乱 | 锚点+峰内工作 |
| 海豚自诊不就医 | 延误 CBT-I | 专业评估 |

**口诀**：型定峰，峰定任务，锚点定全天。

## 二、狮型 — 早峰与午后管理

### 语境要点

狮子天然晨型：**5–12 点**常是黄金深度时段；午后警觉降，宜会议、行政、轻创意，不宜硬扛复杂谈判。

| 时段 | 宜 | 忌 |
|------|----|----|
| 晨 | 深度工作、难决策 | 刷邮件淹峰 |
| 午 | 协作、走动会议 | 无午休强撑 |
| 晚 | 轻量、恢复 | 高冲突家庭会 |
| 睡 | 较早入睡窗 | 23 点后强光 |

### 深度解析

与 [R90]({{ '/2026/06/26/sleep-nick-littlehales/' | relative_url }})：狮子常 5 周期即可，**固定起床**仍比「自然早醒」更稳周末。

### 狮型日程模板

| 时间 | 块 |
|------|-----|
| 6:00–9:00 | 深度工作 A |
| 9:00–10:00 | 邮件/轻沟通 |
| 10:00–12:00 | 深度工作 B |
| 14:00–16:00 | 会议 |
| 20:30 | 睡前仪式 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 晨峰刷短视频 | 保护峰 |
| 晚间逞强社交 | 预约恢复 |

## 三、熊型 — 随社会钟的默认多数

### 语境要点

熊最多，**与朝九晚五重合度最高**——仍须保护上午中段峰与午后低谷（Breus 称「重启」）。

| 时段 | 宜 |
|------|-----|
| 8–12 | 深度、创意 |
| 12–14 | 午Walk、轻餐 |
| 14–16 | 低谷：机械任务 |
| 16–18 | 第二波中等峰 |

### 深度解析

团队排程若以熊为准，狼/狮须**显式协商**；见 [管理的本质]({{ '/2026/06/02/management-essence-interest-exchange/' | relative_url }}) 交换逻辑。

### 熊型场景表

| 场景 | 策略 |
|------|------|
| 开放式会议扎堆上午末 | 保护 10–12 深度 |
| 午后困 | 光、Walk、非咖啡因 |
| 周末 | 社会时差 ≤1h |

### 反模式

| 误用 | 纠正 |
|------|------|
| 熊型=不用优化 | 仍须保护峰 |
| 午后咖啡续命 | 伤夜睡 |

## 四、狼型 — 晚峰不是懒

### 语境要点

狼的相位偏晚：**深度与创意常在 15–20 点甚至更长**。早会低效不等于工作态度差。

| 策略 | 内容 |
|------|------|
| 锚点 | 仍固定起床（可略晚于狮），周末勿暴睡 |
| 晨光 | 必做，减社会时差 |
| 深度块 | 下午–傍晚 |
| 早会 | 争取异步摘要或晚一点 |

### 深度解析

与 [Foster]({{ '/2026/06/26/life-time-russell-foster/' | relative_url }}) 青少年议题：许多狼是**生理+社会夹缝**。

与 [Panda]({{ '/2026/06/26/circadian-code-satchin-panda/' | relative_url }})：狼型进食窗可整体后移，但避免睡前 3h 大餐。

### 狼型谈判话术

| 请求 | 示例 |
|------|------|
| 弹性到岗 | 「10 点前到岗，14–18 交付核心稿」 |
| 异步站会 | 「晨会改书面，15 点同步 15 分」 |
| 证据 | 试点 4 周对比产出 |

### 典型案例

**案例：狼型工程师**  
经理同意 10–19 工时，11 点前无会 → bug 率降、离职意降。

### 反模式

| 误用 | 纠正 |
|------|------|
| 凌晨 2 点睡叫狼 | 可能是失衡 |
| 无锚点自由睡 | 相位漂移 |

## 五、海豚型 — 失眠倾向须双轨

### 语境要点

海豚：**浅睡、易醒、高焦虑倾向**，与慢性失眠重叠多。Breus 给作息建议，但**医疗与 CBT-I 优先**。

| 重点 | 说明 |
|------|------|
| 固定起床 | 比所有型都重要 |
| 床只用于睡 | CBT-I 刺激控制 |
| 咖啡因 | 严格截止 |
| 小睡 | 短、规则；见 R90 |
| 就医 | 长期 >3 月失眠 |

### 深度解析

与 [HSP]({{ '/2026/05/26/highly-sensitive-person-aron/' | relative_url }})：高敏感+海豚常叠加——**气质剂量 + 失眠治疗**并行，不单贴标签。

与 [DBT]({{ '/2026/05/19/dbt-skills-handouts-linehan/' | relative_url }})：夜醒焦虑用痛苦耐受；白天 PLEASE。

### 海豚型红线信号

| 信号 | 动作 |
|------|------|
| 日间嗜睡开车困 | 查 OSA |
| 早醒+心境低 | 抑郁筛查 |
| 躺床>30 分睡不着 | CBT-I |

### 反模式

| 误用 | 纠正 |
|------|------|
| 「我是海豚注定睡不好」 | 可治比例高 |
| 酒精助眠 | REM 损 |

| 任务 | 狮 | 熊 | 狼 | 海豚 |

| 深度分析 | 晨 | 上午 | 下午晚 | 个人较稳的 2h 块 |

## 七、团队与社会钟 — 熊中心世界的协商

### 语境要点

默认办公室为熊设计。Breus 建议团队：

- 核心同步窗放在 **10–15 点**（多数熊+部分狮狼可及）  
- 深度工作 **无会块** 全型受益  
- 狼型争取 **异步** 与 **结果考核**  

### 深度解析

```text
不是人人改点上班 → 而是「会议/深度/弹性」三分
```

### 团队规则示例

| 规则 | 目的 |
|------|------|
| 11 前无会（可选） | 保护狮熊峰 |
| 会议≤25 分 | 减低谷损耗 |
| 书面默认 | 狼晨参与 |
| 结果 KPI | 减「坐班=勤奋」 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 狼型要求零协作 | 交换异步 |
| 管理者凌晨回邮文化 | 组织睡眠债 |

## 九、FAQ

**Q1：四型科学吗？**  
简化工具；相位连续分布，勿教条。

**Q2：中间点怎么算跨夜？**  
入睡 1:00、起床 9:00 → 中间约 5:00（示例；用 24h 换算）。

**Q3：可以既是狼又是 HSP 吗？**  
可以；排程+边界+睡眠治疗分轨。

**Q4：固定起床对狼太残忍？**  
锚点略晚可；周末暴睡更残忍。

**Q5：海豚和失眠症区别？**  
海豚是倾向描述；慢性失眠是诊断，治则 CBT-I 等。

**Q6：与 [R90]({{ '/2026/06/26/sleep-nick-littlehales/' | relative_url }}) 冲突吗？**  
不冲突；R90 定周期，Breus 定日内峰。

**Q7：育儿打乱型？**  
阶段性；锚点仍尽量，任务降级。

**Q8：远程是否更宜狼？**  
可谈判；勿无锚点。

**Q9：咖啡因按型不同？**  
海豚最严；狼下午仍须截止。

**Q10：团队不买账？**  
试点个人峰内深度 4 周，用产出谈。
