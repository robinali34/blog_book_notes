**Sleep: Redefine Your Rest, for Success in Work, Sport and Life**  
Nick Littlehales · 英国睡眠教练（长期服务曼联等运动员）· 核心：**按约 90 分钟睡眠周期规划休息，固定起床比执着于「满 8 小时」更减压**

> 固定起床比固定入睡更重要；睡够周期，比睡够整点更安心。  
> R90 是给普通人的「训练恢复」语言，不是玄学周期崇拜。

## 全书概览

Nick Littlehales 从职业体育睡眠管理走向大众，提出 **R90（每 90 分钟一个睡眠周期）** 思维：一夜通常 4–6 个周期，与其焦虑「差半小时没满 8 小时」，不如规划 **5×90≈7.5h** 或 **6×90≈9h**，并以**固定起床时刻**为全周锚点倒推就寝窗口。配套强调：睡前 **90 分钟降刺激仪式**、卧室功能单一、可控 **20 分钟小睡（CMT）**、出差与时差仍守锚点。

R90 的价值在**减轻睡眠表现焦虑**、给出差/赛事一个简单操作架；局限在周期长度个体差（80–110 分皆常见）、不能替代呼吸暂停与 CBT-I 医疗路径。与 [我们为什么要睡觉]({{ '/2026/06/26/why-we-sleep-walker/' | relative_url }}) 的 NREM/REM 科普互补：Walker 讲「为什么要睡」，Littlehales 讲「怎么排睡才不慌」。

读本书，宜记三问：**我的固定起床锚点是什么？我按周期倒推还是按焦虑盯钟？打鼾嗜睡有没有超出 R90 能管的范围？**

## 一、R90 周期思维 — 从「小时」到「周期」

### 语境要点

传统问：「昨晚睡了几小时？」R90 问：「昨晚完成了几个完整周期？」**周期结束于醒来**，若在周期末自然醒，往往比深睡中被闹钟截断更清爽（个体差异大）。

| 周期数 | 约时长 | 适用情境 |
|--------|--------|----------|
| 4 | 6h | 短睡者、临时（不推荐长期） |
| 5 | 7.5h | 多数成人起点 |
| 6 | 9h | 高负荷训练、青少年、偿还剥夺 |
| 7 | 10.5h | 短期恢复、病后 |

### 深度解析

```text
固定起床 7:00 → 倒推 5 周期 → 就寝窗口约 23:15–00:15（含约 15 分入睡潜伏期）
不在窗口内睡着 → 不焦虑刷手机算小时 → 等下一周期节点或起锚点
```

与 [Walker]({{ '/2026/06/26/why-we-sleep-walker/' | relative_url }})：REM/NREM 构成一周期；90 分钟是**平均模型**，勿迷信手表「深睡分」。

### 周期规划表

| 起床 | 5 周期就寝窗中心 | 6 周期就寝窗中心 |
|------|------------------|------------------|
| 6:00 | 22:15 | 20:45 |
| 6:30 | 22:45 | 21:15 |
| 7:00 | 23:15 | 21:45 |
| 7:30 | 23:45 | 22:15 |

（粗算；个体入睡潜伏期 10–20 分需加减。）

### 典型案例

**案例 A：8 小时焦虑**  
每晚盯满 8:00 睡、7:00 起，差 20 分就挫败。改 R90：5 周期 7.5h，起床固定 7:00，就寝窗 23:00 前后 → 焦虑降、坚持率升。

**案例 B：运动员赛季**  
大赛前增至 6 周期；赛后渐回 5 周期——**周期如训练量，可周期化**。

### 反模式

| 误用 | 后果 | 纠正 |
|------|------|------|
| 周期教条 90.0 分 | 手表焦虑 | 功能为准 |
| 4 周期长期硬扛 | 剥夺 | 至少 5 |
| 不设锚点 | 相位乱 | 先固定起床 |

**口诀**：锚点在前，周期倒推，窗口弹性。

## 二、固定起床 — 全周第一优先级

### 语境要点

Littlehales 与 Foster、Walker **共识**：**起床时刻**是全周最强授时信号。周末赖床 2h = 社会时差，周一崩。

| 规则 | 说明 |
|------|------|
| 全周同一锚点 | 含周六日 |
| 周末最多晚 30–60 分 | 与 [Foster]({{ '/2026/06/26/life-time-russell-foster/' | relative_url }}) 一致 |
| 闹钟放远 | 迫起身，减赖床周期 |
| 醒后立即光 | 见 Panda 光轴 |

### 深度解析

```text
固定起床 → 腺苷与相位对齐 → 困意前移（1–2 周）
比「逼自己 22:00 睡着」更少条件性失眠
```

与 [Breus]({{ '/2026/06/26/power-of-when-michael-breus/' | relative_url }})：狼型锚点可略晚，但**必须有**。

### 起床仪式清单

| 步骤 | 目的 |
|------|------|
| 开窗帘/户外 5 分 | 授时 |
| 水一杯 | 代谢觉醒 |
| 轻活动 | 体温升 |
| 不刷刺激新闻 | 防焦虑开局 |
| 同一顺序每日 | 条件反射 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 工作日锚、周末放飞 | 社会时差 |
| 醒后赖床刷手机 | 床=清醒关联 |

## 三、睡前 90 分钟 — 降刺激仪式

### 语境要点

睡前 **90 分钟**（R90 另一含义）用于从「日模式」切换到「夜模式」：**减光、减争论、减工作邮件、减剧烈训练**。

| 要素 | 建议 |
|------|------|
| 光 | 暖色、低照度 |
| 屏 | 减刺激内容；非教条禁屏 |
| 工作 | 封档仪式：明日三件写下 |
| 食 | 大餐早于睡前 3h，见 [Panda]({{ '/2026/06/26/circadian-code-satchin-panda/' | relative_url }}) |
| 争论 | 暂停；明晨再谈 |

### 深度解析

与 [HSP]({{ '/2026/05/26/highly-sensitive-person-aron/' | relative_url }})：高敏感者仪式收益更大——**神经系统需明确「下班信号」**。

与 [DBT]({{ '/2026/05/19/dbt-skills-handouts-linehan/' | relative_url }})：晚间情绪高激活时先 TIP/正念，再进仪式。

### 仪式模板（可压缩 30 分版）

```text
T-90：最后一杯咖啡因（下午已截止）/ 高强度训练结束
T-60：关工作通知、调暗灯
T-30：洗漱、拉伸、轻阅读
T-0：上床；困否都保持床只睡觉关联
```

### 典型案例

**案例：夫妻夜吵**  
争论延至 0 点 → 双方入睡推迟。约定 22 点后冲突 **暂停符** + 明早 9 点「关系会」→ 入睡窗回稳。

### 反模式

| 误用 | 纠正 |
|------|------|
| 仪式完美主义 | 做 60 分版也好 |
| 床内回工作邮件 | 卧室功能单一 |

## 四、卧室与床 — 刺激控制

### 语境要点

CBT-I 与 R90 交集：**床主要用来睡**；久躺清醒应离床，避免「床=焦虑清醒」条件反射。

| 原则 | 做法 |
|------|------|
| 功能单一 | 睡与性；少电视手机 |
| 环境 | 暗、凉、静 |
| 温度 | 略凉（个体化） |
| 时钟 | 不见秒针焦虑 |

### 深度解析

```text
躺床 20 分仍清醒 → 离床低刺激 → 困再回
比「强迫自己躺着算小时」更符合 R90
```

### 卧室审计表

| 项目 | 合格 | 改法 |
|------|------|------|
| 光漏 | 否 | 遮光帘 |
| 手机 | 床头 | 充客厅 |
| 温度 | 闷热 | 通风/减被 |
| 噪音 | 突发 | 耳塞 |
| 关联 | 床办公 | 书桌完成 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 卧室健身房 | 分区 |
| 追踪器焦虑 | 以白天功能为准 |

## 五、CMT 小睡与恢复 — 非暴睡补债

### 语境要点

**Controlled Mid-day Nap（CMT）**：约 **20 分钟**或谨慎使用 **90 分钟整周期**（需时间充裕）。目的：午后低谷 **重启**，非周末 3h 暴睡还债。

| 类型 | 时长 | 注意 |
|------|------|------|
| 微休 | 10–20 分 | 防进入深睡后昏沉 |
| 整周期午眠 | ~90 分 | 需固定起床仍不变 |
| 周末暴睡 | 避免 | 社会时差 |

### 深度解析

与 [Breus]({{ '/2026/06/26/power-of-when-michael-breus/' | relative_url }})：熊型午后低谷、狼型傍晚前皆可规划 CMT，**勿近夜间就寝**。

### 小睡场景表

| 场景 | 建议 |
|------|------|
| 午后会议前 | 15 分闭眼 |
| 夜班前 | 策略性长nap；职业健康 |
| 带娃夜醒 | 配偶轮值 + 周末不暴睡 |
| 出差时差 | 锚点起床 + 光 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 17 点睡 1h | 伤夜睡 |
| 小睡不设闹钟 | 深睡后眩 |

## 六、出差、赛事与时差 — R90 的体育遗产

### 语境要点

Littlehales 体育案例核心：**锚点不变、环境可变**——出差带眼罩耳塞、酒店温度调节、倒时差先控 **起床+光**。

| 情境 | R90 策略 |
|------|----------|
| 跨 1–3 时区 | 目标时区锚点起床 |
| 赛前 | 临时 6 周期 |
| 赛后 | 渐回 5 周期 |
| 红眼航班 | 落地按当地锚点起 |

### 深度解析

与 [个人成长实践路径]({{ '/2026/06/25/personal-growth-practice-plan/' | relative_url }})：差旅周可降级其他习惯，**睡眠锚点保留**。

### 差旅包清单

| 物品 | 用途 |
|------|------|
| 眼罩+耳塞 | 光声 |
| 枕套 | 熟悉触感 |
| 褪黑素 | 医嘱选用 |
| 起床双闹钟 | 锚点 |

### 反模式

| 误用 | 纠正 |
|------|------|
| 出差完全放飞 | 锚点保相位 |
| 酒精机上助眠 | REM 损 |

## 九、FAQ

**Q1：周期一定是 90 分钟吗？**  
平均约 90；个体 80–110 常见。以几周试验与日间功能为准。

**Q2：必须 5 周期吗？**  
多数成人 5 起点；高负荷或青少年可 6；长期 4 不足。

**Q3：与 [Walker]({{ '/2026/06/26/why-we-sleep-walker/' | relative_url }}) 8 小时矛盾？**  
5 周期 7.5h 常够用；功能不佳先查 OSA 与剥夺史。

**Q4：失眠能用 R90 吗？**  
锚点与仪式有帮助；慢性失眠加 CBT-I。

**Q5：打鼾嗜睡？**  
超出 R90；就医查呼吸暂停。

**Q6：HSP 需要更多周期？**  
可能偏 6；看恢复与过载频率，见 [HSP]({{ '/2026/05/26/highly-sensitive-person-aron/' | relative_url }})。

**Q7：睡前仪式没时间？**  
30 分压缩版：关工作、调暗、洗漱。

**Q8：狼型晚起？**  
锚点可略晚，全周仍固定。

**Q9：亲子夜醒？**  
锚点尽量；配偶轮值；勿周末暴睡还债。

**Q10：手表深睡分低？**  
趋势参考；R90 重功能不重分。
